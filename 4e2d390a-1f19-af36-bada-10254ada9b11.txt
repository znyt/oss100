　　“四箭山河碎！”
<br/>　　声音冷漠无情，却又充斥着无尽杀机，更带着一种坚定的信念。接连三次射箭，射出六道神箭，以后羿顶尖大巫的实力，也有些疲惫了。
<br/>　　当然，这里的疲惫并不是身体的疲惫，而是心力上的疲惫。人们只知道后羿箭术了得，堪称洪荒第一，但他们却并不知道，后羿的箭术之所以如此了得，那是因为他射箭从来不用眼睛，而是用心。
<br/>　　心，不错，就是心。
<br/>　　显然，后羿亦如同元始、伏羲那般，领悟了心之力。
<br/>　　只不过，三人的心力研究方向各有不同，元始的心力主要是对心境的磨砺，对人生的感悟，对心无极限、无束缚的感悟。
<br/>　　伏羲的心力主要是在对人的控制之上，所以他能够用心力演奏琴曲，影响进而控制人的心灵。
<br/>　　而后羿的研究方向就更是不同了，他对心力的参悟全部放在攻击方面。
<br/>　　箭在飞，心在飞！
<br/>　　箭随心动，心之所向，心之所念，心之所及，皆为箭。心，虚无缥缈，让人躲无可躲，藏无可藏。心力的攻击，更加神秘，也更加强大。
<br/>　　因为极致于箭，所以有了心的出现。
<br/>　　按元始所分的心力境界，后羿如今已经处于第三境，主宰境。心之主人，唯自己尔。
<br/>　　心力加持下，只会使得后羿的箭术更加神奇，威力更加巨大，因为这箭术不仅作用在人的肉体神魂伤害上，更为神奇的是，它还能祸乱人之内心，破除他人道心。
<br/>　　“小二、小五、小十，你们快走，我来挡住他！”
<br/>　　大金乌眼见自己走不了了，顿时向着剩下三只金乌大声喊道。
<br/>　　“不——大哥！”
<br/>　　三只金乌大惊失色，目如闪电，一声凄厉尖叫，他们也知道自己不是后羿的对手，来不及多说，只能忍受着心中的沉痛，向着天庭之上飞去。他们不知道为何自己的父皇还没有来救自己等人，但如今已经来不及多想了，唯有拼命逃命。
<br/>　　后羿的第四道箭快速射出，无形的箭光呼啸过天际，洞穿层层虚空，带着呼啸声没入虚空，让人发现不了。一股破碎山河，粉碎苍穹之势，牢牢地锁定剩下的四只金乌。
<br/>　　四大金乌太子只觉得浑身冰冷刺骨，就连身上的太阳真火也不能为他们驱除这股惊天寒意。他们心头一战跳动，元神不住的抖动。他们感受到生死之危，毫无生机。
<br/>　　“尔敢！”
<br/>　　正在此时，一声霸道威严的暴怒声传遍天际，轰动苍穹，让风云为之一滞，乾坤为之一顿，时光为之一停。之后，只见一道璀璨的金色帝皇剑光划破天际，从虚无之中斩了出来，带着无尽锋锐之气，斩破无尽苍穹，斩向那虚无箭光。
<br/>　　“轰隆隆！”
<br/>　　又是三道巨大的火球坠落下来，太阳真火蔓延扑空，带着长长的尾巴，划破天际，砸入大地。
<br/>　　那剑光虽然及时，但终究也只能救下一只金乌，其他三只金乌就没有那么好的运气了，他们直接被后羿的无形心箭射杀了。
<br/>　　“啊——还我儿命来，我要你死！”
<br/>　　一声暴怒大吼，惊天动地，只见九天之上，无边风云涌动，无尽妖云汇聚，自九天之上倾泻下来，一道高大的身影，携带着无尽神威，无双帝威，降临下来，如帝临尘。
<br/>　　“铮——”
<br/>　　几乎是瞬息之间，一道金色的剑光，无坚不摧，随着一声高昂的剑鸣，金色剑气破空而至，可怕的剑光，凌厉到了极致，连天地都被划开，一出手便是毫不留情的斩向后羿。
<br/>　　显然，最关键的时候，帝俊赶到了。
<br/>　　这段时间中，帝俊只觉得自己心头一阵不安，元神抖动，好似要有什么不利之事要发生。修为到了他们这一地步，这种感觉比什么都准，由不得他不相信。他一阵掐算，但是不管他如何演算，他都算不出丝毫的动机。
<br/>　　连他都算不到任何东西，念头涌动，一瞬间，他便知道是圣人出手了。
<br/>　　想到此处，他赶紧叫来羲皇伏羲，让他帮忙推算。伏羲为洪荒顶尖的神算子，果然，他一出手，在耗费无数代价，遭受天机反噬的情况下，终于演算出来，是帝俊的十个儿子有生命之危。
<br/>　　在知道此事后，帝俊来不及多想，他的儿子不是被他封印了吗，为何会有生死之危？他急忙下了天界，之后便看到了眼前这一幕。来不及多想，看见自己儿子被杀了只剩下四个，他含愤出手，全力一击，拯救自己的儿子。
<br/>　　可惜，就算是他出手了，但也已经有些晚了，还是没能全部救下余下的金乌，只能救下靠近他最小的一只。
<br/>　　自己儿子被射杀，帝俊暴怒异常，见是自己的对头巫族，更是愤恨不已。他全力一击，一剑斩出，势要杀了这个杀人凶手。
<br/>　　面对帝俊那霸道的剑光，后羿只感觉自己的身体仿佛被禁锢一般，不能动弹分毫，在他的眼中，一道金色的剑光，带着无尽杀意，向自己斩来，毁天灭地，实力相差太大，不见丝毫生机。
<br/>　　这一刻，他反倒是平静下来，他的心头一阵涌动，过去的一幕幕在他心头快速闪过，巫族的情意，与嫦娥的相知相遇相爱，一点一滴，都是那么令人沉醉。
<br/>　　这些只是在瞬息之间便发生了。他心头一阵冷静，越冷静，越清晰。
<br/>　　“不！我不能死，嫦娥还在家中等着我！部落里的人还在等着我！”
<br/>　　后羿心中一阵怒吼，热血狂涌而起，身体变得通红无比，他全身炙热，就好比被大火烧烤，气血翻滚，热血沸腾。
<br/>　　此时，深藏在他血脉深处的无尽规则因子瞬间变得璀璨无比，强烈的光芒将他包围，一道道血脉因子相互连接，最后连成一条锁链。
<br/>　　“轰！”
<br/>　　后羿全身气势爆发，他突破了大巫的限制，进入到了祖巫境。巫族本就是盘古血脉后羿，在他们的血脉之中，或多或少的都有着盘古的的传承。其中，以十二祖巫最为明显，之后才是大巫。
<br/>　　后羿身为后土祖巫的血脉创造出来的大巫，有着顶尖的血脉，平时有着后土这位祖脉的压制，深藏不漏，可如今后土已死，成了平心娘娘，他头上便再也没了压制。
<br/>　　如今一旦觉醒，便是祖巫，就算没有后土强大，但也能算得上是后土第二了。
<br/>　　来不及高兴，因为危机还远没有解决。眼见剑光就要斩道自己眼前，后羿深吸一口气，他知道自己无论怎么躲，都逃不开这道剑气。
<br/>　　他心力涌动，幻化出一层闪烁着九彩之光的无形铠甲，包裹在自己身上。
<br/>　　同时，后羿一念之间，箭光再次闪动，带着无尽破空之意，与那道金色霸道剑光深深的撞击在了一起。
<br/>　　“轰隆隆！”
