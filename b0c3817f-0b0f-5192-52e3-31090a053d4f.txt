        &lt;h3&gt;第三卷第二百九十五章，你的安好是我最喜欢看到的阳光&lt;/h3&gt;
<br/>
<br/>        告别了陈子澜，陈子昂驱车到了榴花饭店，这是一个五星级的饭店，饭店接待的都是一些国际上的人士，已经有了很久的历史。
<br/>
<br/>        陈子昂到达饭店包间的时候，江俞轩已经到了，陈子昂有事情要和江俞轩商量，就特意提前了一个多小时，他们将事情讨论完之后吃晚饭的时间也就刚刚好。
<br/>
<br/>        “俞轩，你这么早？”陈子昂好像忘记了和江俞轩之间的一切，似乎他们不曾离开过，也不曾有过误会。
<br/>
<br/>        “嗯，我刚到不久。”江俞轩急切的站起来看着陈子昂，陈子昂冷淡的看了他一眼，走到对面的沙发上坐下。
<br/>
<br/>        陈子昂坐下对服务生说：“麻烦来两杯茶，金骏眉就好。”
<br/>
<br/>        她看到江俞轩身边没有茶水，想着江俞轩也是刚到不久，却没有想到江俞轩早到了半个小时，他根本无心喝茶，一直在想着怎么解释那个误会。
<br/>
<br/>        “俞轩，你下午有去找刘翼飞吗？”陈子昂直接就开口问工作上的事情。
<br/>
<br/>        “有，刘翼飞很客气，问了我和张家的关系，还问了你和陆宇霆的关系。”
<br/>
<br/>        “真是一个老狐狸。”陈子昂微微一笑。
<br/>
<br/>        “对你倒是赞不绝口，另外还表扬了我对西城所做的贡献，希望我再接再厉，继续为西城的发展出一份力，还让他谈了谈高新和西岭俩个项目的看法。”江俞轩轻茗了一口茶。
<br/>
<br/>        “你怎么说？”
<br/>
<br/>        “我说各有千秋，但从商业角度来说，西岭比高新的项目更赚钱，从面子上来说，高新更能突出成绩。”江俞轩笑着说。
<br/>
<br/>        “你现在也是狡猾狡猾的了。”
<br/>
<br/>        “那可不，我总不能辜负了你的一番心意。”
<br/>
<br/>        “其实，我也是为西城好，西城是我的故乡，西城发展好了，我骄傲啊。你这么一说，刘翼飞会力排众议，让你拿下高新的项目，因为他要证明他的选择是正确的。”
<br/>
<br/>        “后期会有一场残酷的战斗，毕竟你和我都是狐假虎威，压根就不是陆家的人。”
<br/>
<br/>        “商业上虚虚实实，谁又能说得清楚？”陈子昂是真的喜欢和江俞轩在一起共事，因为他们俩人只需要一个眼神或者是一句话都明白对方的心意。
<br/>
<br/>        可惜这种状态已经被破坏了，而且不复再来了，陈子昂在心底叹息一声，刚刚从三姐哪里回来，她决定要阻止江俞轩去甸城了。
<br/>
<br/>        甸城比她想象的要危险，是真的危险，那里不仅仅只是信息化上的角逐，也不仅仅是金钱的角逐，后期可能是正义和邪恶，生于死！
<br/>
<br/>        “等高新的项目拿下来，我就去甸城，让我姐姐负责这边就好了。到时候我们还是在一起。”江俞轩信心满满。
<br/>
<br/>        “先不说去甸城的事情，我们先来讨论西城出租业的状况，出租业这么几年并没有太大的变化，我们的汽车投入起到了一种明显的对比，让他们感觉到了危机，北京的接受能力比西城强，西城这两天可能会越演越烈，我们需要在两天内将官方的事情解决了。你去找我的四姐陈子乔，让她带你去见马总。告诉他们我们要投资，让马总指定人和你对接。”
<br/>
<br/>        “为什么你不去找他们？”
<br/>
<br/>        “我要离开西城了啊，后期西城的事情我又不负责。另外，你不能去甸城了。”
<br/>
<br/>        “为什么？”
<br/>
<br/>        “俞轩，我需要你帮助我，这个世界上只有你能真心实意的帮助我。”陈子昂一脸的请求，让江俞轩心里又难过又开心。
<br/>
<br/>        “在西城，你有什么需要我帮助的？”
<br/>
<br/>        “张函那一天明确的告诉我，他是陆家的女婿，在利益面前他什么都不是，他会和陆家在一起，我现在是孤掌难鸣，我需要去甸城，我还需要照顾孩子，另外西城分公司也面临着夏颜的刁难，我感觉很吃力，原本想你和我一起去甸城，现在我又担心我去了甸城，北京，西城将举步难行。”陈子昂深深的叹口气，她其实说的都是事实。
<br/>
<br/>        “你让我留下照顾西城和北京？”
<br/>
<br/>        “是，我三姐夫已经开始查洛一和洛二的问题的，相信会影响夏颜一些时日，你得帮着我守住后方的一切，让我没有后顾之忧。还有，西岭的项目影响生态环境，后期将会是鸡肋，一丝一毫都不要接受。西岭将会是一个企业的墓葬场。”
<br/>
<br/>        “这些都不影响我去甸城发展。”江俞轩不同意陈子昂的建议。
<br/>
<br/>        “我知道我将西城和北京，乃至全国的工作交给你，对你来说不公平，因为你已经离开我的公司，因为江俞轩已经和我没有任何关系了，但是俞轩，我真的没有可以托付的人，我只有你！”陈子昂的一句话诛了江俞轩的心又暖了江俞轩的心。
<br/>
<br/>        江俞轩是进不得退不得，拒绝陈子昂，陈子昂会说他们已经没有关系了，不拒绝陈子昂，他心里又不放心。
<br/>
<br/>        “我好难！”江俞轩说。
<br/>
<br/>        “我知道让你为难了，前几年都过来了，何况现在呢？俞轩，甸城的信息化可能得三年，三年我就回来了，况且我只是去赚钱，又不是常驻，还是会经常回来的，我保证两个月回来一次。”陈子昂开始央求江俞轩了。
<br/>
<br/>        “陈子昂，我发现你现在脸皮越来越厚了，只要能达到目的无所不用其极，就差一哭二闹三上吊了。”江俞轩如何能拒绝陈子昂那苦苦的央求？
<br/>
<br/>        “在你面前我啥时候脸皮薄过？”陈子昂毫不犹豫的自黑了一把。
<br/>
<br/>        “那说好了，三年你就回来，这三年里你两个月回来一次，回来要和我在一起，最起码得呆一个礼拜。”江俞轩提出了他的要求，给他一段时间，他解决了和张倩楠的事情，他就将陈子昂追回来。
<br/>
<br/>        “小ks，我肯定要经常回来的啊，大本营在这里，亲人都在这里。”陈子昂欢快的答应了，只要江俞轩不去甸城趟那趟危险的浑水就好。
<br/>
<br/>        除开她的亲人，江俞轩就是陈子昂最在乎的人了。
<br/>
<br/>        虽然江俞轩说他喜欢陈天天，但是不妨碍江俞轩在陈子昂心里所占的重要位置，他喜欢谁，他爱上谁都是他的自由，只要他一切安好，对陈子昂来说，不在一起又何妨，何况他们根本就没有未来。
<br/>
<br/>        你是我心里的一道光，一直陪着我走过漫长的人生，那些痛苦的彷徨的、快乐的忧伤的日子都有你在身边，那一起奋斗过的足迹将是我人生最美好的时光。
<br/>
<br/>        我多想与你一起并肩两人行，笑看残阳沉西方，我多想与你一起走过阡陌赏寒梅，我多想与你一起执壶烹茶香，品飞雪三两行，我多想与你再看四季花，醉谈路何方。
<br/>
<br/>        可是，这都是我心里最隐秘的梦想，你的安好将会是我最喜欢看到的阳光。
