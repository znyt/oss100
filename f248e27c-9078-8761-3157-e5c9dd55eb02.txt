　　主神空间中，已显现无上功法，看得众人眼花缭乱，简直亮瞎周遭众人的双眼啊。
<br/>　　上到盘古开天辟地之无上天功，下到众圣人所创的天经，再到武侠世界的各种功法，最后到基础内功心法。
<br/>　　价值从九百九十九万亿，九十九万亿，九万亿，九亿……到九万，九千，九百，九十，一个价值点，琳琅满目。
<br/>　　简直要逆天啊！
<br/>　　尽管这上面的东西，夜邪等人九成九以上的兑换不了，但是不妨碍他们欣赏啊，毕竟这些东西看上去就感觉高大上档次！
<br/>　　同时，他们也在心中想到，这个混沌神殿，到底是什么存在，就连盘古及道祖鸿钧，还有几位天道圣人的功法都有？
<br/>　　各种各样的功法，成万上亿，看得众人眼都花了。
<br/>　　良久，夜邪才深吸一口气，强忍着心中的不舍，离开“法”门，点开“术”门。
<br/>　　开天八式，盘古开天斧道神通，价值90万亿价值点。
<br/>　　元始九印，元始所创神通印法，价值90万亿价值点。
<br/>　　混沌九技，价值50万亿价值点。
<br/>　　……
<br/>　　盘古大力神通，盘古所创大神通，价值9万亿价值点。
<br/>　　万道森罗，魔祖罗睺所创神通，森罗万象，价值9万亿价值点。
<br/>　　大梦万古，准提圣人所创梦幻神通，沉沦万古梦幻，似虚似实，无法分清，价值9万亿价值点。
<br/>　　九天神技，武祖玄天所创神技，价值9万亿价值点。
<br/>　　天罡三十六变，价值9亿价值点。
<br/>　　地煞七十二变，价值5亿价值点。
<br/>　　《葬剑诀》，至高剑诀之一，价值9亿价值点。
<br/>　　《乱剑诀》，至高坚决之一，价值9亿价值点。
<br/>　　万龙炼界大仙术，武祖玄天所创，价值9亿价值点。
<br/>　　化虹之术，身法神通，化为一道飞虹，瞬之间息，便是百万里，价值9亿价值点。
<br/>　　咫尺天涯，一步虽咫尺，踏过便天涯，身法神通，价值9亿价值点。
<br/>　　……
<br/>　　风刃术，价值100价值点。
<br/>　　火球术，价值100价值点。
<br/>　　……
<br/>　　众多神通法术依旧看得夜邪眼花缭乱，恨不得全部据为自己所有。带着浓浓的不舍，那依旧火辣的眼光，他看向“阵”门，顿时又是一阵惊叹。什么周天星辰大阵，什么都天神煞大阵，什么诛仙剑阵，什么九曲黄河大阵，什么万仙大阵，什么十绝阵，简直不要太多了。
<br/>　　一元两仪三才四象五行六合七星八卦九宫十方，演化出各种千奇百怪的阵法，让人看了眼花缭乱，但夜邪并没有在阵法一项上多耽搁时间，他对阵道并不怎么感兴趣。
<br/>　　来到“器”门，再次亮瞎了他的钛合金双眼，各种先天神器，让他垂涎不已。如：
<br/>　　开天神斧，混沌珠，玄黄珠，太极图，盘古幡，混沌钟，诛仙四剑，五方五色旗，八卦炉，羊脂玉净瓶、芭蕉扇，七星剑，紫电锤，混元拂尘，阴阳两一灯，轮回境，通天碑，翻天印，日月珠，多宝塔，玄黄玲珑宝塔，昆仑镜，指天剑，伏羲琴，神农鼎，昊天塔，太一轮，女娲石，洞天珠，锁妖塔，聚宝盆，罗天网，定天簪、幽天台，索天栓、封天门，震天塔，翻天印……
<br/>　　从顶级开天神器到先天灵宝，后天灵宝，再到一般法器，灵兵，夜邪已经看得麻木了。当然，他并不知道，这些都是混沌道君模仿洪荒的灵宝炼制出的神器，而不是真正的洪荒先天灵宝。当然，就算不是，威力也非比寻常了，比这个世界的帝兵要强上许多。
<br/>　　最后，他再依次到了“药”门与“杂”门，好多东西，有了先前的宝物。他已经看得麻木了，看到九转金丹什么的，什么天武破虚果，什么紫霄雷神果，什么三生花的，他已经见怪不怪了。倒是杂门不愧是杂啊，什么乱七杂八的东西都有些，从最常见的到稀奇古怪的东西，应有尽有，可谓包罗万象。
<br/>　　深深呼出一口气，良久夜邪才恋恋不舍地回过心神来，意念一动，他才开始思考自己如今所要兑换的东西。
<br/>　　“主神，探查一下我的天赋。”夜邪在心中默念一声，他想要看看自己是否有什么天赋，之后才可以相应的选择自己今后的道路。
<br/>　　“叮！需花费一百价值点，是否进行！”
<br/>　　夜邪犹豫了一下，便选择了“是”，虽然他价值点不多，但为了未来，这点价值点还是可以花费的。
<br/>　　“叮！预备子弟夜邪，拥有真魔体，天生对魔道法则亲近，修炼魔道功法，速度加成。”
<br/>　　嘶！
<br/>　　夜邪倒吸一口气，他没想到，自己还真有特殊体质，而且还是真魔体，虽然他不知道什么是真魔体，也不知道它的威力如何，比荒古圣体如何，但他却知道何为真魔。
<br/>　　真魔，乃是一种与真仙相对的种族。而这真仙不是修炼到真仙境界的仙人，而是真正的仙族。仙族，天生体内便具有仙道法则，一出生即为真仙，成年为金仙，大成为大罗金仙，修炼速度极快，带有仙光神通，与大道法则亲和。有说，仙灵体即为真仙体的弱化版，当然，是真是假，便无从可知了。
<br/>　　而真魔，传说便是与真仙族敌对的种族，同样一出生极为真魔境，大成为大罗金仙境，天生与魔道法则亲和。
<br/>　　夜邪想到，真魔体恐怕就是与真仙体类似的体质，不过，不知道什么原因，他到如今也并没有真仙境界的修为，也就是遮天界中五圣人境界。其实，他的魔道本源早已被混沌道君给斩去并封印到身体之中，只等他重新修炼到仙五圣人境界才能解开封印。这就相当于废除修为，重新来过，如无始大帝当年一样。
<br/>　　“我如今只有八千二百价值点，想要兑换一本像样的魔经恐怕都不成，还是实际点吧。”夜邪微微苦笑道，价值点太少，看着如此多宝物却不能兑换，真是不爽。
<br/>　　夜邪回忆一下原著中的剧情，之后再结合自身情况，兑换自己如今所需要的东西。
<br/>　　“功法我一时之间还不能修炼，所以，还是想想其他吧。”
<br/>　　一阵思索，为了方便，夜邪花费两千价值点，兑换了一个隐形储物戒指。戒指只要带到手中，滴血认主之后，便谁也看不见。对于刚开始踏上修炼之路的修士来说，这是必备神器。
<br/>　　之后，他又用五千兑换了一颗小智慧丹，增加自己的灵魂修为。因为他知道，法力易修，灵魂难练。虽然小智慧丹的效果远远比不上真正的智慧丹，但依旧可以让他的头脑变得清晰无比，以后修炼功法将会是一种无形的助力，因为这是增加悟性方面的底蕴。
<br/>　　最后，他只剩下一千二的价值点，他不敢兑换，因为价值点低于一千的，将会被抹杀。
<br/>　　随便利用点价值点兑换点吃的后，夜邪便不再兑换其他的。
<br/>　　在夜邪兑换完后，其他几人才陆续从震惊之中醒悟过来，之后他们便是陆续一阵兑换，不过，因为价值点很少，大家都兑换不到真正的宝物。
<br/>　　不过，一些基础的仙法与灵兵，或者武功心法，还是能够兑换的，也让大家过了一把武侠高手的瘾。
<br/>　　转眼便过去了七八个小时。
<br/>　　……
