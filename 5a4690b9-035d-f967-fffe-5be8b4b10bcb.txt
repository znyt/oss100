　　“就差那么一丁点，气不气？”叶少阳脸上仍然保持着微笑，出言刺激陈晓宇。
<br/>　　陈晓宇怒从心头起，对叶少阳展开了疯狂的进攻，但仍然总是棋差一招，这一次，连鬼婆等人脸上都出现了一丝异样的表情。
<br/>　　“少阳，少阳啊，卧槽可算找到你了，你大哥我来了！！”
<br/>　　山口方向，传来一声遥远的呼喊，听在叶少阳耳中，是那么的熟悉。
<br/>　　叶少阳浑身一颤，转头看去，一个和尚，还有一个白眉毛的道士，一路狂奔过来。
<br/>　　“宝爷！！嘉伟！！”
<br/>　　凤兮已经先叫了起来。
<br/>　　是四宝和吴嘉伟！
<br/>　　叶少阳浑身颤抖，咧嘴笑起来，刚要开口，突然觉得肚子上一麻，低头一看，是陈晓宇一拳打在了自己肚子上。
<br/>　　但是并不是拳头打上去的那种疼啊，叶少阳一抬手，将陈晓宇推了出去，低头看，伤口的位置，插着一把好像匕首的东西，刀币。
<br/>　　叶少阳立刻就明白了，这是磨尖的刀币，是法术界常用的一种防身法器，有尺二、尺八等等不同尺寸。陈晓宇用来插自己这个，是“尺二”。但这东西是对付邪物时，万一被贴身的时候，用来防身的，从来都是不能对人用的。而且……叶少阳把尺二拔出来，上面一片翠绿，摸着冰凉，道：“有毒？”
<br/>　　“淬骨寒冰，半刻之后，你将浑身痉挛、倒地失语，只有我的法术能解，叶少阳，我也是用心良苦啊这是，免得你输了赖账。”
<br/>　　“我不赖账。”叶少阳轻轻一笑，“你活不了了。”
<br/>　　话音刚落，四宝和吴嘉伟一起冲了过来。
<br/>　　他们才赶到桃花山不久，听说山上今晚举办婚宴，所有法师都参加，心想如果叶少阳在的话，说不定也会来参加，这才一路上山，离老远就一眼看到了叶少阳跟人在斗法，叶少阳节节败退，他们不明情况，自然要上去帮忙，也来不及跟叶少阳打招呼，立刻就奔着陈晓宇去了，正好又看见陈晓宇偷袭，顿时怒不可遏，起了杀心。
<br/>　　“狗日的，看老子灭了你！”
<br/>　　四宝怒吼一声，跟吴嘉伟一前一后，夹攻陈晓宇。
<br/>　　“少阳，弄死他吗？”四宝问道。他不知道陈晓宇是谁，也不需要知道。得罪了叶少阳，这个理由已经足够让他们下杀手了。
<br/>　　“你们是什么人，这是两人决斗！你们要是不退下，我们就一起上了！”
<br/>　　说话的是一个法术公会的弟子，招呼十几个师兄弟一起冲了过来。
<br/>　　四宝和吴嘉伟视若罔闻，为叶少阳，别说是这么几个人，就算是整个法术界一起来，也是全然无惧，有死而已！
<br/>　　“你们先下去，听我的。”叶少阳不想这就引起大规模的械斗，喝令四宝和吴嘉伟退下。
<br/>　　两人听叶少阳这么说，就知道他胸有成竹，也不多说，立刻就退下了。
<br/>　　“宝爷你们这边来！”凤兮大声招呼。
<br/>　　“少阳，待会再说啊！”四宝兴高采烈地拍了拍叶少阳的肩膀，吴嘉伟也是一脸激动。两人似乎都忘了几秒钟之前还跟人打斗，而且把人晾在了一边。
<br/>　　叶少阳比他们还要激动，在这里看到两位兄弟，这真是天大的喜讯。
<br/>　　有你们在我身边，我什么都不怕……
<br/>　　“你们先去，一会有你们发挥的时候。”
<br/>　　四宝和吴嘉伟也不多说，朝凤兮那边走了过去。
<br/>　　凤兮为他们介绍，当得知眼前这个是年轻版的道渊真人和现实版毛小方的时候，四宝和吴嘉伟都凌乱了。
<br/>　　这突然杀出来两个人，围着自己打了一会，突然又走了，陈晓宇现在是懵逼状态，望着叶少阳，咬牙冷冷说道：“叶少阳，你个无赖，众目睽睽之下，你居然还找帮手！”
<br/>　　“所以我让他们走了，你继续吧。”叶少阳对他招了招手。
<br/>　　“你……为什么没有毒发？”陈晓宇低声说道。
<br/>　　叶少阳笑了笑，没有跟他解释自己是先天灵体，百毒不侵，一个纵身非扑了上去——单手捏诀，对着陈晓宇拍过去。
<br/>　　“来得好！”陈晓宇也迎了上去，两人缠斗在一起。
<br/>　　张晓寒站在花轿边观战，转头看了一眼，梧桐居然把盖头掀开了一角，正在观战。
<br/>　　“喂，你现在不能掀起盖头！”
<br/>　　被张晓寒一吓，梧桐又把盖头放下了。
<br/>　　“晓寒，你答应我一件事，然后，我会毕生感谢你。”梧桐的声音，从红盖头里传来。
<br/>　　张晓寒心中一动，忙附身下去，问道：“什么？”
<br/>　　“阻止这场战斗，不要让少阳在这么多人面前出丑……”
<br/>　　张晓寒愣了一下，轻轻一笑，说道：“你舍不得他当众出丑是吗？”
<br/>　　“你说什么！”
<br/>　　“我知道的，那么，你就看着他跪地求饶吧。”张晓寒随手把花轿的帘子也放了下去。
<br/>　　那边，战斗还在继续。
<br/>　　之前，叶少阳故意保存实力，一方面是想戏弄陈晓宇，不过更多也是为了不让别人（主要是张晓寒）看穿自己的实力，弄出一个险胜的局面，反正只要能赢，怎么赢都是一样。
<br/>　　但是现在，在中了陈晓宇的毒针之后，叶少阳也是有些恼火了，出手也开始狠辣起来。
<br/>　　陈晓宇自诩是准地仙境界的强者，面对叶少阳的强攻，也硬气地抵挡了一会，但很快就发现自己完全抵挡不住，被逼的节节败退。
<br/>　　在场的观众，被这一幕变化惊得目瞪口呆，一个个瞪大眼睛观察着战局，现场只有叶少阳和陈晓宇打斗的声音，以及……吧唧嘴大口吃饭的声音。
<br/>　　“喂，我说服务员，不对，这位小哥，这是阳春面吧，来来，再给我来一碗！”
<br/>　　这个时候，居然还有人能安心吃饭？
<br/>　　众人忍不住转头看去，是那两个突然闯进来的叶少阳的帮手，一个和尚一个道士，趴在桌子上各种猛吃。关键是……那和尚居然碰着一个大肘子在啃。
<br/>　　连坐在一边的妙心都看不下去了，拍了他一下说道：“喂，大和尚，你真吃的下去啊，你怎么好像不关心少阳能否打赢？”
<br/>(本章完)
