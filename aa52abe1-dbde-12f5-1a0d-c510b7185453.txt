　　王家老店是一家酒铺，店主人很好，乐善好施，虽然店不大赚的钱也不多，但却在周围的街坊邻居中赢得了一个王善人的名声，这便是唐义在王家老店附近一家浴池洗澡时探听到的消息。
<br/>　　搞清报工作的人往往都很小心，唐义自然也不例外，为了保护自己的安全他习惯性的先对自己己要去的地方进行了一些了解，而后才以一种试试看的态度来到了王家老店，打了两角烧酒，用手提着回了称心的府上。
<br/>　　这一切的一切在外人看来都是那么正常，似乎所有的一切都是巧合，但就是在这样的巧合之下，一份情报被唐义在不知不觉间送了出去。
<br/>　　……
<br/>　　一天半之后，远在山西的王家老宅收到了来自远方的情报，几个族老在打开看过之后，一个两个面色变的十分难看。
<br/>　　其中一人狠狠一拍桌子说道：“这可真是岂有此理，竟然如此威胁我王家，这隐王的后人竟如此无理，难道是欺我王家无人不成！”
<br/>　　“老三，稍安勿躁！”另外一个坐在其上首的老者对他虚按了下，扫了一眼众人说道：“这是隐王一脉在向我们要投名状了，诸位觉得我们是否应该做出正面回应？”
<br/>　　“二哥，此事我觉得还是应该慎重考虑，想当初隐王一脉找到我们的时候只是说需要为其提供一些方便，并没有提到其他。”一个坐在靠门位置的老者双眉紧锁，显然是并不赞成家族与隐王一脉的人有过多的交往。
<br/>　　“八弟，从感情上说，我这个当哥哥的支持你的观点，但是你也不要忘了，现在的皇帝陛下对我们世家一直很有看法，虽然这几年他已经偃旗息鼓，但谁知道未来的某一天会不会卷土重来？”
<br/>　　“那六哥的意思是什么？”那个‘八弟’扭头看向左侧。
<br/>　　在他的左侧坐着一个独臂人，头发已经花白，但面色却十分红润，如果不看那一头的白发怕是只会将其当成四十岁左右，显然此人就是‘八弟’口中的‘六哥’了。
<br/>　　“我的意思是不如暂时与那隐王一脉合作，给他一些支持，若是装来他们能够成就大事……”
<br/>　　“六哥以为这可能么？以当今圣上的能力，你觉得隐王一脉能够成功？退一万步说，就算是他们成功了，又拿什么来保证一定不会动我们王家？”
<br/>　　“至少这也是一个机会，否则我们只能坐以待毙，看看外面的世界吧，我们的祖产还剩下多少？整个家族的产业都被迁到了辽东了，那里是什么地方你不会不知道吧？”
<br/>　　“那又如何？皇帝陛下对我们动手只是你的猜测，你完全是拿家族的前途在赌。”
<br/>　　‘八弟’与‘六哥’针锋相对，谁也不让谁，你一言我一语争论不休，而其他人坐在一边，就那么静静的看着他们，各自想着自己的心事，以及这件事情的利与弊。
<br/>　　终于，坐在首位的那个‘二哥’开口了：“够了，都不要争了，这件事情我支持老八的决定，老六，你的意见保留吧！”
<br/>　　“二哥，为什么？难道你真觉得李承乾不会对我们下手？他父亲的时候就一直在针对我们，现在到了他这一代更是变本加厉，你如何保证他以后不会再对我们动手，若是到了那个时候，我们一无盟友，二无援兵……”
<br/>　　“老子说够了！”‘呯’的一声，首位上的‘二哥’狠狠在身边的桌子上拍了一下：“老六，不要忘了你的身份！”
<br/>　　看到除了族长之外最大的‘大佬’发火，一时间所有人都为之肃然，那‘老六’也面色不愉的低下了头：“二哥，是我冲动了。”
<br/>　　“老六，子木是你的孙子，你担心他的安全我很清楚，可是我们不能为了一个人的安全而搭上整个家族。”首位上的‘二哥’见老六低头了，长长叹了一口气，语重心长的说道：“子木那孩子也是我看着长大的，我也不想他出现任何问题，所以你放心，族中一定会派人与隐王一脉好好沟通，争取在这件事上达成一致，努力保证那孩子的安全，你看如何？”
<br/>　　“二哥，话既然说到这个份上，我只能保留意见，只是希望……”
<br/>　　老六的话未说完，房间中的几人便听到一阵急促的脚步声，就在众人一愣看向门口的时候，便见到一个家中小厮跑了进来，气喘吁吁的嚷道：“二，二爷，大事不，不好了，有，有人送，送来了一，一封信，还，还有七，七少爷的，的……。”
<br/>　　“七少爷的什么？子恒怎么了？！”刚刚还在与老六争论的‘八弟’一下子站了起来，厉声喝问道。
<br/>　　小厮被那‘八弟’吼的一个哆嗦，退了两步颤抖着说道：“还，还有七少爷的尸，尸体！”
<br/>　　“什，什么？”听到小厮的话之后，‘八弟’瞬间就呆住了，良久才发出一声苍然的呐喊：“子恒，老夫的子恒啊！是谁，到底是谁下的手，为什么要对老夫的独孙下手啊！”
<br/>　　这突如其来的变故几乎惊呆了房间中的所有人，直到‘八弟’的一声苍凉呐喊才将众人惊醒。
<br/>　　‘二哥’眼见着‘八弟’已经失控，连忙命人将其扶了下去，同时将那小厮叫到面前：“说，到底怎么会事，子恒是怎么死的。”
<br/>　　“回二爷，七少是被人送回来的，送回来时胸口插着一把匕首，上面，上面还穿着一张字条。”小厮显的很是紧张，但好歹还是把事情说明白了。
<br/>　　‘二哥’听完了小厮的回禀，神情变的异常难看，一连串又问了好几个问题：“字条？什么字条？子恒的护卫呢，他们哪里去了？”
<br/>　　“二爷，字条就是这个！”小厮在‘二哥’的追问下，从怀里掏出一张被血染红的纸，递到了‘二哥’的手中，同时继续说道：“至于七少爷的护卫，他们也全都死了，全都是一刀毙命，尸体是同七少爷的遗体一起送回来的。”
