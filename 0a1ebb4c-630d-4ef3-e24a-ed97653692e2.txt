　　第266章
<br/>　　周兴于西，封国八百载；秦兴于西，一统神州。
<br/>　　西北自古以来就是王兴龙出之所在。
<br/>　　程大雷过着刀口舔血的生活，对于风水之术也没办法不信，所谓玄能救非，氪金改命么。
<br/>　　还有一个原因，崔相派出大批杀手来暗杀程大雷，为的就是不让程大雷入琴川，如果单单是为了替杨龙停报仇，也不必花这么大力气。
<br/>　　琴川究竟有什么？
<br/>　　如此，程大雷倒是真想去西北琴川看看？
<br/>　　“不管大当家去哪儿，我们都跟着一起去。”众人都如此表示。
<br/>　　“不着急，琴川还不知是什么情况，大家一起去，也没有那么多粮食养活，现在是一动不如一静。”程大雷道：“大家最近都过得怎么样？”
<br/>　　高飞虎高飞豹当初带着五百弟兄护送山寨老弱去了南方，如今仍然是占山为王。而如熊大熊二，小白狼，廖甲廖乙手上也都有了一票兄弟。
<br/>　　仔细一算，原先的蛤蟆寨虽然散伙了，可又分出十几家山寨，所有弟兄加在一起，足有五千余人。
<br/>　　程大雷对这还是蛮满意的，一年过来，如高飞虎，高飞豹，熊大熊二等人过得其实还称得上不错。他看着一张张熟悉的面孔，心底忽然发出一声叹息。
<br/>　　少了林少羽。
<br/>　　不知他现在情况如何，他若心生厌倦，择一山清水秀之地，老死江湖，在这样一个乱世，也未必不是一件好事。
<br/>　　“大家暂且按兵不动，以后说不得我还有用到你们的时候。”程大雷道：“等我在琴川混好后，大家再去琴川投奔我。”
<br/>　　诸人分开一年，好不容易相逢，都不愿再次分别。可程大雷已经说得很明白，琴川还不知是怎样一个情况，所有人都过去，连个落脚点都没有。况且，大家已经知道程大雷的去向，日后想找他也不困难。
<br/>　　“大当家，你总不能一人过去，还是多带几个弟兄，不管到哪儿也都有个照应。”高飞虎。
<br/>　　程大雷点点头，高飞虎说得也有道理。于是就在大部队中点了几个人，秦蛮，徐神机，徐灵儿，再加上刘关张赵，高飞豹等诸多人，凑够一百余人。
<br/>　　众人难得相遇，却又得再次分离，不过都是生死场走过来得弟兄，即便不舍，也只是付之一笑。
<br/>　　诸人化整为零，连夜入关，程大雷与诸人分别后，带着一百弟兄前往西北琴川。
<br/>　　程大雷现在的情况，就像游戏中的销号重练。第一次进入游戏，往往随心所欲，练着练着练岔了，发现敏捷不够，体力不够，一个法师不加智力加力量，于是只好砍掉重练。第二次进入游戏，就要认真些，主修什么副修什么，侧重向那个方向发展，心里都得大概有个盘算。
<br/>　　实话实说，青牛山的基础盘相当不好，山地多于田地，气候又太过寒冷，没有太大发展前途，所以就算舍掉，程大雷也并不觉得可惜。
<br/>　　而第二次选择基础盘，则一定要慎之又慎。重新开始，总是能重新带给人希望。
<br/>　　在入关之后，程大雷便带着手下弟兄，一路西行，前往琴川。
<br/>　　而在这个过程，帝国之内还发生了一些事情。
<br/>　　鱼龙卫将明玉公主被劫走的消息禀报给明帝，这一次，本来是抱着掉脑袋的危险，因为有时候明帝很像一个暴君的。
<br/>　　只是没料到，明帝在知道明玉公主被程大雷劫走之后，只是长叹三声，却并没有惩罚任何一个人。
<br/>　　明玉公主被劫走，两族的和谈自然做不得数，朝廷在边关布下大军，随时防备戎族的入侵。只不过，除了一些小部落的骚扰外，戎族并未大举侵犯。
<br/>　　关于戎族的消息，帝国方面知道得很少。呼延跋一死，戎族上层属于权力真空状态，有了权力真空，自然需要新的势力进入。于是戎族各王子，各部落便进入权力斗争状态。
<br/>　　这个时候的他们，自然顾不得侵略帝国。
<br/>　　而帝国方面的情况也极其堪忧，帝国一败再败，皇权的威信早已荡然无存，各诸侯蠢蠢欲动，在民间也有新兴势力崛起，暗中窥伺着那天下至尊的位置。
<br/>　　而对于这些，程大雷并不了解，甚至谈不上关心。
<br/>　　他只是继续向西，一路长途跋涉，琴川已快到眼前。
<br/>　　“大当家，再有二十里路，咱们就能到琴川关了。”徐神机看着远处隐约的山峦道。
<br/>　　“是啦，是啦。”程大雷忽然想起什么，道：“我的官服呢，等我换上官服入关。”
<br/>　　“这个……至于么？”徐神机道。
<br/>　　程大雷抱着官服走进树林中换衣服：“怎么不至于，琴川不知有多少兵，按道理都是我的兵，如果我们按山贼进去，他们可是会进攻我们的。”
<br/>　　程大雷一边换衣服一边回答，林静草寂，周围一只鸟叫都没有。
<br/>　　程大雷忽觉得有些不对，林子太安静了，这种情况只有一种解释，是谁提前埋伏，把林鸟都惊走了。
<br/>　　正寻思着，一根套索从空中落下，拴在程大雷身上，一矫健的人影从树上跃下，借着坠力把程大雷倒吊在半空中。
<br/>　　等程大雷回过神来时，见面前多了两个人，皆是蓬头垢面，衣衫褴褛。
<br/>　　“呔，此山是我开，此树是我栽，要想从此过，留下买路财。”一个身型略高的人哇哇大叫着。
<br/>　　程大雷心底哭笑不得，终日打雁却被雁啄了眼。
<br/>　　“还差一句？”程大雷。
<br/>　　“差一句，差什么？”
<br/>　　“牙嘣半个不，管杀不管埋啊。”
<br/>　　程大雷忽然喝了一声，身在半空中一个挺身，猛地一挣，全身骨骼发力，将绳索挣断。
<br/>　　二人一见这一幕，同时都睁大了眼睛，脸上露出错愕表情。
<br/>　　程大雷把衣服穿起来，笑笑道：“那条道上的，连我都敢劫，不认得你们这一行的祖宗么？”
<br/>　　两个毛贼还没反应过来，秦蛮和赵子龙已经潜入树林，轻而易举的将两人制服。
<br/>　　“喂，别打，别打。”一个毛贼大叫着：“我认得你了，程大雷！”
<br/>　　程大雷刚换好衣服，还没认出对方，李婉儿闻声音过来，也是一愣。
<br/>　　“六哥？”
<br/>　　“小妹？”
<br/>　　
<br/>(本章完)
