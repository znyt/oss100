        说到“那个男人”的时候，童筱颖脑海里忽然想起了傅夜沉。
<br/>
<br/>        她躲了他三天，冷落了他三天，他每次走到她房门口时，门底下的那条缝隙就会变暗，在他驻足片刻离开后，门底下那条缝隙又会恢复光亮。
<br/>
<br/>        “小卫，你……真的受不了，我冷落你吗？”童筱颖话中有话地问。
<br/>
<br/>        傅昼景苦笑：“如果能再给我一次机会，我一定能受得住你的冷落。”
<br/>
<br/>        要说起“开冷战”的胜率，她童筱颖绝对是百分百的胜率。
<br/>
<br/>        她可以做到，一个电话不给，与他分房睡，哪怕是同住一个屋檐底下也可以跟他不打照面。
<br/>
<br/>        比起开冷战，他宁愿她跟他大吵大闹，至少闹到最后、吵到最后，她会先低头跟他妥协。
<br/>
<br/>        但如果她跟他开冷战的话，他若不主动打破这冷战的僵局，她就会永远陷入了这冷战里，对他不闻不问，甚至当他是空气。
<br/>
<br/>        以前，他会为了这事苦恼、心烦，甚至觉得她枯燥无趣。
<br/>
<br/>        现在，他没有机会再为了她而苦恼、为了她而心烦，再认为她枯燥无趣了。
<br/>
<br/>        但是此时此刻，童筱颖心里想的是，傅夜沉会不会也受不住她的冷落？
<br/>
<br/>        她这个坏毛病是不是要改改？
<br/>
<br/>        可是，其实，她每次陷入冷战里的时候，只是需要人哄哄就行了。
<br/>
<br/>        小卫不愿意哄她，就连傅夜沉只怕也不愿意吧！
<br/>
<br/>        毕竟他俩是兄弟，性子应该也差不多。
<br/>
<br/>        不然，这三天的时间里，傅夜沉为什么每次只是停驻在她的房门外，而不主动跟她说句话？
<br/>
<br/>        等等！她怎么想到傅夜沉身上去了！！
<br/>
<br/>        童筱颖立马拉回自己的思绪，拿着锅铲翻炒着锅里的菜。
<br/>
<br/>        一盘干煸四季豆，一盘鱼香茄子，两道素菜端上了桌。
<br/>
<br/>        他和她坐在了灶台旁的朽木餐桌前，各自端着各自的白米饭，安安静静地吃饭。
<br/>
<br/>        此情此景，好像又回到了那时年少。
<br/>
<br/>        外婆坐在他俩中间，她和他面对面而坐，高高兴兴地吃着外婆做的饭菜。
<br/>
<br/>        现在，他俩长大了，再次面对面坐在这张朽木餐桌前，却一点也不像是已经离异的夫妻，反而像是许久未见的亲人。
<br/>
<br/>        “筱颍，你恨我吗？”傅昼景深沉地问。
<br/>
<br/>        童筱颖点了点头，又微微摇了摇头：“因爱生恨是有，但是……想通了，就不恨了。或许是因为懒得去恨了吧！没意义，而且对自己也不好。”
<br/>
<br/>        与领完离婚证后的那些天，她确实恨过他，躲在这里每天以泪洗面，后来为他心痛过了，就麻木了，为他把眼泪也流干了，还是改变不了他已经和别的女人有了孩子的事实。
<br/>
<br/>        所以，她想通了，不恨了，却仍旧希冀着他能回到她的身边。
<br/>
<br/>        但是，这样的希冀，被季思妍的那封电子邮件给彻底打破。
<br/>
<br/>        那日，在雨中，透过婚纱店的落地窗，她看到他陪着季思妍带着小尚轩在挑选婚礼礼服，那一刻，她的心碎得彻底。
<br/>
<br/>        在也不知不觉中，封闭了自己的心。
<br/>
<br/>        受过伤的女人，总是不会再轻易地交出自己的真心，对感情有所抵触。
<br/>
<br/>        她抵触的不仅仅只有傅夜沉，更多的是他傅昼景。
<br/>
<br/>        “你是想告诉我，没有我，你也能过得很好，是吗？”傅昼景垂着眸子，不敢直视童筱颖的眼睛。
<br/>
<br/>        童筱颖慢悠悠地往嘴里扒饭，其实她的肚子也饿了，但是饭到嘴里却如同嚼蜡。
<br/>
<br/>        她并不是因为傅昼景的话而影响到了胃口吃饭，而是心里总觉得哪儿有点慌，但又说不出来哪儿不对劲。
<br/>
<br/>        “筱颍，你在听我说话吗？”见童筱颖半天不吭声，傅昼景加重了语气，提高了嗓音。
<br/>
<br/>        童筱颖这才回过神来，应道：“我会过得更好。”
<br/>
<br/>        顿时，傅昼景沉默了。
<br/>
<br/>        经过三天的时间沉淀，傅夜沉终于不淡定了。
<br/>
<br/>        她倒是可以做到，三天对他不闻不问，甚至避而不见。
<br/>
<br/>        但是，他却做不到。
<br/>
<br/>        这三天的时间里，于他而言，简直是度日如渡年。
<br/>
<br/>        今天，他非得把那丫头从房间里“抓”出来不可！
<br/>
<br/>        回私宅前，他开车特意绕道去了花店。
<br/>
<br/>        花店里的老板娘，看到一辆豪车停在自己花店门口，只见一个比电视里的男明星还要英俊帅气的男人从车上下来，于是笑盈盈地亲自出门来迎接。
<br/>
<br/>        “先生，您是买花送女朋友吗？”老板娘走到傅夜沉的跟前，十分殷勤地询问。
<br/>
<br/>        “送老婆的。”
<br/>
<br/>        “是生日还是纪念日？”
<br/>
<br/>        “我老婆她……”傅夜沉欲言又止，突然间不知道自己该说什么好。
<br/>
<br/>        他没谈过恋爱，没经验，面对筱颖突然冷落他，他这几天一直不知所措。
