　　从这老者的胸口之上，一道灰色影子清晰可见，这一道灰色影子，仿佛是一道利刃一般从，刺入到了这老者的胸口之中，不断的损耗这老者的力量。
<br/>　　在这一道灰色影子之中，蕴含着狂暴毁灭。
<br/>　　这其中蕴含的力量，让一尊半步天人境强者，稍微沾染，都会落入万劫不复的下场。
<br/>　　纵然是以这老者实力，也只能够是借助一点点这宫殿之中所蕴含的混沌气息，才可以压制住身上的伤势。
<br/>　　可终究，只是苟延残喘罢了。
<br/>　　这老者身上的伤势，他人不了解，可楚风眠却是一清二楚，那一道打入老者身躯之中的灰色影子到底是什么，楚风眠也一清二楚。
<br/>　　混沌之雷。
<br/>　　曾经楚风眠在度过逆天大劫的时候，就承受过这混沌之雷的轰击。
<br/>　　这老者的身上，也有着混沌之雷，在加上他身上蕴含的一点点龙之本源，楚风眠便是看的出来。
<br/>　　眼前这一位老者，也是曾经试图想要度过逆天大劫的人物，也想要逆天而行，成为天地异种。
<br/>　　可惜这老者却是跟楚风眠的结局不同。
<br/>　　楚风眠成功了，而这老者，却是失败了，那一道混沌之雷，打入到了这老者的身躯之中，随时随刻都在磨灭着他的生命。
<br/>　　如果不是这老者寻找到了这样一处蕴含混沌之气的宝地，现在已经是陨落了。
<br/>　　“躲过逆天大劫不死，也算是一代传奇了。”
<br/>　　楚风眠看着老者，不由的说道。
<br/>　　楚风眠的话并非是假话，不算恭维。
<br/>　　逆天大劫，敢当这逆天二字，便是代表着想要度过逆天大劫的难度，到底是有着多高。
<br/>　　几乎是十死无生的劫难。
<br/>　　在历史之中，无数记载于古籍之中的妖孽天才，敢于去度逆天大劫的人物，都寥寥无几
<br/>　　单单是这种勇气，都并非是所有人可以拥有的。
<br/>　　更何况这老者，虽然没有度过逆天大劫，可却是保全了性命，这的确已经无比惊人了。
<br/>　　从历史记载之中，能够承受逆天大劫还活着的人物，根本没有几位。
<br/>　　楚风眠虽然度过了逆天大劫，可当时楚风眠度过逆天大劫的机会，也几乎是十死无生。
<br/>　　可以说如果不是最后，那吞天兽心脏，也助了楚风眠一臂之力的话，只怕是楚风眠也要死在这逆天大劫之上。
<br/>　　眼前这老者，可没有楚风眠那般的机遇，能够从逆天大劫之中活下来，此人已经是一代传奇了。
<br/>　　“不过是苟延残喘罢了，在公子的面前，更是不值一提。”
<br/>　　老者开口，语气之中有些没落的说道。
<br/>　　他度逆天大劫，也是曾经做好了万全的准备，信心满满，可最终却是失败了，虽然留下一命，可却同样留下了不可恢复的伤势。
<br/>　　余生都只能够是躲藏在这宫殿之中，本体根本不能够踏出这山峰一步。
<br/>　　一旦是离开了这山峰，这老者身上的混沌之雷，便是不在被混沌之气压制，足以让他陨落当场。
<br/>　　他本以为天下间根本没有人可以度过逆天大劫了，可没想到今日居然是碰到了楚风眠，真正度过了逆天大劫的人。
<br/>　　也让他无比好奇，想要来跟楚风眠相见一面了。
<br/>　　“如何称呼？”
<br/>　　楚风眠随意开口道。
<br/>　　“外人称我狂龙帝，不过如今我不过只是在这小小宫殿之中，苟延残喘的活死人罢了。”
<br/>　　老者，低声说道。
<br/>　　狂龙帝，这三个字便是足以看出当年这老者到底是有着怎样的狂傲气势，以狂龙为名。
<br/>　　可逆天大劫，却是打碎了这老者一切的狂傲，苟延残喘，也称不上狂龙二字了。
<br/>　　“此生能够见到公子，老朽也算是死而无憾了。”
<br/>　　狂龙帝看着楚风眠，不由的感叹了一声。
<br/>　　“你不想死。”
<br/>　　楚风眠目光看着狂龙帝，平静的开口道。
<br/>　　听到这话，狂龙帝苦笑一声，摇了摇头道。
<br/>　　“我是不想死，我若是想死，也不过苟延残喘到今日。”
<br/>　　“不过这混沌之气，也不可能一辈子压迫住我的伤势，楚公子也看得出吧，老朽这这幅身躯，已经是到了油灯枯尽的地步，不出几年时间，必死无疑了。”
<br/>　　狂龙帝的语气之中，不免有些失落。
<br/>　　“如果我有办法救你呢？”
<br/>　　楚风眠淡淡一笑道。
<br/>　　“你的寿元，其实远远未尽，只是因为因为那一道混沌之雷，导致你的生机被磨灭。”
<br/>　　“化解了这一道混沌之雷，你至少还可以活上几万年。”
<br/>　　“化解混沌之雷？”
<br/>　　狂龙帝听到前一句话，脸上本来还抱有这几分希望，可接下来听到下一句话，狂龙帝脸色便是失落了不少，苦笑的说道。
<br/>　　“老朽在这时间内，也尝试过无数次，混沌之雷，已经是跟我的身躯，跟我的魂魄，融为一体，化解了混沌之雷，老朽也会死。”
<br/>　　“就连曾经天人境的强者，曾经见过我，都对我说，此雷无法化解。”
<br/>　　狂龙帝也曾经想要尝试化解这混沌之雷，可是他耗费了一切的手段，甚至是请来了天人境的强者。
<br/>　　可最终结果，却是让他大失所望。
<br/>　　天人境的强者，都没有办法化解这混沌之雷。
<br/>　　这混沌之雷，已经是跟他融为一体，可以说已经成为了狂龙帝身上血肉一样的东西，一旦是强行化解，他也会随之陨落。
<br/>　　“天人境做不到的事，我未必做不到，我既然说能救你，自然不会失言。”
<br/>　　楚风眠随意开口道，语气之中，却是蕴含无边自信。
<br/>　　“不过我要救你，自然你也要付出代价才行。”
<br/>　　“楚公子如果能救我一命，我狂龙帝，愿意一生一世为公子所用，为公子效力。”
<br/>　　狂龙帝看着楚风眠，一字一字的说道，语气无比坚决。
<br/>　　谁能化解这混沌之雷，便是给了他新生，这种恩情，值得他一生回报。
<br/>　　“好，既然如此，那就不浪费时间了！”
<br/>　　楚风眠听到这狂龙帝一口答应下来，便是不在浪费口舌。
<br/>　　“公子需要什么准备？”
<br/>　　狂龙帝有些激动的问道。
<br/>　　对他来说，只要是有着一点点的机会，他也愿意尝试。
<br/>　　“不用。”
<br/>　　楚风眠随意开口道，仿佛是即将要做一件微不足道的小事一样。
<br/>　　楚风眠抬起手，一股灵力凝聚在了他的手心之中，一把打入到了这狂龙帝的身躯之中。
