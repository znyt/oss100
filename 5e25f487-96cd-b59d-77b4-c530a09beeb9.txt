<br/>				<br/>　　    外衣脱下，露出了胸口处那被剑刺伤的狰狞伤口，伤口处还在涌着鲜血，而因失血过多，那名佣兵已经晕死了过去，由另外一人扶着。<br/>　　<br/>　　    凤夜见夜，拧开盖子后将止血药倒在那伤口处，他倒出了很多，一瞬间止血药散盖住了伤口，血有那么一瞬间没涌出来，紧接着，他又倒了一些在手掌中，连同手掌一起按在他的伤口处。<br/>　　<br/>　　    众人看着，一时间没有说话，只感觉一颗心提了起来。当他们看见随着时间的过去，那按在胸口前的小手掌并没有渗出鲜血时，一个个不由的瞪大了眼睛，露出不可思议的神色。<br/>　　<br/>　　    “怎、怎么可能？那止血药竟、竟真的止住血了？”<br/>　　<br/>　　    他们都是在刀口上过日子的人，自然知道这伤伤到大动脉，血涌出的量大，就算是药洒下去也是止不住的，这样的伤，十有**都是必死无疑，可没想到……<br/>　　<br/>　　    凤夜按着那人的胸口，没有说话，只是一直盯着自己的手掌，直到过了一会后，他才收回手，只见，那涌出鲜血的伤口此时被那止血药止住，真的没再涌出鲜血了。<br/>　　<br/>　　    小小孩子一见，不由地朝佣兵们露出得意的笑容：“怎么样？我说了这止血药很厉害的吧？”<br/>　　<br/>　　    他侄女可是鬼医，很厉害的，这药他试过的，真的很有用，要不是看这队佣兵们拼死护着他们，他还舍不得拿出来呢！<br/>　　<br/>　　    半空中，坐在白鹤上的男子看着下方的一幕，脸上也浮现着诧异，尤其是，他看着阳光之下折射着光芒的那个熟悉的瓶子，又看了看那小孩，面色越发的古怪起来。<br/>　　<br/>　　    如果他没记错，那瓶子好像是凤九的，她的瓶子跟一般的药瓶子不一样，很好认的。可这么一个小孩，怎么会拿着凤九的止血药？还说是他侄女炼制出来的药？这样厉害的止血药分明就是凤九炼制出来的药好么？<br/>　　<br/>　　    “队、队长，真、真的止住了……”一名佣兵不可思议的说着。<br/>　　<br/>　　    徐言心头震动，他深深的看了凤夜一眼，拱手道：“还请小公子救救他们。”<br/>　　<br/>　　    凤夜因手中染了血，便将手中的瓶子递给他：“你自己喂吧！我要洗手。”他说着，一转身便跑到赵阳身边去：“阳阳，我手都是血，我要洗手。”<br/>　　<br/>　　    赵阳默默的从空间里取出水囊来，倒了一些给他洗手，见他身上也有一些被气罡划出的小伤，便道：“主子，你也受了伤，我先帮你上点药吧！”<br/>　　<br/>　　    一听这话，凤夜眨了眨眼睛，小嘴一扁：“阳阳，你不说我都没感觉到疼，你一说我感觉好疼。”<br/>　　<br/>　　    半空中的男子嘴角一抽，轻拍了白鹤一下，便从半空中下来。他走到凤夜的身边盯着他看着：“你身上只是皮肉伤而已，上了药就好了，疼不死人的。”<br/>　　<br/>　　    “谁说的？疼是会疼死人的，要不然怎么人家说话都是疼死了疼死了？”<br/>　　<br/>　　    听着孩子稚嫩的话语，男子张了张嘴，居然发现自己找不到话来说，只是摇头笑了笑：“好好好，你先上药吧！”<br/>
