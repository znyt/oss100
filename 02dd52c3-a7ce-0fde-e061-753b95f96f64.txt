　　“逸...逸儿，你刚才说什么？”三长老萧重不可置信地问了一声。
<br/>　　萧逸笑笑，打趣道，“怎么，我记得萧重叔叔您还未到耳背的年龄吧？我说，我也是一品炼药师，有问题吗？”
<br/>　　“真的是一品炼药师？”三长老还真觉得自己是耳背了，甚至以为自己幻听了。
<br/>　　“真的。”萧逸笑着点点头。
<br/>　　“真的！哈哈哈哈。”三长老顿时狂喜，甚至刚才萧逸打趣他耳背也没往心里去。
<br/>　　当萧逸点头时，全场的气氛骤然停格，只有三长老的大笑声不断回旋。
<br/>　　“好，很好。”三长老停下了笑声后，重重地点了点头，他甚至没有反应过来该如何表达对萧逸的满意，心头只涌上来‘好’和‘很好’两个词。
<br/>　　“我们走吧。”萧逸淡淡地笑了一声，同时心中蓦然地流淌过一道温暖。
<br/>　　他知道，三长老是发自内心地在喜悦。
<br/>　　全场族人中，或许都会对他展现出来的修炼天赋和炼药师资质而惊讶和赞扬，那是一种对天才的赞许和友好。
<br/>　　只有三长老，只是单纯地因为紧张关心着他，对他感到满意和放心，随后发自内心地表露出狂喜。
<br/>　　这是一种亲人间才会有的真心喜悦。
<br/>　　“给我站住。”
<br/>　　忽然，又是一声气急败坏的声音响起。
<br/>　　不用想都知道，这把让人厌恶的声音，来自五长老。
<br/>　　“五长老，还有何时？”萧逸冷冷地转过头，瞥了他一眼。
<br/>　　“萧逸，你莫要一派胡言，信口开河。”五长老怒喝道，“你也是一品炼药师？骗鬼去吧！”
<br/>　　“就是。”萧若狂不屑地讥讽道，“就凭你那微末资质，也想成为炼药师？真是荒天下之大缪。”
<br/>　　“我看，你只是想逃避责罚，随意捏造谎言罢了。”萧若狂肯定地说道，认定了萧逸在说谎。
<br/>　　这两人的话语一出，族人们，乃至长老席上的长老们都露出了疑惑的表情。
<br/>　　二长老皱眉道，“萧逸如今才16岁吧？年纪轻轻，怎么可能是一品炼药师呢，据我所知，要成为炼药师，先不说资质问题，但也需要极丰富的炼药知识吧。”
<br/>　　二长老继续道，“但，以这些天萧逸的表现，他又不像是信口雌黄的人。大长老，你怎么看？”
<br/>　　大长老乃是尊贵的三品炼药师，在这方面，他最有发言权。
<br/>　　大长老思索了一会儿，道，“要成为炼药师，的确需要极丰富的炼药知识。而且，就算有了知识，也需要研习丹方，熟练炼药步骤，还有积累炼药经验。”
<br/>　　“选材，温炉，投药，控火，凝丹，开炉....等等一系列步骤非常复杂，而且对时机的把握不可差分毫，这才能炼制成一颗一品丹药，并成为一名一品炼药师。”
<br/>　　“这种种严格的要求，导致了要成为炼药师，必须有两个先决条件。一，名师的指导，二，大量的学习时间。”
<br/>　　“但据我所知，萧逸大半个月前才从炼药堂借走了炼药录，而且家族中也没有炼药师指导过他。”
<br/>　　大长老没有给出定论，但却给出了自己的分析。
<br/>　　同一时间，比武台上，五长老一副欲要拆穿萧逸说谎的样子，得意地说道，“萧逸，你以为你耍这些小聪明就可以逃避责罚吗？”
<br/>　　萧若狂嘲讽道，“萧逸，我在玄火门中，有一位二品炼药师的老师，再加上门中长老倾力培养，苦苦学习四年，才终于在前不久炼制出淬体丹，成为一名尊贵的一品炼药师。”
<br/>　　“炼药一道，何其艰难，你年纪轻轻，却妄言自己已是一品炼药师，哼，徒惹他人笑话罢了。”
<br/>　　在萧若狂看来，自己学习了四年就成为了一品炼药师，已经非常了不起了。
<br/>　　五长老直接喝道，“萧逸，打伤家族炼药师，还在众长老面前撒谎，意图蒙骗家族长辈。哼，这两罪并罚，我看杖刑八十还是轻的了。来人，给我将这个孽畜拿下。”
<br/>　　长老席上，二长老听完大长老的分析，叹了口气，道，“如此说来，萧逸真的是在撒谎了。”
<br/>　　孰料，大长老却摇了摇头，沉声道，“不。”
<br/>　　“哦？”二长老愣了愣。
<br/>　　大长老忽然露出了笑意，道，“我之前的分析，只是以常理来说，只是针对普通人。至于萧逸嘛，从他这些天的表现来看，你觉得他是普通人？”
<br/>　　“大长老的意思是？”二长老微微疑惑。
<br/>　　“呵呵。”大长老从来都是一副严肃的样子，如今却笑逐颜开，道，“需知道，这个世界，是拥有天才的。天才，自然不能以常理计算。”
<br/>　　“嗯？”二长老顿时眼前一亮，问道，“你认为他就是那种天才？”
<br/>　　大长老笑道，“是不是炼药师，一试便知，这种一戳就破的谎言，也只有笨蛋才会去撒。而以萧逸的表现，他显然是个武道天才。既然他不是笨蛋，那么便证明，他不会撒这种慌。”
<br/>　　大长老活了一辈子，眼光极准，相信自己不会看错萧逸。
<br/>　　而这时，五长老正扬言要将萧逸拿下，更是准备出手。
<br/>　　二长老顿时大喝道，“都给我住手，谁敢拿萧逸，族规伺候。”
<br/>　　“阿？”
<br/>　　二长老一声大喝，所有人都露出了惊疑的表情。
<br/>　　五长老惊怒道，“二长老，您这是什么意思？难道您真的相信萧逸这个大言不惭的骗子？”
<br/>　　“萧逸，你敢不敢当众炼药？”五长老不敢质问二长老，将矛头指向了萧逸。
<br/>　　“有何不敢。”萧逸冷冷地回应了一声。
<br/>　　“好，死到临头，还敢嘴硬。”五长老冷笑一声，“来人，准备好药材，我要当众揭穿萧逸的谎言。”
<br/>　　这时，一旁的三长老道，“五长老，你质疑逸儿，我何尝不质疑萧若狂。说不准，他这个一品炼药师，也是吹嘘出来的。”
<br/>　　“混账。”五长老喝道，“若狂的炼药师身份，自有玄火门证明，岂会如萧逸那般，黄口小儿，信口雌黄。”
<br/>　　三长老冷笑道，“这里是萧家，可不是玄火门，我只问萧若狂，敢不敢当众炼丹？”
<br/>　　“哼。”萧若狂冷哼一声道，“没问题，就让萧逸那种废物看看，不是谁都能当一品炼药师的。”
<br/>　　.....
<br/>　　不多时，一堆药材摆到了比武台上。
<br/>　　萧若狂傲然地挑选了数份淬体丹药材，不屑地看了萧逸一眼后，开始炼药。
<br/>　　而萧逸则只拿了一份材料。
<br/>　　“怎么回事？”族人们纷纷疑惑，“炼药存在着很大的失败几率，萧逸只拿一份是什么意思？”
<br/>　　“莫不是他如此自信，一份材料足以炼制成功？”
<br/>　　五长老冷笑一声，“死到临头，还在装模作样。”
<br/>　　萧逸、萧若狂，两人开始了炼丹。
<br/>　　然而，炼丹刚刚开始，萧若狂和五长老看到萧逸的炼药炉，顿时嗤笑一声，“竟然是低等的鎏金炉，炼药效果本就差，我看他这次如何出丑。”
<br/>　　一旁的三长老急得跺脚，暗道，“糟了，逸儿也是的，既然会炼丹，为何不早跟我说呢，我好准备个三旋火炉阿。哎呀，要是炼制失败了，该如何是好。”
<br/>　　鎏金炉，价格大概在二百两，只是普通的炼药炉。
<br/>　　而三旋火炉，是一品炼药师的标配，萧若狂现在就是用这种炼药炉。
<br/>　　所有人都在紧张地看着两人的炼制过程。
<br/>　　长老席上，大长老更是紧紧地盯着，目光凝重。
<br/>　　他不相信自己会看错人，但终究还是心存一点疑惑。
<br/>　　时间慢慢过去，萧若狂使用三旋火炉，炼丹的速度更快，但已经失败了两次。
<br/>　　反观萧逸，仍旧在第一份材料中炼制着。
<br/>　　又过了十多分钟，两个药炉中，尽皆传来阵阵丹香，让人闻之神清气爽。
<br/>　　又过了数分钟，两人同时结束了炼丹。
<br/>　　萧若狂率先揭开了药炉，炫耀似地看了萧逸一眼。
<br/>　　药炉中便是成品的淬体丹。
<br/>　　长老席上，大长老远远看去，赞赏道，“不错，虽然炼制了三次才成功，但以你这个年纪，已实属不易，很了不起。”
<br/>　　大长老观察了那些淬体丹一眼，点头道，“丹药成形，虽不算上等，但已经合格，萧若狂你确实有一品炼药师的水准。”
<br/>　　这时，萧逸也揭开了炼药炉。
<br/>　　当炼药炉被揭开的瞬间，浓郁的丹香竟瞬间传遍了整个比武台，甚至飘扬到长老席和其他族人之中。
<br/>　　“好浓郁的丹香。”大长老一惊，连忙看去，一看之下，顿时大惊失色。
<br/>　　“丹药完美无瑕，浑圆通透，药香挥洒浓郁，证明材料被最大限度地炼制，完全融入淬体丹中。厉害，当真厉害，这等炼药技术，堪称完美。”大长老毫不掩饰自己的赞赏之情。
<br/>　　“完美丹药？”族人们也是大惊，“怎么可能，虽然只是最低等的淬体丹，但若要炼至完美，起码需要数十年的经验积累，萧逸是怎么做到的？”
<br/>　　“而且他只用了一份材料就炼制成功了吧，天才，这绝对是天才。”
<br/>　　“想不到萧逸不仅是武道天才，更是天才炼药师，天佑我萧家阿。”
<br/>　　长老席上，大长老满脸笑容。
<br/>　　“自从家主失踪后，老夫足足愁了十年。”大长老自语道，“这下好了，他失踪了，留下来的儿子却比他还出色。好，很好，萧家后继有人，我也就放心了。16岁的一品炼药师阿，老夫这一身炼药知识，终于不用靠百炼那个笨蛋了。”
<br/>　　然而，正当他大怀欣慰时，却忽然目光一凝，暴喝一声，“小子，尔敢？”
<br/>　　比武台上，萧若狂竟不知何时起出现在萧逸后背，后天一重的修为全开，一掌朝萧逸后背打去。
