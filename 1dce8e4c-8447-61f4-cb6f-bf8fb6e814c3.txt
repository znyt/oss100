　　“可恶！”
<br/>　　在少帝打爆了无数剑光之后，这一位神秘人物，已经是带领楚风眠，进入到了佛寺的深处。
<br/>　　在他的眼神之中，露出了无比的愤怒，跟深深的杀意。
<br/>　　这一次，本来楚风眠已经是必死之局，可居然是被这样一名神秘人物突然插手，救走了楚风眠。
<br/>　　“既然是还在佛寺之中，就逃不走！”
<br/>　　少帝怒喝一声，身形连连闪动，也进入到了佛寺深处之中去，同时将一道道消息传递了出去。
<br/>　　整个净土佛寺的所有魔族，都接到了少帝的消息，疯狂的寻找着楚风眠的踪影。
<br/>　　另一边。
<br/>　　这一位神秘人物的遁光的速度极快，一路之上穿越了无数区域，带着楚风眠进入到了净土佛寺的深处。
<br/>　　这一位神秘人物，似乎是对净土佛寺之中，无比熟悉，许多区域危险的区域，都被他一一避过。
<br/>　　很快便是进入到了净土佛寺的最深处，到了一处密室之内，才缓缓的停了下来。
<br/>　　楚风眠，也看清楚了这一名老者的样子。
<br/>　　这一名老者的样子，楚风眠无比眼熟，正是他之前在归墟城之中，见到过的那位卖东西的老者。
<br/>　　楚风眠正是从他的手中，买到了那剑柄，玉符，还有如今楚风眠施展出的剑术，三阳烈炎剑，这三件第一剑帝的至宝。
<br/>　　楚风眠刚刚，也是听出了这老者的声音，所以在这老者抓走楚风眠的时候，楚风眠没有反抗，任由他将楚风眠带到了这净土佛寺之中。
<br/>　　“不知道前辈是？”
<br/>　　楚风眠看向这老者，急忙行了个礼，这老者的实力，深不可测。
<br/>　　能够是从少帝的手中，救走楚风眠，绝对是半步天人境级别的强者。
<br/>　　就连楚风眠都无法看穿这老者的实力，更何况这老者，救了楚风眠这一次，于情于理，楚风眠都要尊敬。
<br/>　　“不用那么拘束，老夫名为樊，不过世人知晓的名字，应该是太初。”
<br/>　　老者看着楚风眠，也是打量了一眼，眼神之中露出了几分满意道。
<br/>　　“果然，以你对剑道的领悟，足以领悟出三阳烈炎剑了，看来老夫没选错人。”
<br/>　　“这老者，是故意将三阳烈炎剑交个我的？”
<br/>　　楚风眠听到这老者的话，都是暗自吃惊，听他的意思，只怕是这老者将那古籍，交给楚风眠的时候。
<br/>　　就已经是绝对，楚风眠可以从古籍之中，领悟出三阳烈炎剑来。
<br/>　　而刚刚三阳烈炎剑的威力，突然增长，只怕也是跟这老者有关。
<br/>　　“樊，太初……”
<br/>　　樊，只有一个字的名字，这一号人物，楚风眠倒是从未听说过。
<br/>　　在九域古籍的记载之中，应该是没有过这样一位强者，但是太初……
<br/>　　“前辈跟太初剑，是何关系？”
<br/>　　楚风眠突然出声问道。
<br/>　　之前在归墟城之中，楚风眠所得到的，除了三阳烈炎剑，这第一剑帝的仙术之外。
<br/>　　还有一个，便是太初剑的剑柄。
<br/>　　曾经第一剑帝的四把极道仙兵之一的太初剑。
<br/>　　这老者说世人知晓他的名字，便是太初，那绝对是跟太初剑，脱不开关系。
<br/>　　而且打量着老者身上的气息，隐隐约约之间，在楚风眠的脑海之中，已经是想到了一个可能。
<br/>　　“老夫，就是太初剑的器灵。”
<br/>　　樊也不避讳，直接开口道。
<br/>　　器灵。
<br/>　　听到樊开口，楚风眠脑海之中的猜测，也是确定了下来。
<br/>　　在听到太初这个称呼的时候，楚风眠就多半猜测，眼前这老者，就是太初剑的器灵了。
<br/>　　只是令楚风眠有些吃惊的是，一个器灵，居然都拥有如此可怕的威力，居然是有着力量，从少帝的手中，救走楚风眠。
<br/>　　这样一尊强者，居然只是器灵？
<br/>　　器灵，一切强大的灵器之中，都有可能诞生器灵，诞生能够诞生意识的器灵，都无比罕见了。
<br/>　　大部分的器灵，仅仅只有一点点模糊的意识，也只有云羽天宫，诞生出了云老来。
<br/>　　但是云老的实力，跟眼前这樊相比，却相差甚远。
<br/>　　极道仙兵的器灵，居然都有着如此可怕的实力？
<br/>　　楚风眠脑海之中不由的想到。
<br/>　　“小子，看来你还真不知道什么叫做极道仙兵，你眼前这家伙，如果不是实力受损，那少帝只怕是在他的面前不死，也要灰溜溜的逃走，怎么敢跟他抗衡。”
<br/>　　就在这时，脑海之中，星玄的声音响起。
<br/>　　星玄的突然出现，楚风眠也是习惯了，平日里星玄一直是在沉睡，只有当出现一些让他感兴趣的事的时候，他才会突然苏醒过来。
<br/>　　“这樊老，如此厉害？”
<br/>　　听着星玄的话，楚风眠又是一愣。
<br/>　　听星玄话中的意思，眼前这樊，还不是最为厉害的时候。
<br/>　　如果这樊在全盛之时的话，要比起少帝，还要厉害？
<br/>　　少帝的实力，已经是几乎半步天人境的巅峰了，再进一步，就是天人境。
<br/>　　那岂不是意味着，樊的实力，如果是在巅峰之时，那就是天人境的实力。
<br/>　　一位器灵，实力可以达到天人境？
<br/>　　楚风眠都有些无法想象。
<br/>　　“小子，看来你还是不懂的什么叫做极道仙兵，你以为天下之间，极道仙兵为何如此稀少？哪怕是在强大的灵器，跟极道仙兵，也有着云泥之别。”
<br/>　　星玄也不卖关子了，对着楚风眠解释道。
<br/>　　“能够被称为极道仙兵的，便有一点，器灵之力都有着天人之力，换句话说，任何一件极道仙兵的器灵，都是天人境的强者，纵然是没有主人，都是绝对的强者。”
<br/>　　“甚至一些强大的极道仙兵，一生都从未有过主人，因为他本身就是绝对无敌的霸主。”
<br/>　　任何一件极道仙兵，都是天人境的强者。
<br/>　　楚风眠都是被深深的震撼住了，极道仙兵，本身便是天人境的强者，强横的极道仙兵，甚至是一方霸主。
<br/>　　极道仙兵的强横，从古籍之中都有所记载，但是楚风眠还是第一次知晓这一点。
<br/>　　怪不得星玄说，如果眼前樊，是全胜时期的实力的话，少帝在他的面前，只能够是灰溜溜的逃走。
<br/>　　少帝再强，也只是半步天人境的强者，跟真正的天人境，根本无法抗衡。
