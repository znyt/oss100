　　松纹古定剑光是听名字就知道跟木有关。至于鱼肠剑……道风还真是头回听说五行属火。
<br/>　　道渊真人摆出一副知识帝的样子，捋着胡须，侃侃说道：“鱼肠剑的鱼，你知道是哪个鱼？北冥有鱼，其名为鲲。化而为鸟，其名为鹏……鱼肠剑，取的正是这个典故，鲲鹏为火之鸟，鱼肠剑的属性，自然也就属火了，而且据我所知，鱼肠剑通体赤红，为燧石打造，燧石非石，乃是火之精华，鱼肠剑的属性，自然是火。”
<br/>　　道风没有纠结这些细节，反正道渊真人不可能骗自己，想了一下，说道：“你想让我做什么？”
<br/>　　“不是我想让你做什么，我说到这，你还不明白吗？”
<br/>　　道风沉默了一下道：“你让我想办法得到轩辕剑？可是就算凑齐五把剑，要怎么样才能进入太阴山？”
<br/>　　“这个我不知道，我只是参出了这句谒语而已。”
<br/>　　道风又想了一会，说道：“这件事很难办成。我总不能去轩辕山去抢。”
<br/>　　道渊真人道：“你这人脑子不转，龙华会上啊，光明正大地夺得这把剑。”
<br/>　　道风苦笑，“他们要是允许我参加倒是好了。”
<br/>　　“你不行，少阳可以，他毕竟还是茅山掌教，只要打赢星月奴那几个弟子，法术公会不可能食言。”
<br/>　　“但是少阳失踪了，不知道什么时候回来。”
<br/>　　道渊真人又拿起茶壶，把里面的牛奶一口气喝干，摸了一下嘴唇，说道：“我可以跟他们谈，同意跟他们合办这场龙华会，不过我的条件是延期……我最多可以将时间延长到一个月之后，这一个月时间内，你无论如何要把叶少阳找回来……否则就没希望了。”
<br/>　　道风睁大眼睛，看着他，说道：“你为什么要帮少阳？”
<br/>　　“帮他？”道渊真人摇摇头，随即又叹了口气，“我不是帮他，我只是赌一把，赌他是应劫之人！把法术界的未来交到他手上，总比交给法术公会要好。”
<br/>　　“你就不怕，他为了芮冷玉……转世鬼童，背叛法术界？”
<br/>　　道渊真人笑了笑，“这个问题，我之前想过，我想……赌一把，我相信他。”
<br/>　　道风定定地望着他，最终点了点头，道：“多谢。”
<br/>　　道渊真人问道：“你救走白起，是为了什么？”
<br/>　　道风笑了笑，“没想到你也听说了。”
<br/>　　“我虽然人在这里，但是有些事，总有人来告诉我的。”
<br/>　　对方既然站在叶少阳这边，道风对他也没什么好隐瞒的，在周围布置了一个避免天听的结界。这个一向喜欢装逼，见人没有三句话的道风，在道渊真人面前也是不浪费口水，把徐福找自己的事从头讲了一遍。
<br/>　　道渊真人听完，久久无语。
<br/>　　“徐福，在世上呆了快两千年，很多事情，他比任何人看得都清楚。你我都不知道他到底要做什么，不过，他既然也是站在我们这边的，那我们对他放心即可。”说到这，道渊真人转头看道风一眼，“阴司就没几个明白人，人间也没几个明白人。我只想知道，大帝对这件事怎么看？”
<br/>　　“我怎么知道。”道风轻描淡写。
<br/>　　道渊真人微微一笑，也不追问，说道：“我最后问一件事，你师父，现在什么地方？”
<br/>　　“去紫云观修行去了。好久没出来了，最近发生了这么多事，我也想找他，但是找不到。”
<br/>　　“那就不要去找他，非到万不得已，他不会再插手这些事的。”
<br/>　　道风狐疑地看着他，“你又怎么知道？”
<br/>　　道渊真人瞥了他一眼，道：“他舍弃肉身，未必是一件坏事，没了身份，没了肉身的限制，他可以做的事情还有很多。”
<br/>　　道风沉吟半晌，笑道：“大家都以为你老了，在这里枯坐一辈子，什么事都看不懂，其实，你比很多人都清楚。”
<br/>　　道渊真人双手按着膝盖，用一种耐心寻味的眼神望着星空，喃喃道：“当局者迷，我这个没人管的老头，往往看的更清楚，但看的清楚，有时候只会更痛苦……”
<br/>　　他把目光移到道风身上，缓缓说道：“你懂的越多，在这个世界上就越孤独，因为你看透，却不一定能做得到。我，你师父，甚至你自己，我们都是这样。”
<br/>　　“试试吧，不试试怎么知道。”
<br/>　　道渊真人缓缓点头，重复道：“试试吧。”
<br/>　　两人相对坐了很久，都没有再说话，最后道风站起来，说道：“我走了。”
<br/>　　“一个月，我只能延长一个月，一个月之后，少阳不回来，再想拿轩辕剑，就只能杀人越货了。”道渊真人眼睛望着远处，说道。
<br/>　　杀人越货，自然是不现实的，除非对方找死，否则从别人手上就算抢到轩辕剑，也只会为法术界同仁所不齿，谁还会听你号令？
<br/>　　所以，一个月之内，叶少阳必须回来。
<br/>　　道风点点头，什么也没说，单手破开了虚空，钻了进去。
<br/>　　回风之谷的一路上，道风为了避免被阴司发现，一路上都是从丛林中穿过，走在黑暗之中。
<br/>　　他想到自己的经历，曾经的人间道神，法力冠绝天下，所有人都看好他成为一代宗师，荣登三宝，结果他先是入魔，舍弃肉身，又在鬼域开辟一方势力。
<br/>　　铲昆仑，灭悬空，为救少阳，力战半个法术界，逼死无极天师，得罪于人间。
<br/>　　为寻找玄武，大闹青冥界，梨山战老母，星宿斩青牛，得罪了整个阐教，空界现在也没有容身之处。
<br/>　　最近又大闹阴司，救走白起元神，震碎地藏菩萨分神，一举得罪了佛国和阴司，成了阴司一等通缉犯人……
<br/>　　人间，青冥界，无量界，阴司……自己现在一个地方都去不了。
<br/>　　被无数人嫉妒，咬牙切齿地恨，视为眼中钉，可是……那又怎么样？
<br/>　　天上地下，没有自己的容身之处，那又怎么样？
<br/>　　道风走出树林，走上了一座山峰，看着下面一片茫茫的原野，深吸了一口气，笑了笑，纵身飞了下去。
