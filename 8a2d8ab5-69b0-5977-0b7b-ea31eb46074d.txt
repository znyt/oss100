        “委屈你了。”傅昼景再次揉了揉童筱颖的头。
<br/>
<br/>        童筱颖摇了摇头，再次匍匐在傅昼景的怀中，轻轻地搂着他，抿了抿唇，一往情深地说：“嫁给你，我一点都不觉得委屈。”
<br/>
<br/>        因为，是他将她从水深火热之中解救出来的，不是吗？
<br/>
<br/>        如果不是他及时出现，她会继续漫无目的地相亲，最终说不定还会随便嫁一个男人了事。
<br/>
<br/>        虽然俗话说，贫贱夫妻百事哀。
<br/>
<br/>        但是，在童筱颖的心里，一个男人有不有钱无所谓，只要是她喜欢的男人，而且他对她好、爱她、爱这个家、不背叛她，赚得钱足够过日子就行。
<br/>
<br/>        所以，童筱颖一点都不在乎傅昼景是否有钱。
<br/>
<br/>        女人的下半生，只要是嫁对了人，才是最大的幸福。
<br/>
<br/>        童筱颖希望自己是个幸福的女人。
<br/>
<br/>        两人躺床上这么久了，真的只是单纯地拥抱，傅昼景对童筱颖一点那方面的欲望也没有。
<br/>
<br/>        一来是他觉得她年龄还小才十八岁，二来是他觉得应该把她的第一次留在洞房花烛夜那晚，将来可以给她一个美好的回忆。
<br/>
<br/>        “我们去吃饭吧！不然，饭菜都要凉了！”童筱颖微笑着说。
<br/>
<br/>        傅昼景应了声“嗯”，支起身子坐起来的时候，顺道把童筱颖也搂着起了身。
<br/>
<br/>        坐在餐桌前，童筱颖不停地往傅昼景的碗里夹菜，连鱼肉都是她剔除了鱼骨头才夹到他的碗里的。
<br/>
<br/>        傅昼景会心一笑，用同样的方式，夹起一块鱼肉，并将鱼肉里的小刺剔除后，亲自喂到了童筱颖的嘴中。
<br/>
<br/>        他想用自己的方式宠着自己的这个小娇妻。
<br/>
<br/>        童筱颖看着傅昼景，漂亮的大眼睛眯成了月牙，嘴里嚼着鱼肉，嘴角则幸福地微微上扬。
<br/>
<br/>        虽然自己今天过得不顺，但并不影响傅昼景下午继续去公司上班的情绪。
<br/>
<br/>        他是一名建筑设计师，努力多接几个业务，钱自然就赚回来了，只是从今以后，他可能会辛苦点。
<br/>
<br/>        筱颖是一个什么样的女孩子，经过这段时间的相识相知，他心知肚明，看得一清二楚。
<br/>
<br/>        十八岁的筱颖，有着与外表不相符的心理年龄。
<br/>
<br/>        她待人友好温和，不拜金，不懦弱，不化妆，不崇洋媚外，生活上勤俭节约，工作上兢兢业业。
<br/>
<br/>        现在，在他们滨江城里，很多十八岁的女孩，大部分刚步入大学，一个个还都是温室里的花朵。
<br/>
<br/>        而筱颖，或许是因为她的家庭背景的缘故吗？
<br/>
<br/>        傅昼景曾听童筱颖的养兄诺司峻提起过，筱颖的养母对待筱颖尖酸刻薄，家里的家务基本上是筱颖全包了，还逼迫过筱颖辍学打工，而她的养父虽然对她很好，但是懦弱无能。
<br/>
<br/>        至于她的养兄诺司峻，即便以前护着她，可他也只不过是个半吊子的男人，他养自己的女朋友都有点儿困难，更别提会给她这个养妹预支什么生活费。
<br/>
<br/>        筱颖的坚强与独立，傅昼景是看在眼里，怜悯在心里。
<br/>
<br/>        她和他，在某种精神上，那么相似，不是吗？
<br/>
<br/>        所以，哪怕初恋前女友季思妍回来了，想要跟他求复合，他也不想辜负这个与自己同命相怜的女孩。
<br/>
<br/>        “筱颖，你想不想找回自己的亲生父母？”傅昼景忽然语重心长地问。
<br/>
<br/>        筱颖的身世，确实有点儿坎坷。
<br/>
<br/>        要寻到亲生父母，很难很难，不是吗？
<br/>
<br/>        时隔这么多年，她还是被贩卖到西镇的，一点线索都没有，要从何寻起？
<br/>
<br/>        梁家对她有养育之恩，而童家……
<br/>
<br/>        童筱颖回想起自己和童爸童妈所生活在一起时的那段日子，她把他们两当亲生父母对待，他们却瞒着她，看着她拿自己的幸福去为他两还债。
<br/>
<br/>        但值得庆幸的是，童爸童妈没有自私到底，他们拿了钱债务清除后就把真相告诉了她。
<br/>
<br/>        至少，他们对她算是有赐姓之恩吧！
<br/>
<br/>        这人情冷暖，世态炎凉……
<br/>
<br/>        童筱颖微微垂下眼帘，一边收拾着桌上的碗筷，一边紧抿薄唇，揪心地摇了摇头。
<br/>
<br/>        抬眸间，她看向了傅昼景，莞尔一笑：“我已经有你了！”
<br/>
<br/>        从今以后，他就是她的家人。
<br/>
<br/>        傅昼景明白了童筱颖的心意后，对上了她的明眸，会心地微笑着，然后和她一起收拾桌上的残羹冷炙。
<br/>
<br/>        “筱颖，你在我之前，有没有过恋爱？”茶余饭后，傅昼景搂着童筱颖坐在沙发上休息的时候，突然问道。
<br/>
<br/>        童筱颖怔了怔，回想起自己的前男友何明旭的时候，心口莫名其妙地抽痛了一下。
<br/>
<br/>        那天，她跟他相亲的时候，她说她没谈过恋爱，其实他知道，她跟他撒了谎。
<br/>
<br/>        筱颖其实是个素颜就很漂亮的女孩子，而且还是那种纯天然的美，所以，她在认识他之前，一定有男生追求过她。
<br/>
<br/>        “有！”童筱颖沉默了一会儿，才弱弱地回答道，顿了顿后，又像个犯了错的孩子般，为自己辩解道，“但我没跟他睡过，我只是跟他拥抱过，牵过手，连吻都没有接过……”
<br/>
<br/>        说到吻……
<br/>
<br/>        童筱颖这才想起，自己的初吻，被那个三更半夜翻她家阳台闯入她出租屋的陌生男人给强行夺走了。
<br/>
<br/>        而且，她的身体，也被那个男人强行抚摸过……
<br/>
<br/>        虽然第一次还完好无损的保留着，但是，回想到这里，童筱颖顿时觉得自己很脏，甚至有些自惭形秽，觉得自己配不上昼景。
<br/>
<br/>        她其实是那种比较传统保守的女孩子，对那方面的事情，她放不开。
<br/>
<br/>        童筱颖一直坚持着，自己的第一次，一定要留在新婚洞房花烛夜那晚。
<br/>
<br/>        “我都知道的，只是随便问问。就算你不是第一次，我也不在乎。我只知道，从今以后，你只专属于我一个人就行了。”傅昼景会心一笑，抬起手来，温柔地在童筱颖的头顶揉了揉。
<br/>
<br/>        忽然间，他也不知道自己为什么就好奇起她过去的感情史了。
<br/>
<br/>        童筱颖抬眸见傅昼景相信了自己，还说不在乎她是不是第一次，顿时舒心地松了口气。
<br/>
<br/>        换句话说，就是，他不在乎她以前被什么男人吻过，抚摸过……只要她从今以后，是他的人就行了。
