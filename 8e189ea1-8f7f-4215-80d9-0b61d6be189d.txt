　　沉吟片刻，慕清霄不顾周围的闲言碎语，径直向万药斋走去，若是记得不错，小医仙貌似就在那里，也只有万药斋能够保她一时。
<br/>　　转过几条街道，喧哗声渐渐散去，慕清霄顺着幽静的小道行走着，几分钟后视线中就出现一座别致的小庄园。
<br/>　　这里就是万药斋，只是在庄园的门口，防守有些森严，慕清霄身形一闪就消失在原地。
<br/>　　庄园戒备虽然森严，但对他来说却是犹如后花园一样，斗皇强者隐匿气息后，哪怕有人从身边走过，都不能发现，倒是有点像隐身。
<br/>　　却与隐身有所不同，而是返璞归真，自身融入自然。
<br/>　　修炼过《御女心经》的慕清霄，本身就对万物有着绝对的亲和力，隐匿身影对他来说就跟吃饭一样简单，哪怕在萧家，若非他主动现身，凌影根本无法察觉。
<br/>　　进入庄园后，迎面而来是一名老者，见到慕清霄后，脸色顿时沉了下来，呵斥道：“最近不是说过，没事不要在庄园里乱走动，你是负责什么岗位的，明天不用过来了。”
<br/>　　完全不理会老者，慕清霄瞳孔中浮现三颗黑色勾玉，道：“带我去小医仙住的地方。”
<br/>　　老者身躯一震，旋即目光就变得呆泄起来，同时浮现三颗黑色勾玉，语气木讷道：“是。”
<br/>　　老者在万药斋的地位似乎不低，有他带路，路过的巡逻护卫都恭敬的打招呼，慕清霄则悠哉悠哉的跟在他的身后。
<br/>　　万药斋不是很大，占地却也不小。
<br/>　　几分钟后，在老者的带领下，慕清霄就来到庄园后面一座四合院外，门口有四名护卫把守，或者说是监视。
<br/>　　见老者过来，四名护卫一齐行礼：“大长老。”
<br/>　　老者根本不理会四人，径直带着慕清霄向四合院内走去，院内种植着花草树木，整体来说环境非常不错。
<br/>　　不过，身为监视对象，哪怕周围环境在好，小医仙恐怕也过的不好吧。
<br/>　　来到一扇门前，老者停了下来，慕清霄轻轻的敲响了房门，屋内无人回应，但他却已经感知到了小医仙的存在。
<br/>　　“咚咚咚...”
<br/>　　片刻后，房门打开，小医仙见来者是慕清霄后，美眸中的冰冷瞬间散去，取而代之的是一抹温柔。
<br/>　　“在外面等着。”
<br/>　　“是。”
<br/>　　老者点头称是，慕清霄走进房间，目光在里面扫视了一圈，道：“看来，他们并没有强迫你做什么。”
<br/>　　小医仙微微一笑，自怀中掏出一个小玉瓶，然后从中滴出一滴红色液体，最后轻轻的搽拭在慕清霄掌心。
<br/>　　“我知道你会来救我的。”
<br/>　　慕清霄轻轻捏了捏她的鼻子，道：“你就这么有自信？”
<br/>　　俏脸上露出一个温柔的笑容，望着慕清霄那无可挑剔的完美面孔，小医仙笑盈盈的说道：“嗯，我相信你不会丢下我不管的。”
<br/>　　听闻她的温软细语，慕清霄忍不住搂上她的纤腰，道：“青山镇实在是太小了，想不想去外面的世界看看？”
<br/>　　“外面的世界？”
<br/>　　小医仙美眸中有些向往，靠在他的怀中并没有挣扎，但美眸中却闪过一丝黯然，道：“但是，外面的世界，危险程度也不是青山镇能比的，我会拖累你。”
<br/>　　真是傻的可爱！
<br/>　　慕清霄替她捋了捋额前的青丝，语气平淡，道：“我都不怕，你怕什么，而且外面的世界能够帮你掌控厄难毒体。”
<br/>　　闻言，小医仙猛的一颤，娇躯微不可查的颤抖起来，豁然抬头，道：“你......你怎么知道，我可没有跟你说过这些！”
<br/>　　慕清霄温柔的握住她如凝脂一般的玉手，放在鼻尖闻了闻，道：“厄难毒体是劫，何尝不是机遇，相信我吗？”
<br/>　　他的语气极其平淡，可正是这简简单单的四个字，使得小医仙眼眶中泪水打转，同时心中也升起了一丝希望，一份念想。
<br/>　　抿了抿粉唇，思索片刻，小医仙重重点头，靠在他的怀中，道：“我相信你，带我离开吧。”
<br/>　　小医仙的东西不多，能够派上用场的都在这里，简单的收拾了一下，便轻装上路。
<br/>　　两人站在城外，由于狼头佣兵团在搜寻慕清霄和小医仙的缘故，城内已经乱作一团。
<br/>　　小医仙转身看了一眼青山镇，望着身旁的慕清霄，美眸中闪过一丝坚定，俏皮笑道：“我们走吧，早就想离开这里了。”
<br/>　　慕清霄微微点头，道：“下一站就定在塔戈尔大沙漠，在那里有我需要的东西，我必须去一趟。”
<br/>　　塔戈尔大沙漠，是加玛帝国最大的沙漠，位于东部省份，小医仙曾经也听说过，据说那里很危险。
<br/>　　但是，慕清霄决定的事情，小医仙只是温柔一笑，美眸中蕴含着关切，犹如他的小妻子一样走在身旁。
<br/>　　有佳人相伴，路途也不再枯燥乏味，一路上，慕清霄也将自己所知，关于厄难毒体的信息全部说给小医仙听了一遍。
<br/>　　等慕清霄说完，小医仙呆呆的望着他，真的没想到他知道这么多信息，同时心中更加期待了。
<br/>　　“你说，只要暂时压制毒体爆发，等实力提高一些，便能控制厄难毒体？”小医仙一脸震惊道。
<br/>　　“嗯，确实有控制毒体的方式，但我们现在的实力，想要走到那里很难，在等几年，等我们实力都相互提高些，就带你去那里走一遭，顺便把你体质的事情解决。”
<br/>　　听慕清霄的话语，明明是很恐怖的体质，到了他嘴边却显得如此轻松，连带着小医仙本人都轻松起来，乖巧的点了点头。
<br/>　　望着小医仙背上的包裹，慕清霄自储存空间中取出一枚纳戒，温柔的戴在她修长的玉指上，道：“喜欢吗？”
<br/>　　望着手中的戒指，小医仙美眸中尽是震惊，她自然知道这东西叫做纳戒，极其珍贵，整个青山镇都找不到一枚。
<br/>　　却不想到，慕清霄如此轻描淡写的送给自己，望着手指上的戒指，小医仙绝美的脸颊上涌上一抹嫣红，语气清脆温柔，道：“喜欢。”
<br/>　　“将包裹放进去吧，空间不是很大，放一些药材暂时是够了，等有机会在给你弄一个好的。”
