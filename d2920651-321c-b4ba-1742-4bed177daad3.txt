　　萧逸手中血戮剑轻轻一震。
<br/>　　半空中，一个灵气漩涡陡然而现。
<br/>　　比冥空剑所凝聚的灵器漩涡大了10倍不止。
<br/>　　“上…上品灵器。”顾长空有那么一刻，不敢相信自己的眼睛。
<br/>　　别说他，就是他顾家，百武城中的大家族，也不过是拥有下品灵器。
<br/>　　哪怕是自己有剑堂长老帮助，得了星芒石。
<br/>　　也不过是有一把可以媲美上品灵器的冥空剑罢了。
<br/>　　真正的上品灵器，有多恐怖，又代表着什么，他很清楚。
<br/>　　那是连破玄九重武者都会眼馋的东西。
<br/>　　这时，萧逸淡淡地挥剑一指。
<br/>　　“顾长空，还记得一年多前的账吗？”
<br/>　　“本来，你这么久没来找我麻烦，我差点儿忘了。”
<br/>　　“现在，既然你迫不及待想要算个清楚，那我便与你好好算一算。”
<br/>　　话音刚落。
<br/>　　萧逸手中血戮剑重重一挥。
<br/>　　“崩山斩。”萧逸冷喝一声。
<br/>　　“我…”顾长空一惊，想说些什么。
<br/>　　但看到一道惊天剑气斩来，立刻闭上了嘴巴。
<br/>　　也是一剑斩出，七煞斩。
<br/>　　七条煞气更深，威势更猛的游龙，恐怖袭出。
<br/>　　惊天剑气，与七条煞气游龙，瞬间对撞。
<br/>　　这已经不再是萧逸于顾长空的战斗了。
<br/>　　而是血戮剑与冥空剑的碰撞。
<br/>　　更是崩山斩与七煞斩的相争。。
<br/>　　无论是剑气还是七条煞气游龙。
<br/>　　它们的力量来源，都是半空中的灵气漩涡。
<br/>　　‘轰’，一声惊天爆响。
<br/>　　一股暴走的力量，瞬间蔓延整个比武场。
<br/>　　一阵耀眼光芒，刺得人睁不开眼睛。
<br/>　　数秒后。
<br/>　　一道剑气，从光芒中冲出，而后直直地打在大长老布下的灵气屏障上。
<br/>　　灵气屏障，一阵颤抖。
<br/>　　待得光芒消失。
<br/>　　观战子弟才看清面前状况。
<br/>　　七条煞气游龙，早已被剑气绞碎，不复存在。
<br/>　　偌大个比武场，地面碎裂，几近全毁。
<br/>　　顾长空，倒在碎石砂砾中，满身是血，生死不知。
<br/>　　唯有灵气屏障上，那不断颤动的波纹，正证明着那道剑气的强大。
<br/>　　屏障之外，观战的弟子看得心惊肉跳。
<br/>　　生怕屏障被毁，剑气飞向他们。
<br/>　　半晌，待得剑气消散，灵气屏障仍旧未能冲破，他们才松了口气。
<br/>　　很显然，血戮剑与冥空剑的碰撞，以血戮剑轻松完胜而结束。
<br/>　　萧逸很清楚，自己的剑气，不可能冲破大长老的灵气屏障。
<br/>　　更不会伤到屏障外的弟子。
<br/>　　这时，萧逸手中握剑，淡淡地走向碎石堆中的顾长空。
<br/>　　那散发着妖异血色的剑光，映衬在他的脸庞上。
<br/>　　让他看起来帅气得妖异，却也帅气得让人着迷，移不开眼睛。
<br/>　　场外，一众弟子惊骇地看着，仿佛不敢相信这个结果。
<br/>　　“败了，剑堂首席顾长空，竟然完败。”
<br/>　　“一个外门弟子，竟然完败内堂首席，这是裂天剑派头一次出现的事。”
<br/>　　另一边，林劲四人会心一笑。
<br/>　　“我就知道，萧逸这个家伙不会让我们失望的。”林劲笑道。
<br/>　　“害我们白担心一场，这个家伙。”秦飞扬难得地认真笑着。
<br/>　　“这就是萧逸的真正实力吗。”柳嫣然注视比武台上，那道正受着万人瞩目的身影，心中小鹿乱撞。
<br/>　　不过，她也发现，附近的一些女弟子，同样也在这样紧紧地看向比武台那道身影。
<br/>　　一个个两眼放光。
<br/>　　“哼。”柳嫣然愤愤地跺了跺脚，美眸中带着幽怨和嗔怒，“臭家伙，没事长这么帅做什么。”
<br/>　　这时，比武台上，萧逸已来到早已昏厥过去的顾长空身边。
<br/>　　此举，让得所有人心头一揪。
<br/>　　“萧逸，你想做什么。”不远处的剑堂长老大喝一声。
<br/>　　台上的裁判，也立马喝道，“萧逸，别冲动。”
<br/>　　身影一闪，瞬间来到萧逸面前。
<br/>　　但，他却不敢有任何动作。
<br/>　　他只是个内门执事，洞玄九重修为。
<br/>　　刚才萧逸的攻击，很显然已经达到破玄境层次。
<br/>　　他很清楚，自己远不是萧逸的对手。
<br/>　　所以，只能愣愣地看着，张开嘴想说些阻止的话，却什么都说不出来。
<br/>　　这就是这个世界的残酷法则，实力为尊。
<br/>　　哪怕他是内门执事，此刻却也在萧逸的气势下慑服着。
<br/>　　“还在等什么？”萧逸淡淡地看向裁判。
<br/>　　“还不宣布结果，是打算让我给顾长空补一剑吗？”
<br/>　　那淡淡的语气，慵懒的表情，让得台下女弟子已经不是双眼放光，而是仿佛要将他吃掉一般。
<br/>　　这时，一众弟子才发现。
<br/>　　与顾长空的衣衫褴褛、模样凄惨相比。
<br/>　　萧逸，竟衣衫整齐，连衣角都不曾凌乱过一分。
<br/>　　刚才一战，对他而言，竟是轻松如斯吗？
<br/>　　比武台上，裁判反应了过来，连忙道，“第一战，外门…不，萧逸胜。”
<br/>　　萧逸收回剑，转身离去。
<br/>　　长老席上，大长老也挥了挥手，收去了灵气屏障，脸上写满了满意之色。
<br/>　　“真不愧是未来的剑主，萧逸刚才的一剑，足以媲美破玄四重武者的全力一击。”
<br/>　　一旁的剑堂长老不屑地冷哼一声，“不过是靠上品灵器罢了，算什么本事。”
<br/>　　大长老笑道，“你弟子长空不也一样使用灵器了。”
<br/>　　“哼。”剑堂长老冷声道，“萧逸不过是区区洞玄七重武者，怎么可能拥有上品灵器。”
<br/>　　“这把灵器，外表血色妖异，也不知是不是邪器。”
<br/>　　“加之来历不明，我看，还是先收归剑派保管吧。”
<br/>　　极品灵器，整个北山郡，只有两把，轻易不能动用。
<br/>　　上品灵器，也只有猎妖殿分殿主、北山郡王、裂天剑派大长老、暗影楼楼主。
<br/>　　等等几个少数武者拥有。
<br/>　　剑派内，就是长老，也不过拥有中品灵器罢了。
<br/>　　上品灵器，是可遇不可求的东西，能否拥有，一看实力，二看机缘。
<br/>　　言归正传。
<br/>　　一旁的易老闻言，冷笑道，“剑堂长老是不是还要说，这把灵器是剑灵器，收归剑派后，放到你们剑堂最为合适。”
<br/>　　剑堂长老确实是这个想法。
<br/>　　现在被易老说穿，却也没有丝毫不好意思，反而说道，“本就该如此。”
<br/>　　“还本就该如此。”易老冷声道，“我手上也有一把来历不明的上品灵器，四长老你要不要。”
<br/>　　“我，是否也要上缴剑派？”
<br/>　　易老的声音，陡然间变得冷漠起来。
<br/>　　“你…”剑堂长老脸色难看道，“你这是故意包庇你弟子。”
<br/>　　“好了好了。”大长老出言道，“都是同门师兄弟，有什么好吵的。”
<br/>　　“十一，我看萧逸那把上品灵器，确实不简单。”
<br/>　　“要不，就收归剑派好了。当然，由你来保管。什么时候你觉得萧逸能掌控，你便给他。”
<br/>　　“还有你。”大长老看向剑堂长老，道，“顾长空技不如人，你也怪不得人，不必再针对萧逸。”
<br/>　　“反正萧逸是日后的剑主，顾长空输给他也没什么好丢人的。”
<br/>　　“好了，此事就这么定了。将比武台弄好，开始下一场比武吧。”
<br/>(本章完)
