　　朱博文顿时懵了，死死盯着叶尘，两眼瞪得老大，
<br/>　　“怎么可能！楼下的那些人，可是我花费重金请来的一帮职业打手，甚至还有退役的特种兵，你竟然没事？”
<br/>　　直到此时此刻，朱博文这才发现，自己原来小瞧了这个毫无背景的大一新生！
<br/>　　叶尘一言不发，缓缓走进房间之中，一步步朝朱博文走来，朱博文感受到了叶尘身上的气势压迫，顿时开始有些慌了，
<br/>　　“你你……你要做什么？我叔叔是天海大学的校长，你如果敢动我，我有一百种办法，让你在天海大学待不下去！”
<br/>　　叶尘听到这话，顿时嘴角微微一扬，露出一抹玩味的笑容，
<br/>　　“是吗？那我倒是很想见识一下，你那一百种办法！”
<br/>　　说完这话，叶尘脚下一步迈出，瞬间便已经来到朱博文的跟前，然后伸手按住他的肩头，让他无法动弹，继而缓缓拿起一旁桌上，之前朱博文吃西餐用的钢叉，然后直接插在了朱博文的大腿之上！
<br/>　　叶尘并没有打算立即杀掉朱博文，所以他出手很有分寸，避开了朱博文的大腿动脉，废掉了他这条腿，却又不会伤及他的性命。
<br/>　　“啊！！”
<br/>　　朱博文顿时发出一道凄厉的惨叫声，险些痛的晕厥过去。
<br/>　　一旁的白玲，看到这血腥的一幕，顿时脸色煞白，浑身乱颤，不由得死死抓住萧若曦的胳膊，生怕叶尘会找自己算账。
<br/>　　“小子！！你要么现在就杀了我，否则我朱博文发誓，一定会让你后悔！！”
<br/>　　这朱博文倒还有几分骨气，竟然没有开口求饶，转而出言威胁起来。
<br/>　　萧若曦听到朱博文这话，顿时脸色微微一变，
<br/>　　以她对叶尘的了解，杀人对他而言，简直就像是吃饭喝水一样简单，朱博文如此激怒叶尘，只怕今天是难以活命了。
<br/>　　然而，
<br/>　　出乎萧若曦的预料之外，叶尘两眼微微一眯，
<br/>　　“好！我等着！”
<br/>　　说完这话，叶尘又缓缓拿起，一旁桌上吃饭用的钢刀，随手插在了朱博文的另外一条大腿之上。
<br/>　　“啊！！”
<br/>　　朱博文再次发出一道凄厉的惨叫声。
<br/>　　叶尘一脸淡漠，转头望一眼萧若曦，以及早已经快要吓晕过去的白玲，淡淡道：
<br/>　　“走了！”
<br/>　　……
<br/>　　三人出了会所，
<br/>　　萧若曦早已经不止一次，见识过叶尘的手段，而且很清楚，以叶尘如今的地位，就算真的把朱博文杀了，也肯定不会有什么事。
<br/>　　可是白玲并不知道这些，此时整个人都快要崩溃了，看到萧若曦一脸淡然，顿时忍不住了，
<br/>　　“若曦，你这男朋友也太暴力吧！竟然把朱学长伤成这个样子？他这是犯罪！万一朱博文要是报警的话，可是要坐牢的！”
<br/>　　萧若曦却浑若无事，
<br/>　　“没事，你放心，他能搞定的！”
<br/>　　白玲顿时急了，
<br/>　　“你说的容易！若曦，你别犯傻了好不好？朱博文是谁？他叔叔可是咱们学校的校长，朱家在天海市，也是有名的豪门！”
<br/>　　“你男朋友把他伤成那个样子，不但学校不会放过他，朱家也不会善罢甘休？”
<br/>　　“他这个官司怕是吃定了，警察马上就会找上他，你还是赶紧和他划清界限的好！”
<br/>　　白玲一番苦口婆心的劝说，却见萧若曦根本没听进去，顿时又气又急，转而望向一旁的叶尘，道：
<br/>　　“喂，你叫叶尘是吧？我告诉你，你如果真的在乎若曦的话，这件事就不应该把我们牵扯进来！你自己做的事情，就要自己承担！”
<br/>　　叶尘继续缓步向前走着，理都没理。
<br/>　　白玲顿时气得不行，转而又开始继续劝说萧若曦，
<br/>　　“若曦，你倒是说句话啊！我知道你们萧家很厉害，可是这里毕竟是天南省，不是广南省！”
<br/>　　“万一到时候，朱博文把我们也一起告了，那可怎么办？”
<br/>　　萧若曦正要开口，一旁的叶尘忽然冷冷道：
<br/>　　“说到底，你不过就是担心自己的安危罢了！何必说的这么冠冕堂皇？”
<br/>　　白玲被叶尘一语道破心思，顿时有些恼羞成怒，
<br/>　　虽然她之前对叶尘的恐怖手段感到害怕，可此时三人在大街上，想来叶尘也不敢把她怎么样，于是胆子也就大了起来，
<br/>　　“人是你打伤的，本来和我们就没有关系！”
<br/>　　叶尘两眼一眯，眼眸中闪过一抹寒意，冷冷道：
<br/>　　“如果不是你把若曦哄骗过来，今天的事情，你觉得会发生吗？”
<br/>　　白玲顿时有些无言以对，继而越发羞恼，
<br/>　　“反正不管怎么说，人是你打伤的，警察如果真要是问起来，我肯定会实话实话的！”
<br/>　　“嗯？”
<br/>　　这个自私自利的女人，一再挑衅叶尘的耐心，顿时让他生出了杀意。
<br/>　　白玲顿时感觉浑身一哆嗦，仿佛整个人，一下子坠入了冰窖之中，浑身瑟瑟发抖，脸色瞬间惨白。
<br/>　　萧若曦似乎也感觉到叶尘身上的杀意，连忙将白玲拉到身后，
<br/>　　“叶尘，白玲一向心直口快，你别和她一般见识！”
<br/>　　见萧若曦求情，叶尘这才将杀意缓缓收起，
<br/>　　“白玲是吧？以后若是让我知道，你再敢利用若曦，后果自负！”
<br/>　　白玲经过方才那一吓，早已经心惊胆寒，哪里还敢和叶尘顶嘴，乖乖躲在萧若曦的身后，不敢再多说一句话。
<br/>　　连她自己也暗暗纳闷，刚才被叶尘盯着那一瞬间，她感觉自己就像是被一头恶魔盯上，似乎随时都能把自己撕的粉碎！
<br/>　　……
<br/>　　三人沿着街道，向前走了大约二十来分钟，眼看马上就要回到天海大学。
<br/>　　“乌拉~~”
<br/>　　后方忽然警笛大作，
<br/>　　紧接着，
<br/>　　三辆警车从后方呼啸而来，拦住了三人的去路。
<br/>　　白玲顿时脸色大变，
<br/>　　“完了！完了！肯定是朱博文已经报警了！”
<br/>　　就连叶尘也微微感叹，
<br/>　　天海市的出警速度真的很快，竟然在这么短的时间内，就锁定了他的踪迹。
<br/>　　当然，主要还是因为，叶尘没有刻意去躲，否则这天下，又有谁能找的到他？
<br/>　　哗啦啦！
<br/>　　七八名警察从警车上下来，
<br/>　　一名英姿飒爽的美貌女警，走上前来，在三人身上扫了一遍之后，明显露出一抹诧异之色，
<br/>　　“三位！刚刚雅尚会所发生打架斗殴事件，我们怀疑，这件事情和你们三位有关，请跟我们去警局走一趟吧！”
