<br/>　　武英可是这一届武斗大会上，几乎是有着资格冲击前十的天才。

<br/>　　夜天君面露狰狞的低吼道。
<br/>　　“我要亲手杀了他！”
<br/>　　“剩下的人，碰到他，先主动退避！”
<br/>　　楚风眠能够一招轰杀了武英，这种实力，已经不是剩下的圣党弟子，可以对付的了的了。

<br/>　　这一战，也让在场任何任何一人，还敢来轻视楚风眠了。

<br/>　　这三人的目光之中，蕴含的，只有杀意。
<br/>　　所有的天才，都不愿意会让一个人骑在他们头上，在这武斗大会上，所碰到的所有人，都是敌人。

<br/>　　“有意思，这楚风眠跟夜天君，可是有着大仇，他们两人总有一刻会碰到的，倒时候才是真正的大战！”

<br/>　　楚风眠斩杀了圣党弟子，夜天君本来就不会善罢甘休，更何况在传闻之中，楚风眠跟夜天君，本来就有着仇怨。
