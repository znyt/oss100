<br/>　　梧桐冷哼一声：“这样我名节也一样毁了。”
<br/>　　叶少阳道：“不会不会，你只跟他一个人说，男人都要面子，他绝对不会往外说的，到时候最多吃个哑巴亏。你觉得呢，我这办法真的不错的，一般人我不告诉他。”
<br/>　　梧桐脸上带着一副耐人寻味的笑容，道：“你救过我，开玩笑适可而止吧，你如果再说，我可就认真了。”
<br/>　　叶少阳一下子怔住，道：“我就是要你认真啊。谁跟你开玩笑了。”
<br/>　　“你不怕我听了你的提议，真这么去做？”
<br/>　　“我去，我说这个，就是要你听的啊。”
<br/>　　梧桐盯着他看了一会，看他表情认真，不像是闹着玩的，自己反倒有点吃惊，说道：“你的办法其实不错，我那么做，他顾忌自己的名节，肯定不会出去乱说，但是……他会杀了你。”
<br/>　　“哦，你是担心他来找我麻烦啊。”叶少阳恍然大悟。
<br/>　　“当然，你不怕么？”
<br/>　　“有什么好怕的，让他来就是了。”
<br/>　　梧桐有些无奈。“什么啊，他差不多是二代弟子中最强的一个，天选之子，你自己也是法师，如果他想针对你……你只是茅山外门弟子，没人能帮你扛，说的简单一点，你会死得很难看。”
<br/>　　叶少阳轻轻一笑，“这个，你不用担心，我既然敢帮你出主意，自然不怕他来找我麻烦。”
<br/>　　“为什么不怕？”梧桐很不解。
<br/>　　“为什么要怕。”叶少阳更不解，随即笑了笑，说道，“认真的，我真的不怕。我管他什么第一少年、天选之子，真的不怕，只要能帮到你就行。”
<br/>　　梧桐盯着他看了好一会，从他的眼中，只看到一片真诚，于是她信了。“可是，就算你不怕，但你为什么要帮我？”
<br/>　　叶少阳摸了摸鼻子，说道：“如果我说因为你长得像我一个朋友，你信不信？”
<br/>　　梧桐笑了，她当然不信，但是很少笑的她，居然对叶少阳笑了，是那种真诚的笑。
<br/>　　“你的主意，我会考虑的，如果真的有那么一天，我希望你不要反悔。否则我会很失望。”
<br/>　　“从不反悔，任何时候都行。”
<br/>　　梧桐笑笑，突然想起什么，说道：“你之前说，我们走的是工人走的路，那就是说，这里还有一个出口了？”
<br/>　　叶少阳也想起了这茬，只因为一进来就看到坛城，立刻被它吸引了注意，倒是把这个给忘了。
<br/>　　当下两人绕过坛城，在这件偌大的墓室里走动起来，在坛城的背后，找到了一座修成佛塔形状的拱门，里面很幽深，也不知道通往什么地方，干脆没有过去。
<br/>　　不过在墓室的另一侧，两人也发现了很多壁画，上面都是一些菩萨和佛像，或者一些凶神恶煞的连梧桐也认不出的神祇。
<br/>　　两人静静地等了一会，他们来的那条路上传来脚步声，等了没一会，有人走了过来，正是妙心等人，几个人拿着两盏灯台照亮。
<br/>　　叶少阳一眼扫去，一个人都不少，只是几人头发和衣服都有点乱，神色也比较疲惫，猜得到是跟那只小蝾螈大战的结果。陈晓宇的额头还伤了一块，流出的血都干了，看上去很狼狈。
<br/>　　看到梧桐和叶少阳，妙心和毛小方都很高兴，上来打招呼。
<br/>　　叶少阳听毛小方简单说了大战蝾螈的经过，虽然紧张，但不是很惊险，如叶少阳所推测的，这一帮法师，对付那只小号的蝾螈还是不在话下的。
<br/>　　陈晓宇阴阴地看着叶少阳，说道：“你倒好，既然安全了，为什么不去帮我们对付蝾螈，躲到这里来？”
<br/>　　叶少阳瞥了他一眼，“你们三个都是地仙准地仙的，一个小号的蝾螈都对付不了，还非得我亲自动手，我把大的都杀了，你们还要我怎么样？”
<br/>　　叶少阳本来是不想装这个逼的，但是陈晓宇咄咄逼人，那就只好借力打力一下。这一席话说得陈晓宇哑口无言。
<br/>　　“这是什么！”妙心发现了坛城，碰着一盏鲛油灯走了过去。
<br/>　　梧桐跟她说了坛城的作用和自己的推测。
<br/>　　妙心围绕坛城观察了半天，缓缓说道：“没错，这里本是龙脉，水月洞天，这坛城还有采集风水灵气的作用，加上自身的法阵，能更好地吸收这里的风水妙处……”说着转头看了看四周，道：“这么好的风水，再加上这座坛城，后代一定会出一个枭雄。”
<br/>　　叶少阳皱眉道：“为什么不是皇帝？这不是龙脉吗？”
<br/>　　妙心笑道：“龙脉是没错，不过帝王者，福临天下，容不得妖邪之气，这里为了这座坛城，杀了一百个人，煞气太重，后人就算出头，也只是个杀伐天下的枭雄，绝对当不了真正的皇帝。”
<br/>　　叶少阳还是好奇，道：“那为什么不直接弄个能出帝王的什么法阵之类的？”
<br/>　　妙心笑起来：“你以为，帝王的灵台就那么好搭吗？不是说给你龙脉，埋在这里就一定能当皇帝的，很多龙脉从外表看就是荒山野岭，不少当地人死了也都埋在那里，为什么没有出皇帝？”
<br/>　　一句话问得叶少阳说不出话来。
<br/>　　妙心本身是不大看得上叶少阳的，不过之前对抗蝾螈的那一战，叶少阳表现出了超乎他自身的能力，让妙心也对他刮目相看，彼此也亲近了许多，对他打开了话匣子，大致说了龙脉聚养帝王之气的灵台是怎么搭的。
<br/>　　“我听我爷爷说，大清国的龙脉，是在长白山龙脉下面，修建了一座金龙吐水，不过涂的不是水，而是金沙，并且有极其高超的工匠制作，金沙循环使用，不断被喷涂出来，将风水印活，长盛不衰……”
<br/>　　叶少阳听了这话，当场震惊。
<br/>　　“所以他们家做了几百年的皇帝？那难道古代那些开过皇帝，都是因为祖上大费周章地在龙脉埋过人？”
<br/>　　妙心忍不住发笑，“你本末倒置了。风水只有禳补，哪有能决定一切的，找龙脉埋先人，只是希望将来能出一位帝王，哪里是所有皇帝都是这么来的。
