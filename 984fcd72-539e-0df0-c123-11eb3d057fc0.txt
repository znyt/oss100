　　转世重生。
<br/>　　在上一世的时候，楚风眠也觉得这件事是虚无缥缈。
<br/>　　只流传于神话传说之中。
<br/>　　纵然是在古籍之中所记载的转世重生，记载的也都无比模糊，寥寥无几。
<br/>　　所以当时的楚风眠并不在意。
<br/>　　可这一次转世重生之后，也让楚风眠明白了过来，天下间许多虚无缥缈的事情，未必是假的。
<br/>　　就像是楚风眠，他可以转世重生，那未必没有其他人，可能做到转世重生，只是无人知道罢了。
<br/>　　像是楚风眠，也将转世重生，作为他的最大秘密，不敢暴露出来。
<br/>　　这个秘密，一直被楚风眠藏在心底，楚风眠也只敢在剑凌的面前，暴露出来了。
<br/>　　“踏入天人境，羽化登仙就足以转世重生？”
<br/>　　听到楚风眠的话，剑凌一愣，随即笑了起来。
<br/>　　“你可知道，我当初是什么境界？”
<br/>　　“天人境？”
<br/>　　楚风眠眉头一皱。
<br/>　　曾经剑道门的诸位师兄，都是九域赫赫有名的强者，但是他们的实力，楚风眠却一概不知。
<br/>　　那个时候的楚风眠，没有任何的力量，就连他们的实力都看不出来。
<br/>　　现在剑凌留下的禁制，在万年之后的威力，都足以逼退九劫古帝，剑凌前世的实力，只怕不是半步天人境，就是真正踏入到了天人境，羽化登仙的存在了。
<br/>　　“在前世，我是仙士境界。”
<br/>　　剑凌看着楚风眠笑着说道。
<br/>　　“仙士？仙士是什么境界？”
<br/>　　听到剑凌的话，楚风眠眉头一皱。
<br/>　　这个名词，他还是第一次听说。
<br/>　　“看来你还是不知道一些关于天人境的事，其实天人境，只是天位三境的第一境。”
<br/>　　剑凌看了一眼楚风眠，开口解释道。
<br/>　　“天位三境，分别是天人境，玄天境，至天境，这三重境界，也别称为仙人九阶。”
<br/>　　“仙人九阶，分别为仙人，仙士，仙将，仙侯，仙王，仙君，仙圣，仙尊，仙帝，这九阶分别代表九个称呼。”
<br/>　　“其中天人境，对应的便是仙人，仙士，仙将，前三重为被称为仙人，中三重为仙士，后三重为仙将。”
<br/>　　“玄天境，则是分为仙侯，仙王，仙君三个层次。”
<br/>　　“至天境，则是分为仙圣，仙尊，仙帝三个层次。”
<br/>　　“我前世的时候，达到了天人境六重巅峰，所以算是仙士境界的巅峰。”
<br/>　　楚风眠一直是安静的听着。
<br/>　　天人境之上的事，楚风眠还是从未听说过的。
<br/>　　对楚风眠前世来说，哪怕是帝尊境，对楚风眠都是虚无缥缈一般，更何况是成就天人，羽化登仙。
<br/>　　“就算是对天人境来说，转世重生，依然是虚无缥缈的一件事。”
<br/>　　剑凌感叹了一声道。
<br/>　　“师尊当年，也曾经说过，转世重生，虚无缥缈，就算是师尊，都触及不到。”
<br/>　　“师尊生前是什么境界？”
<br/>　　楚风眠突然问道。
<br/>　　就连剑凌，都是达到了天人境的话，那曾经的剑道之主……
<br/>　　要知道剑道之主，可是当时那个时代之中九域真正的第一人，所向无敌，从来没有任何一人，可以跟剑道之主争锋。
<br/>　　纵然是剑道之主陨落，也是被数百强者，联手围攻的结果。
<br/>　　“至天境巅峰，位列仙帝！”
<br/>　　剑凌沉声说道。
<br/>　　仙帝！
<br/>　　至天境巅峰！
<br/>　　纵然楚风眠已经是有了心理准备，听到这里也还是吓了一大跳。
<br/>　　仙帝，至天境巅峰，可以说是仙中之帝了，仙人九阶的最后一重。
<br/>　　楚风眠曾经只是猜想了，剑道之主可能是已经踏入天人境，羽化登仙，却未曾想到，剑道之主已经是迈入到了仙的巅峰。
<br/>　　可纵然是这样，剑道之主，竟然也陨落了？
<br/>　　“七大宗门，怎么可能杀的了师尊？”
<br/>　　楚风眠不由的开口道。
<br/>　　如果说剑道之主只是刚刚踏入天人境的级别，那死在七大宗门的围攻之下，还情有可原。
<br/>　　可是，剑道之主可是一位仙帝啊，七大宗门就算是加起来，都不可能是剑道之主的对手才对。
<br/>　　“看来师弟你只知其一，不知其二，其实当年杀了师尊的人，并不只是七大宗门，在他们的背后，还有一方势力的存在。”
<br/>　　剑凌突然语气变得冰冷无比，他一生想要报仇，却是未曾成功，才在建立了北邙学院，留下了传承。
<br/>　　“还有一方势力的存在？”
<br/>　　楚风眠眼神冰冷，他可不知道还有这件事。
<br/>　　“是谁？”
<br/>　　“天九域，师弟你可听说过天九域？”
<br/>　　剑凌开口道。
<br/>　　“天九域？传闻之中九域背后的哪一方世界？”
<br/>　　楚风眠想了一下，开口道。
<br/>　　在九域之中，一直有着一个传闻，便是在九域的背后，有着天九域的存在，哪里才是真正的九域，被称为圣域的九域。
<br/>　　而如今的九域，只是当初天九域的一部分，被隔绝了出来，随即衰败的。
<br/>　　“就是天九域的人跟七大宗门的人一起，杀了师尊！”
<br/>　　剑凌冷声说道，语气中充满了恨意。
<br/>　　“天九域，真的存在？”
<br/>　　楚风眠不由的脱口而出道。
<br/>　　天九域的传闻，在万年之前就有流传了，可真正天九域在哪里，一直也没有人寻找到过。
<br/>　　这种只存在于传说之中的圣域，竟然真的存在？
<br/>　　“这是师尊亲口告诉我的，师尊曾经得到了一件东西，就是因为得到了哪一件东西，最终才遭到了天九域的追杀，最终被逼入陨落。”
<br/>　　剑凌摇了摇头道。
<br/>　　“至于那是一件什么东西，师尊也没有告诉我，只是警告我，千万要小心天九域。”
<br/>　　“小师弟，你要小心，你的资质，天下无双，但是还是太小了，你的身份，千万不能暴露。”
<br/>　　“师尊所得到的那件东西，并未被天九域得到，天九域的人，曾经也追杀过我，想要得到那件东西，如果你的身份暴露，天九域的人，也不会放过你的。”
<br/>　　“我早已陨落，尸骨无存，这一缕灵力，也是我留下的念想，能再见你，已经是意外之喜了。”
<br/>　　“今日，我将消散，你要自己小心……”
<br/>　　剑凌的话，声音越来越小，最终彻底消失。
<br/>　　这眼前的剑凌，毕竟只是剑凌的一股灵力所化，终究会有消散的一日。
