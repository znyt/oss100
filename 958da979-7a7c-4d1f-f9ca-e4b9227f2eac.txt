
<br/>　　<br/>　　    王惊蛰也是吃惊的，只是按耐住了惊异，尽管已经听到了那第五道呼吸声知道神庙里会有第五个人，可是当看见对方的背影时，吃惊还是难免的，这里是天坑下，是藏区的无人地，一座不知道矗立了多少年的神庙里，怎么可能会有人呢。<br/>　　<br/>　　    然后，那人转过了身子。<br/>　　<br/>　　    这是一个你看不出什么年纪的老人，因为你根本就没办法判断他的年龄，他的头发很长，可能长到及膝，他的胡子很长，可能长到及膝，他的指甲也很长，倒是没有长的夸张到膝盖那里，但泛黄的指甲也弯曲了十几公分左右，明显很久没有修剪过了。<br/>　　<br/>　　    这是个很老的僧人，他的皮肤蜡黄，脸上布满了褶皱，嘴唇上都是死皮，看起来脏兮兮的，唯一干净的地方就是他那双眼睛，看着好像一汪清水，和他的外表一点都不相符。<br/>　　<br/>　　    这老僧人看起来很像藏区，尼bo尔，印du等地常见的那种苦行僧人，他们不拘小节不修边幅，赤着脚宣扬着自己的教义，对自身的认知就是觉得这是不过是一身皮囊罢了，没有什么可在乎的，他们崇尚的是灵魂上的修行。<br/>　　<br/>　　    这老僧人缓缓的站了起来，一手拿着经轮，一手竖在胸前，然后施了一礼，等他再次抬头的时候就忽然开口说话了，从对方的语调和声音上判断，王惊蛰觉得先前他听到的经文，就是这老僧人吟诵的。<br/>　　<br/>　　    “我刚才诵读的是古象雄佛法化身佛心咒，能够洗涤人的心灵，感谢你们于佛同在……”这老僧人讲的居然还是汉语，虽然有点绕口，但绝对能很轻易的听得懂。<br/>　　<br/>　　    神庙里，一个神秘兮兮的老人，似乎即将要揭开这天坑下神秘的面纱了？<br/>　　<br/>　　    陈三岁看了一眼王惊蛰，心道他怎么比我聪明了不少，还真是被他给说对了。<br/>　　<br/>　　    同时，三井美奈也望了过来，心道这个人太不简单了。<br/>　　<br/>　　    在进入神庙之前，王惊蛰对陈三岁，孔良和三井美奈说，这可能是苯教神庙的一种试探，或者说寻找有缘人比较合适，这跟喇嘛庙在寻活佛基本是一个道理。<br/>　　<br/>　　    那些疯了，不是，是失常了的人不是被逼的，恰恰相反是因为灌顶的原因而承受不住考验，才失常的。<br/>　　<br/>　　    进庙之后那无边无际的黑暗和寂静，算是先来了个下马威，当这老僧人诵读古象雄佛法化身佛心咒的时候，则是开始洗涤人的心灵，孔良没坚持住，崩溃了，再往前三井良河等人也是如此，再再往前，二战时期马丁鲍曼带来的那些士兵也是一样。<br/>　　<br/>　　    这算是一种考核，挺不住就崩溃，挺住了就像王惊蛰他们现在这样，神庙里亮了，老僧人转过了身。<br/>　　<br/>　　    这时候，自然就得面对下一项了。<br/>　　<br/>　　    王惊蛰猜测，神庙应该是在寻找转世或者有缘人，传承自己的教义，他是从小林昙一的魂魄上看出这一点的。<br/>　　<br/>　　    之前，王惊蛰推演，孔良和陈三岁包括他面临的是一个九死一生的局，所谓的九死就是这种考验，挺不过去gameover，挺过了的话，这间神庙的传人可能就此出现了。<br/>　　<br/>　　    这就是就死之后的一生，孔良肯定已经没机会了，他出局了，那自己和陈三岁谁会是？<br/>　　<br/>　　    有人可能会讲了，这苯教，这老僧人不是出家人么，为了寻找转世的有缘人，居然整死了那么多人，不是说我佛慈悲的么？<br/>　　<br/>　　    抱歉了，这是辛饶弥沃佛，可不是大慈大悲的小乘佛教，苯教里认为人身就是具皮囊，灵魂才是最洁净的，所以死了也没啥，不是还有魂呢么。<br/>　　<br/>　　    陈三岁小声问道：“你还要继续么？”<br/>　　<br/>　　    “你呢？”王惊蛰反问了一句。<br/>　　<br/>　　    陈三岁想了想，点头说道：“要的”<br/>　　<br/>　　    “那我也是”<br/>　　<br/>　　    陈三岁不解的说道：“我都看破生死了，无所谓的，反正我要是疯了后死去了，你就把我葬在诗远的旁边就行了，我是不怕死，你又图的什么啊？”<br/>　　<br/>　　    “我看破生死那时候你还在玩推油呢，你只是不怕死，而我呢……”王惊蛰顿了顿，意味深长的叹了口气：“我是相当不怕死啊”<br/>　　<br/>　　    王惊蛰心里还有一句，反正我也离死不远了。<br/>　　<br/>　　    老僧人听着他俩的对话没有插嘴，但也没有什么表情，三井美奈则是听的一脸懵。<br/>　　<br/>　　    王惊蛰和陈三岁对视一眼，含情脉脉。<br/>　　<br/>　　    苯教的老僧人伸手一礼，也没询问他们，就迈开脚步走了过来，当他走路的时候，王惊蛰低头看了一眼，对方迈开步子时脚就从僧袍中伸了出来。<br/>　　<br/>　　    两只脚，白皙的如同婴儿的皮肤，嫩的好像姑娘的脸蛋，一捏就能出水的那种嫩，吹弹可破。<br/>　　<br/>　　    这说明对方应该是有很多年都没有走过路了，你若不信就脱下袜子看看自己的两脚，脚底板上肯定都是死皮和老茧，这老僧人的脚却森白的有些吓人了。<br/>　　<br/>　　    老僧人先是来到了三井美奈的身前，她顿时就紧张兮兮的吓了倒退一步。<br/>　　<br/>　　    陈三岁扭头问道：“这是开玩笑么，僧人寻找转世活佛，居然会找个女人？”<br/>　　<br/>　　    王惊蛰说道：“在他们的眼中男人和女人都是人，没有任何的区别，再比如，有一种菩萨叫观世音……你应该听过吧？”<br/>　　<br/>　　    陈三岁顿时悟了，同时也对自己的反应迟钝和才疏学浅深深的鄙视了一下，心里叹了口气，这就叫不学无术吧。<br/>　　<br/>　　    那老僧人见三井美奈退了一步也没再动，伸出一只手递到了她的面前，然后没有任何表示的就那么伸着，眼睛很平淡的望着对方。<br/>　　<br/>　　    三井美奈不知所措，脸上表情阴晴不定的变换了几下后，她最终还是又回了过来，就见那老僧人轻轻的在她的头顶拍了三下，王惊蛰发现这老人的手掌上散发着柔和的一抹光。<br/>　　<br/>　　    “从前化身索郎翁仲前，发微妙心发殊胜大愿，三无量劫积累二资粮，成遍知佛导师无有上……”<br/>　　<br/>　　    老僧人闭着眼睛，嘴中念念有词，这次王惊蛰和陈三岁听清楚了，但没听明白这是哪一卷藏经。<br/>　　<br/>　　    随着老僧人念的速度越来越快，三井美奈眼中渐渐的失神了，到最后她也闭上了眼睛然后缓缓地盘腿坐在了地上，两手结下了一个个法印。
