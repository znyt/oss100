
<br/>　　<br/>　　    无论是现代社会还是古时候，不管是现在的特权事务处理办公室还是东厂西厂锦衣卫，无疑都是国家的强力权利部门，手中掌握着大量的资源和渠道，最关键的是还掐着一把尚方宝剑，所以这种国字头的单位的掌控力那是相当强悍的。<br/>　　<br/>　　    黄九郎就跟王惊蛰讲了，从六几年开始再往后，他们这个部门在民间就搜罗了不少的东西，其中也包括不少典籍和古书，光是这些东西的收藏量就比一个图书馆还庞大了，而且还一直都被封存着，从来都未向外展露过。<br/>　　<br/>　　    “当年秦始皇焚书坑儒，可谓是让文化产业大受打击啊，咱们国家呢六几年虽然在扫荡牛鬼蛇神，可也没蠢到要把这些文化瑰宝一把火都给烧了的地步，所以我们就全都封存了起来”黄九郎用手指敲着桌子桌子，意味深长的说道：“我记得吧自己就亲自参与过多次了，好像是在七四年夏天的时候，我们接到线报说在西北某地有人怪力乱神，是个五十几岁的老人，这人很有一些门道，当地民众都说他是老神仙转世，甭管什么事找他都很灵验，有谁撞了邪的话找到他，直接一碗香灰水喝下去顿时就没事了，有谁家丢了牛羊找他算算，他说在哪就是在哪，于是我们就赶了过去，看看这妖言惑众的人是什么来头，呵呵……这人自然不可能是什么老神仙转世，不过把他抓了以后送到牛棚里，我们就从他嘴中问了出来，他这些本事都是从一本叫阴阳手记上的书学来的，我后来翻看了一下，这书确实挺玄的，应该是不知道什么年代一个叫秋离道人的人写的”<br/>　　<br/>　　    王惊蛰叹了口气，说道：“你是想告诉我，类似于这种事还发生过不少，然后其中就有关于那张丹方的？”<br/>　　<br/>　　    “呵呵，都留下来了，我们可不想成为华夏民族的罪人啊，这些东西虽然不能公布于众，但还是要留下来的”黄九郎顿了顿，笑着说道：“挺凑巧的是，知道了你在寻那丹方的事之后，我就特意去库里查了下，我就想看看我们的库存中会不会有关于这方面的收录，你还别说……”<br/>　　<br/>　　    “还真让你找到点什么旁枝末节的线索？”王惊蛰打断他问道。<br/>　　<br/>　　    “喝茶，喝茶，你看你眼睛瞪的，都好像要吃人了”黄九郎笑了笑，端起茶杯朝他示意了下。<br/>　　<br/>　　    两人抿了口茶，王惊蛰心绪难宁，他知道无论自己在怎么掩饰，也很难在黄九郎的面前把他的渴望给藏起来了。<br/>　　<br/>　　    一杯茶喝完，王惊蛰知道自己掩盖不住了，就很坦诚的问道：“您说说看吧……”<br/>　　<br/>　　    “唐禾祥应该也不会具体有多了解，以他的经验估计只是猜测，能分析出个四五成左右？”黄九郎忽然探过身子，小声说道：“但是我却可以给你个肯定的消息”<br/>　　<br/>　　    “什么？”王惊蛰舔了舔嘴唇问道。<br/>　　<br/>　　    “我给你肯定的消息是这个方子我很了解，你想象不到的那种理解程度”黄九郎拍了拍手，随即摇头说道：“不过我现在跟你说什么都有点早，毕竟那两味药你很难寻得来，等你找到了以后，在联系我，剩下的我和你好好交代，交代”<br/>　　<br/>　　    王惊蛰鄙夷的说道：“你这不是和我开空头支票呢么？先把网给撒了下来，等着我往里钻啊？”<br/>　　<br/>　　    “啪”黄九郎忽然从身上掏出一张照片，放在桌子上指了指。<br/>　　<br/>　　    王惊蛰一低头，仔细端详了两眼，顿时震惊的抬起头，禁不住的咽了口唾沫，黄九郎给他看的这张照片上面还有着日期，显示是昨天刚拍的，照片里面内容很简单，只有一个古朴老旧的石鼎，但这个鼎王惊蛰却不陌生。<br/>　　<br/>　　    那条白蛇送给他的方子外面，有个盒子，盒子的一面上印着的鼎和这张照片里一模一样。<br/>　　<br/>　　    黄九郎眯着眼睛说道：“我可以明了告诉你，你寻到了那几味药，最后如果要提炼的话，也只能在这个鼎里炼，炼丹炼丹，可不是熬中药啊，你找唐禾祥他都得束手无策，那丹方得要配着这个鼎才行，否则无论是火候还是力道，你都掌握不好，那可就功亏一篑了”<br/>　　<br/>　　    王惊蛰长长的吁了口气。缓缓的点头说道：“我信了！”<br/>　　<br/>　　    “呵呵，回见吧……”黄九郎起身拍了下他的肩膀，说道：“顺便跟你说一嘴，有什么需要和条件我全力支持，暂且我们部门就把你当个编外的特派人员吧，我尽可能的对你敞开一些有用的资源和信息”<br/>　　<br/>　　    “这话，真中听！”王惊蛰点头说道。<br/>　　<br/>　　    黄九郎走了以后，王惊蛰坐在茶楼里半天都没有离开，一直在思索着对方给他带来的消息，他料想最后事成了的话，黄九郎可能也是想要分一杯羹的，从这一点上就能看出，这个丹方真要是提炼成了，其作用恐怕要相当的惊人了。<br/>　　<br/>　　    和黄九郎见完面后，王惊蛰就跟陈三岁启程了，他俩没有直接飞往雪域高原，因为这个高原的面积太大了，想要地毯式的收缩无疑于相当在大海捞针，找一株可能都消失了不知道几百上千年的草药，用两条腿和一双眼睛肯定不靠谱。<br/>　　<br/>　　    所以，王惊蛰和陈三岁先是飞往了天水，在这之前发动了不少的关系询问和打听，以求能够找到哪怕一丁点的蛛丝马迹和线索。<br/>　　<br/>　　    他俩到了天水后，机场外面就有人开了一辆兰德酷路泽等着了，这是临下飞机前王惊蛰跟黄九郎要的，这一路上他们无论是赶路还是寻找龙骨草，都得有一辆车代步才行，有关系不用那是浪费，所以就要了一辆堪称越野之王的酷路泽。
