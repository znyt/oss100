　　第342章
<br/>　　众人都屏住呼吸，且要看程大雷降服黑牛怪，暗中都有比试较量的意思。刚才关鱼和秦蛮都试过，皆收拾不了这黑牛，说不定程大雷和大家一样丢人呢。
<br/>　　冷不防程大雷突然喝了一声，然后大家惊讶的看到，刚才还呼哧呼哧喘气，浑身像冒火的黑牛瞬间安静下来。在一双牛眼看到程大雷时，目光似乎变得柔和，慢慢走到程大雷身边，打了个响鼻，似乎是向程大雷谄媚一般。
<br/>　　“咦！”众人齐声发出惊呼。
<br/>　　程大雷哈哈大笑，手一拍牛头，翻身跃在牛背上，返回将军府。
<br/>　　等程大雷穿好盔甲，骑着黑牛从将军府出来时，只见门口围了一大片人。众人窃窃私语，交头接耳议论着什么。
<br/>　　见到程大雷出来，都是眼前一亮，程大雷当真好扮相。
<br/>　　盔甲像是为他量身定做的一般，只见他身披黑甲，脚蹬皂靴，浑身上下一片漆黑。整个人和身下的黑牛一样气质，都是又丑又凶。
<br/>　　不怪众人奇怪，刚开始看到这黑牛，见它身上负着盔甲，还以为是谁家走丢的，可为何这黑牛别人不认，专门找程大雷。而这盔甲程大雷穿在身上，是不是也太合体了些。
<br/>　　“大当家，你以前认识这黑牛？”有人问出声来。
<br/>　　“不认识呐。”程大雷乍穿新衣高仰脸，还沉浸在自我欣赏中。
<br/>　　“那么，那么它为何专门来找你的？”众人心中却愈发奇怪。
<br/>　　“这……”程大雷一时语结，系统以前送什么东西，都是悄悄摸摸，这次完全是明目张胆，让黑牛托着装备过来了。如此招摇，倒是令程大雷不太好解释。
<br/>　　“哈！”吴用突然分开人群占出来，道：“诸位怕还是不知道，大当家一身本事乃是仙人传艺，太白金星与大当家义结金兰，天蓬元帅和他称兄道弟。定是上天感应，差黑牛前来给大当家献宝。这黑牛……是太上老君那头。”
<br/>　　呃……
<br/>　　程大雷咧咧嘴，真能扯啊。
<br/>　　他看向周围，发现吴用一席话说话，周围人看自己的目光充满了畏惧，又惊又惧。
<br/>　　吴用这席话当真还有人信，大家普遍文化水平不高，更何况今天这件事的确有些玄异。就连徐神机都含糊了，更何况其他人。
<br/>　　程大雷想了想，还是没有纠正吴用。本来么，古往今来不缺这些套路，陈胜还搞出陈胜王之类的，后来也有石人一只眼，挑动天下反……能借此收一收民心，程大雷何乐而不为。
<br/>　　黑牛驮着盔甲过来，还真有些仙人指路，异兽献宝的意思。如此，程大雷兵器、盔甲和坐骑都齐了。
<br/>　　如今的程大雷，早不是当初那个吃了上顿没下顿的破落的户，如今的他是要人有人，要装备有装备，身上简直可以说是一身橙装。
<br/>　　座骑是墨玉乌月兽，顶级，附带技能：野蛮冲撞、石力。
<br/>　　兵器有二，一是鬼面斧，附带技能热血，二是匹夫剑，明帝亲自监造，存世只有一柄，都是来头不小。
<br/>　　盔甲是照月蛤蟆甲，顶级，附带技能：坚体（每十二时辰抵挡一次致命攻击）。这蛤蟆甲通体漆黑，穿上后保暖又透气，一口气上五楼一点儿不费劲。
<br/>　　当然啦，在后面自然而然有一只丑陋的癞***案。
<br/>　　程大雷回到将军府，在铜镜前照了照，无语的点点头。说实话，自己穿上这套盔甲，威风是真威风，但丑……也是真丑。
<br/>　　骑着黑牛，拎着大斧，身上还一套黑漆麻溜的盔甲。想来自己不可能是赵子龙那样的白袍小将，也不可能是李行哉那样放浪不羁的王孙贵族……人设只能是又丑又猛的山贼。
<br/>　　当一个山贼也不错啦，天不管地不管的，想做什么做什么。
<br/>　　程大雷在小桃和柳芷的伺候下把盔甲脱了，谁没事穿这个，以后自己上战场的机会怕是会越来越少，多往幕后黑手方向发展。
<br/>　　黑牛交给银眸，由她平时管着，这种事倒也不需要程大雷操心。
<br/>　　程大雷来到书房，把柳芷和小桃哄了出去，自己打开了系统界面。
<br/>　　他进入了系统商店，果不其然，被收了十万恐惧值的过路费。
<br/>　　商店升级，这过路费也水涨船高，很明显，这系统比自己会打劫。
<br/>　　程大雷更关心的是，开四级本后，商店冒出了什么东西。
<br/>　　果然，系统商店的出售的商店，由一样变成了三样。
<br/>　　皮甲：售价一万恐惧值。
<br/>　　青苓散：普通药物，售价一万恐惧值。
<br/>　　连击弩：顶级武器，售价十万恐惧值。
<br/>　　皮甲是蛤蟆寨的特产，而青苓散和连击弩则是琴川关特产。程大雷猜测得不错，一地有一地的特产，而山寨级别越高，商店出售的商品就越高级。
<br/>　　当然，过路费也越贵，售价也越贵。
<br/>　　这连击弩很明显就比皮甲和青苓散有威力么。只是没开出火药，程大雷还是很遗憾的。
<br/>　　只有热武器，才能带来碾压力量呐。只是不知道怎样才能买到火药，总不能一个个山寨升级，升到三级看开不出火药，再升下一座……
<br/>　　这也不是一个办法。程大雷心里琢磨着，不如把青牛山的蛤蟆寨捡起来，那下面可还埋着宝藏呢。
<br/>　　当然，两地遥隔万里，程大雷这个念头暂时也只是想想而已。
<br/>　　青苓散和连击弩程大雷都买了一些，十万恐惧值的过路费已经交了，下次再进来还得付。
<br/>　　青苓散先不去管他，疗伤药日后总用得着。程大雷重点研究这连击弩，既然称得上顶级的武器，自然差不到哪里。
<br/>　　连击弩不算太大，双手就可操纵，五十步……大概也就是二十米左右，能造成致命伤，这威力甚至比不上弓箭。
<br/>　　但连击弩的优势是体积小，方便携带，一次装填，可以射击十次。
<br/>　　程大雷准备把这个交给工匠组，看他们能不能仿制出来，毕竟十万恐惧值一件，价格还是有些贵的。看上去似乎也无什么技术难度，如果真的可以仿制，程大雷琢磨着倒是可以培训一支特种作战部队。
<br/>
