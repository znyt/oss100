　　水银的熔点是零下39摄氏度，而液氮的沸点是-196.56℃，在液氮的浇灌下，水银平衡装置被冻的跟砖块一样坚硬，没有了水银平衡装置的威胁，李然很顺利的把塑胶炸药从油气管道上拆了下来，只是塑胶炸药上的计时器却还在进行着倒计时。
<br/>　　五分钟，计时器上还剩下五分钟的倒计时！
<br/>　　李然把塑胶炸药交给一名战备一队队员，让他把塑胶炸药带去厂区外，准备对塑胶炸药采取主动爆破。
<br/>　　“等一下。”
<br/>　　李然的耳麦中突然传来林晓阳的声音，“把塑胶炸药带去安全地方后，试着用液氮对电路板惊醒冷冻，如果可以将电路板破坏，把这个塑胶炸药带回来给我，可以把它研究透彻的话，可能会对我们有所帮助。”
<br/>　　李然看了眼那个战备一队队员，说道：“按照蜂鸟的要求去做。”
<br/>　　另外一组被派去原油库寻找塑胶炸药的战备一队队员很快也传回来了消息，他们也找到了塑胶炸药并且按照林晓阳所说的方法，使用液氮将塑胶炸药上的水银平衡装置进行冷冻，然后拆除送去安全地带进行爆破。
<br/>　　低温状态下电路板上的锡焊会变脆掉落，在液氮倾倒中，那块从油气管道上拆除下来的塑胶炸药上的电路板便是被破坏掉，计时器上的倒计时也是变成了几根红色的线条。
<br/>　　李然敲开水银平衡装置外的有机玻璃管，把里面凝结成块的水银清理出来后，才把这块即便重新装上引信后都不知道还能不能引爆的塑胶炸药放置在一个防爆箱内，带回指挥中心交给林晓阳那家伙研究去。
<br/>　　……
<br/>　　指挥中心，休息区。
<br/>　　罗昊回来后就一直在房间里照顾着云熙，原本罗昊以为云熙很快就会苏醒过来，可是一直等到天黑，云熙都没有丝毫要苏醒的迹象，罗昊去吃过晚饭回来后，发现云熙居然发起了高烧，这让罗昊紧张起来。
<br/>　　这次前来K国，龙隐部队这边并没有随队医生，所以只能把云熙送往就近的医院。
<br/>　　病房内，云熙在挂上点滴后，高烧很快就退了下去，但是她却似乎是做起了恶梦，两只手挥动着，嘴里不停的重复着“老师快逃”。罗昊抓住云熙挂着点滴的手，防止血管被针刺破，造成皮下出血。
<br/>　　赵天宇推开病房门走进病房，对陪在床边的罗昊问道：“丫头现在情况怎么样？”
<br/>　　“高烧是下去了，但一直在做恶梦。”
<br/>　　罗昊很是担忧的看了眼躺在病床上的云熙。
<br/>　　“这丫头跟子午之间不仅仅是师生，子午对于她来说也是半个父亲，子午的死对她的打击很大。”赵天宇叹了口气，说道：“回国的事情，我已经安排好了，明天中午的飞机，今天晚上你就留在这里陪她，我回指挥中心把我们的东西收拾好送去机场，明天上午会有车来接你们去机场。”
<br/>　　罗昊点了点头，把赵天宇送走后，又回到病床前，握着云熙的手，轻吻着她冰凉的手背。
<br/>　　……
<br/>　　K国总统府邸。
<br/>　　夜已深，但K国总统卡尔还在办公桌前处理着工作。
<br/>　　这次事情，让K国政府遇到了前所未有的信任危机，民众对政府的信任程度降到了历史最低点，这两天每天都会有民众来到总统府邸外高举着写着标语的牌子静坐示威，令得卡尔根本无法离开总统府邸，只能让人把东西送到总统府邸来，在总统府邸中进行办公。
<br/>　　处理完手中最后一份文件，卡尔放下手中的钢笔，双手揉着自己的太阳穴，显得十分疲惫。国会那边给了卡尔很大的压力，要求卡尔在半个月内解决这次K国民众的信任危机，如果卡尔无法在半个月内把事情解决，国会将启动弹劾预案，对卡尔进行弹劾，以他的下台作为处理这次信任危机的方案。
<br/>　　当人坐上权利最高的位置后，不会愿意轻易的离开。权利对于任何人来说都充满着致命的诱惑力，卡尔自然也是不愿意就这么被弹劾下台，可是如果不想被弹劾，他就必须要想出解决的办法！
<br/>　　而国会那边显然已经是做好要弹劾卡尔的准备，因为今天下午在写字楼连续发生爆炸后，聚集在总统府邸外的示威群众中喊出了让总统下台的口号，很显然这是卡尔那几个竞争对手授意为之。只要卡尔被弹劾下台，他们中间必然有一个人会上台接任总统之位，为他们所代表的党派获取最大化的利益。

<br/>　　啪！
