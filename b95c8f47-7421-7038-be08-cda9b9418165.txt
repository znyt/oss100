　　吼……一声巨吼震耳欲聋，众人呆滞当场。眼前这头狰狞巨兽，就是青云镇山灵兽水麒麟。
<br/>　　伴随一声吼叫，众人都不敢动。水麒麟硕大眼珠左右扫射，在众人之间不住打量。然后突然欺身而进，巨大头颅靠近张小凡，不住的抽动鼻子好似在闻什么东西。
<br/>　　苏重恍然，看向别再张小凡腰后的烧火棍。这东西是至凶至邪的法宝，被水麒麟发现不足为奇。
<br/>　　不过，两者巧妙融合为一，煞气内敛，任凭水麒麟感知灵敏，也只能捕捉一瞬间的气息，无法明确分辨气息来源。
<br/>　　苏重知道会没事，还有心情大量眼前这个庞然大物。
<br/>　　“这可是真的灵兽啊，比神雕他们厉害多了。”苏重两眼放光，他是真想抓一头回去当宠物。
<br/>　　神雕他们本质上是普通动物，只是经过本源多次孵化才显出神异。拥有异于常兽的力量和寿命。水麒麟是天生灵种，本身就有不凡血脉，神通惊人寿命悠长。
<br/>　　眼前这头水麒麟，甚至是组成诛仙剑阵的一部分！
<br/>　　诛仙剑阵运用凶戾气，凌厉霸道。如果太极玄清道修为不够，强行运用只会被煞气入心走火入魔。可眼前这头灵兽，却敢把诛仙剑放进嘴里丝毫不受影响，可见不凡。
<br/>　　苏重正在赞叹，眼前突然就出现一个篮球大小眼珠子。橙黄色透明眼珠里满是疑惑不解。
<br/>　　苏重浑身一僵，怎么跑自己面前来了。
<br/>　　水麒麟左眼睛看了换右眼睛，鼻子嘴巴齐上阵，就是绕着苏重左右打量。
<br/>　　不是张小凡有问题吗，逮着我不放是几个意思？难道这货已经那么厉害，能看出我是个异界来客？
<br/>　　“不可能的，有破界珠掩护，消耗本源遮掩，旁人是没办法发现我的身份的。”苏重心下稍安，仔细打量眼前灵尊。
<br/>　　这是他才想到，这家伙看似庞大，实际上还没完全成年，甚至来说只是幼年。如果是成年水麒麟，张小凡那根烧火棍就不一定能瞒过去。
<br/>　　可这家伙盯着自己干什么。
<br/>　　灯笼大的眼睛对着苏重左看右看，然后有用鼻子使劲闻了闻。水麒麟鼻孔翕张，猛然缩回脑袋，砰的一下打了个大大的喷嚏。
<br/>　　肉眼可见的水珠迎面喷了过来。苏重脸色大变，伸手就拉过站在旁边看热闹的杜必书，一下就窜到他身后。
<br/>　　噗嗤一声，杜必书就被浇了个满身通透！
<br/>　　杜必书这会儿想死的心都有，可看着近在咫尺的灵尊，他只能强压住逃跑的心念，战战兢兢的不敢动弹。
<br/>　　水麒麟最后看了眼苏重，晃晃悠悠走回水潭。
<br/>　　一道剑光从天而降，掌门道玄突兀出现，凭空站在水潭之上，施展法诀开始和灵尊沟通。
<br/>　　也不知道说了什么，一人一兽嘀咕半天，水麒麟晃了晃大脑袋，噗通一声就跳进水里，溅起的水花泼了众弟子满身。
<br/>　　苏重见机的快，又把杜必书拉倒了身前帮他当了一次灾。
<br/>　　“师弟，你怎么老是拿我挡着啊。旁边不还有大师兄吗。”杜必书满是怨气。
<br/>　　苏重瞥了对方一眼：“我为什么拿你当挡箭牌，你心底没点儿逼数吗？”
<br/>　　杜必书一滞，心说自己就是手贱，刚才在天上没事去吓唬他干嘛。报应来了吧！
<br/>　　他倒是想躲，但作为聚元功创始人，苏重身体素质那可不是闹着玩的。杜必书反应不如苏重，力量更比不过。他刚才就想站在原地不动，可架不住苏重力量大，直接就把他给拽到身前。
<br/>　　造孽啊！
<br/>　　宋大仁同情的看着杜必书，这家伙还不知道，自家小师弟已经在琢磨修改黑竹阵法，准备给他来个单间。真要知道，他估计自家六师弟想死的心都有！
<br/>　　不知道道玄最后和水麒麟交谈了什么，跳入水潭的灵尊很快就潜入深潭内不见踪影。
<br/>　　道玄回头看了眼众人，没发觉异常，吩咐弟子继续登山，然后就化作一道剑光飞遁而走。
<br/>　　苏重长出一口气，看了眼平静水潭若有所思。然后就跟着一种师兄弟离开，前往属于自己等人驻地。
<br/>　　……
<br/>　　通天峰大殿，七脉峰主全都在大殿内端坐等待。
<br/>　　一道剑光闪过，道玄出现在大殿之内。
<br/>　　苍松道人立刻起身：“掌门师兄，可知灵尊为何异动。”
<br/>　　道玄回到座位上，脸上带着思索道：“我施展法术和灵尊沟通，只能得到些模糊意思。灵尊好像发现了什么奇异气息，但具体也没查出来气息来自哪里。”
<br/>　　“我看灵尊好似对大竹峰的几个弟子格外重视。”苍松瞥了一眼田不易突然道。他们几个峰主刚才在大殿，通过法术把山下事情看得一清二楚。
<br/>　　当然知道灵尊不住打量张小凡和苏重的事。苍松这是故意恶心田不易呢。
<br/>　　往常时候，田不易总是和苍松不对付，只要开个头，田不易必然会脸红脖子粗的吵闹不休。
<br/>　　这次不仅没有气恼，反而得意洋洋：“那说明我的弟子天赋异禀，是不可多得的天才啊！”
<br/>　　苍松不屑一笑：“天才？呵呵……”
<br/>　　他龙首峰弟子才是真正的天才，自己悉心调教，每次七脉会武总能压过大竹峰一头。这个田胖子竟然还有脸说自己弟子是天才。
<br/>　　“怎么就不是天才，你的天才徒弟可不就打不过我徒弟吗。”田不易脸上笑容更盛。
<br/>　　苍松脸色一滞，心中怒气升腾。想到爱徒林惊羽上次狼狈模样他就来气。
<br/>　　“灵尊不看中别人，单单针对大竹峰两人。这二人说不定有问题。”苍松冷笑看着田不易。
<br/>　　“有没有问题不知道，可你徒弟打不过我徒弟。”田不易乐呵呵
<br/>　　“如果是魔门奸细，这种实力高强弟子更应该早作应对！”苍松就看不惯田不易这死胖子得意，张嘴就要搞事情。
<br/>　　田不易丝毫不以为意：“是不是魔门奸细，掌门师兄法眼无差，绝不会让魔门奸细潜伏青云。可不管怎么样，你徒弟就是打不过我徒弟！”
<br/>　　小竹峰峰主水月大师一脸厌烦的瞪着身边这个死胖子。当年就是这种死皮赖脸模样才把自家师姐拐走。看着这无赖模样她就心烦，恨不得一剑活劈了对方。
<br/>　　“好了，两位师弟不要争吵了。他们三人都是草庙村遗孤，身家清白，不可能是魔门奸细。大概是灵尊感觉错了吧。”道玄挥挥手笑盈盈的打圆场。几个师弟争吵已经不是一次两次，和起稀泥来他轻车熟路！
<br/>　　田不易笑的越发畅快，得意洋洋看着苍松。
<br/>　　苍松忍下一口怒气，冷冷道：“希望比武的时候，你还能这么得意。”他对自家弟子极为自信。就大竹峰那几个歪瓜裂枣，他一个都不放在眼里！
