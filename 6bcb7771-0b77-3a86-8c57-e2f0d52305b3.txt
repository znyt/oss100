　　叶尘因为是倒数第一，所以是最后一个进入空间门的。
<br/>　　整个迷罗星海，从外至内，一共分为九重星域！
<br/>　　而越靠近外围的星域，星球数量就越多。
<br/>　　叶尘记得，第一重星域，应该是128颗星辰，然后每深入一重，星球的数量就会减少一半，到了第八重星域，便只剩下一颗星球了。
<br/>　　前世的叶尘，第三次进入迷罗星海的时候，最终止步于第八重，至于第九重是什么样子，应该没有人知道。
<br/>　　因为根据之前吞天蟒所说，只要有人进入第九重，就能得到昊天大帝的传承，那迷罗星海自然也就没有存在的必要了！
<br/>　　所以，无论是前世还是今生，应该都没有人得到过昊天大帝的传承。
<br/>　　就在叶尘踏入空间门之后，这道空间裂缝便开始缓缓合拢。
<br/>　　而此时，远在初始星之外的无尽虚空之中，突然出现了几道，强横到极致的气息！
<br/>　　似乎在这几道强横气息的面前，连周围的星球，都要被他们生生压碎，虚空都要为之静止！
<br/>　　“公孙宗主！看来你那徒弟碰到对手了！那小子如此羞辱你们擎苍宗，你也能坐得住？”
<br/>　　一道沧桑中充满威严，又略带调侃的苍老声音，陡然在无尽虚空中幽幽响起，周围一些离得近的星球，顿时一阵簌簌颤抖，仿佛无法承受住声音主人的威压，随时都要裂开一般。
<br/>　　这说话之人，身穿一身淡蓝色的道袍，道袍上则是一个太极鱼的图案，看上去也就是一个普普通通的老道士，但是昂然立在虚空之中，如履平地。
<br/>　　“嘿！玉不琢，不成器！我这徒儿什么都好，就是脾气过于暴躁，若是用那小子，能够磨磨他的脾气，也未尝不是一件好事！区区一个小辈，哪里值得老夫亲自动手？”
<br/>　　又有一道苍老的声音响起，而这道声音的主人，则是一名身穿金色长袍的中年人，但是若论气息，竟丝毫不比之前那名老者差！
<br/>　　原来，这金色长袍的中年人，正是八大仙门之一擎苍宗的宗主公孙无我！
<br/>　　至于那道袍老者，则正是无极宗的宗主，云苍生！
<br/>　　而另外，还有四道强悍的气息，则同样是其他八大仙门中强者。
<br/>　　如果让叶尘那些人知道，八大仙门宗主这种巨擘级的大人物，竟然在秘密关注着他们，不知道又会作何感想？
<br/>　　公孙无我说完那话之后，话锋忽然一转，接着又道：
<br/>　　“云宗主，反倒是你，老夫听说，那灭世魔刀，就在刚才那小子的手里？你居然还能坐得住？真是让老夫佩服啊！！”
<br/>　　公孙无我的声音落下之后，另外四名强者，顿时也纷纷议论起来，
<br/>　　“哦？还有这等事？”
<br/>　　“灭世魔刀，竟然在那小子的手里吗？”
<br/>　　“刚刚那种情况之下，他竟然可以忍住没用？”
<br/>　　“话说这小子到底是什么路数？我观他的气息，似正非正，似邪非邪，竟然完全看不出此子的来历！”
<br/>　　……
<br/>　　听到众人的议论，云苍生哈哈一笑，洒然道：
<br/>　　“管他是什么来历，不过就是一个小辈而已！正如公孙宗主所说，哪里值得我们亲自出手？小一辈的事情，就让小一辈去解决吧！我们只管静观其变即可！”
<br/>　　……
<br/>　　叶尘一脚踏入空间门，
<br/>　　只感觉眼前空间一阵晃动，
<br/>　　哗啦啦~
<br/>　　下一刻，
<br/>　　他已经出现在一个陌生的地方，而且全身上下，似乎突然多了一道无形的枷锁，将他的全部修为，都尽数封印了起来！
<br/>　　叶尘瞬间从元婴境巅峰境界，沦为了一个毫无任何修为的普通人，连一丝一毫的真元，也施展不出来了。
<br/>　　非但如此，连同乾坤镯里的各种法宝，也被一股无形的力量，全部封印了起来，无法动用。
<br/>　　“昊天大帝的神通手段，果然是高深莫测啊！”
<br/>　　叶尘不由得再次暗暗感慨了一句。
<br/>　　要知道，哪怕是渡劫帝君到了这里，全身修为同样会被封印起来，根本没有任何反抗的余地！
<br/>　　不过好在，叶尘的真元虽然被全部封印，但是他的肉身强度，在修炼过九天焚身诀之后，已经比普通的返虚真君还要强大，相比于其他人，反而有着很大的优势。
<br/>　　就在叶尘正思量之时，脑海之中忽然多出了一道信息：
<br/>　　“以一个时辰为限，每斩杀一头妖兽，便可恢复一定数量的真元！”
<br/>　　叶尘先是一愣，继而大喜过望，
<br/>　　迷罗星海每次开启，考验规则都不相同，根本毫无任何规律可言。
<br/>　　而这一次的考验规则，显然对叶尘极为有利！
<br/>　　只要斩杀妖兽，便能恢复真元，而且斩杀的数量越多，真元恢复的也就越多。
<br/>　　眼下其他人，真元肯定也已经被全部封印，而肉身强悍的人，在这一关，显然具有很大的优势！
<br/>　　“吼！”
<br/>　　这道念头才刚刚在叶尘脑海中升起，伴随着一道咆哮声，一旁的密林之中，竟然窜出了一头妖兽。
<br/>　　这妖兽，形状似狼，高达四五米，一声青色毛发，根根倒竖，浑身犹如长满了钢刺！
<br/>　　青刺妖狼，二阶妖兽！
<br/>　　“来得好！”
<br/>　　叶尘不惊反喜，不但没有退缩，反而主动迎了上去，
<br/>　　“吼！！”
<br/>　　那青刺妖狼张着血盆大口，朝叶尘猛扑了过来，想要一口将其吞入腹中。
<br/>　　叶尘猛然加速，同时身体一矮，沿着地面滑行了数米，刚好躲过了青刺妖狼的利爪和巨口，而且一举冲到它的下腹部，然后一拳朝它的腰部狠狠轰落下去！
<br/>　　铜头、铁骨、豆腐腰！
<br/>　　任何狼类妖兽，它们最大的弱点，便是腰部！
<br/>　　纵然叶尘的肉身强度不弱，可是没有了真元的加持，想要正面斩杀一头二阶妖兽，还是要费一点功夫的。
<br/>　　现在时间有限，容不得他慢慢猎杀，所以必须找到对方的弱点，一击必杀！
<br/>　　砰！
<br/>　　叶尘这一拳砸出，狠狠击中在青刺妖狼最软弱的地方。
<br/>　　“嗷！！”
<br/>　　青刺妖狼顿时发出一道凄厉的惨叫声，那庞大的身躯，竟然被叶尘一拳轰飞了出去，继而蜷缩在地上，喘息了片刻之后，便彻底没有了气息。
<br/>　　嗡~~
<br/>　　青刺妖狼被叶尘一招斩杀之后，顿时化作了一道光芒，飞射进入叶尘的体内。
