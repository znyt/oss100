　　燕古，牧野战。
<br/>　　这两位新一届的天才弟子，如今全部都拜倒在了楚风眠的手中，跪在了地上。
<br/>　　如今这二人，身为长老弟子，本事来挑衅楚风眠，耀武扬威的。
<br/>　　却是谁都无法想到，居然是会落到了这样一个结局，全部都被楚风眠镇压。
<br/>　　在这众目睽睽之下，跪倒在了地上。
<br/>　　看着周围无数北邙学院弟子的目光，他们二人内心的耻辱，简直到了极点，恨不得要自杀。
<br/>　　对他们这些天才来说，尊严，比起生命还要重要。
<br/>　　可现在在楚风眠灵力的镇压之下，他们二人，就算是想要自杀的机会都没有。
<br/>　　“你们就在这里，好好的跪着反省吧。”
<br/>　　楚风眠瞥了那燕古，牧野战一眼，冷冷的开口道。
<br/>　　毫不理会他们二人，直接对着旁边的白萱说了一句。
<br/>　　“走吧。”
<br/>　　两道身影，直接离开，只留下那燕古，牧野战二人，跪在地上，被无数北邙学院弟子围观。
<br/>　　“这楚风眠的实力，只怕是足以达到种子弟子的地步了吧。”
<br/>　　“很有可能，这种实力，晋升，已经是随时可能的了。”
<br/>　　“这楚风眠跟白萱，不知道是得到了什么奇遇，居然是一飞冲天。”
<br/>　　一尊内门弟子，嫉妒的发狂，本来白萱的实力，也就是跟他在伯仲之间。
<br/>　　而楚风眠，更是他从来都看不起的一尊小人物。
<br/>　　可现在这小人物，都是远远的超越了他，至于白萱，虽然没有出手，可是他也能感觉得到，现在白萱的实力，已经跟他不是一个级别了。
<br/>　　“不要去管了，这二人只要是晋升成为了种子弟子，都将会成为我们北邙学院的大人物了，地位不在许多长老之下，我们只能去多巴结巴结了。”
<br/>　　一尊尊武者，纷纷开口道。
<br/>　　墙倒众人推，本来楚风眠跟燕古对战，众人都是站在燕古这一边的。
<br/>　　可是现在楚风眠表现出实力之后，他们也都纷纷的站在了楚风眠的这一边，毕竟实力为尊。
<br/>　　拥有实力，也就拥有了地位，没有人还敢看不起楚风眠这个外门弟子，人人敬仰。
<br/>　　“看来在这北邙学院，真是不能隐忍。”
<br/>　　楚风眠看着众人的变化，笑了笑道。
<br/>　　越是隐忍，也就越是被人欺凌，只有足够强势，表现出自己的实力，在这北邙学院才有着真正的立足之地。
<br/>　　北邙学院的人，都是天之骄子，除非是以实力降服，不然没有其他任何可能。
<br/>　　“你现在轻易击败了燕古，牧野战，出尽了风头，这个消息应该很快就会被北邙学院的高层注意到了，一会应该就会有长老出面，将你邀请晋升种子弟子了。”
<br/>　　白萱在一旁开口，突然话锋一转道。
<br/>　　“不过你打碎了玉皇镜，更是让燕古跪在地上，他背后的人，他的师尊，玉归长老，只怕是不会善罢甘休。”
<br/>　　“是吗？玉归长老，不过最多生死境高阶的实力，他要是敢来针对我，一样也要跪下。”
<br/>　　楚风眠冷冷的开口道。
<br/>　　在北邙学院，实力为尊，像是那些种子弟子，甚至是核心弟子，对这些北邙学院的长老，都是丝毫不会理会的。
<br/>　　像是夜天君，更是就连许多长老都要巴结，讨好，他想要杀一个长老，都不需要费多少时间手段。
<br/>　　实力为尊，是北邙学院唯一的规则。
<br/>　　“这话倒是不错，只要有着实力，长老算什么，一样可以杀了，我们晋升成为种子弟子的话，在地位上本来就是跟长老平起平坐了，也不怕玉归长老的报复。”
<br/>　　白萱开口道。
<br/>　　突然就在她的手心中，一枚玉符，突然闪现起来，打开玉符，看到其中的信息，白萱的脸上微微一笑道。
<br/>　　“果然，北邙学院的几位长老，要在功德殿中见我们。”
<br/>　　如今楚风眠击败燕古，牧野战，让他们二人跪下，镇压。
<br/>　　这么大的消息，当然很快便是传遍了北邙学院，北邙学院的高层，注意到，倒也是很正常。
<br/>　　“过去吧。”
<br/>　　楚风眠随意的开口道。
<br/>　　他这次耀武扬威，也正是为了晋升种子弟子，如今机会来了，他自然不会错过。
<br/>　　北邙学院的传承，楚风眠也想要见识见识。
<br/>　　两道遁光，直接从虚空中破碎离开，几息之间，便是来到了一座巨大无比的宫殿前。
<br/>　　这巨大无比的宫殿，便是功德殿，是北邙学院弟子晋升的地方。
<br/>　　楚风眠跟白萱，一步踏入其中，在其中，便是一眼三尊老者。
<br/>　　这三尊老者，每个人身上的气息，都是打破了临界点，实力强大无比，都是生死境高阶的人物。
<br/>　　在北邙学院中，真正的长老，几乎都是生死境高阶的人物，像是那些生死境的长老，其实只是一些外门长老，跟真正的长老，还有着一些距离。
<br/>　　这三尊长老，看到楚风眠跟白萱进来，都是纷纷走了出来。
<br/>　　“血长老，墨长老，南宫长老。”
<br/>　　白萱看着这三位长老，都是行了个礼。
<br/>　　“白萱，行礼就不必了，你马上要晋升成为种子弟子，地位已经跟我们平起平坐了，不必客气。”
<br/>　　其中一尊长老，那血长老笑了笑开口道。
<br/>　　“真没想到，这次要晋升的人，居然是你们二人。”
<br/>　　另一尊老者，墨长老也是开口道。
<br/>　　“本来我还以为，是那落长恨会第一个晋升成为种子弟子的，没想到他居然是死在了虚空海中。”
<br/>　　墨长老的话，也有些惋惜。
<br/>　　落长恨一直以来都是地榜的第一人，晋升种子弟子，几乎是板上钉钉的事情。
<br/>　　这样一尊天才，居然是陨落在外，也是一大损失。
<br/>　　要知道北邙学院的种子弟子，核心弟子，加起来也不到一百位，这还是无数年来积累下来的数量。
<br/>　　每年可以晋升一位，都是很少见了。
<br/>　　像是如今楚风眠跟白萱，二人一起晋升成为种子弟子，数百年来都没有遇到过了。
<br/>　　“落长恨那个废物，既然是死在了外面，死了就死了，我们北邙学院需要的不是废物。”
<br/>　　最后的南宫长老，冷冷的开口道，言语中杀伐无比，打量了一眼白萱跟楚风眠，脸上倒是露出了些笑容。
<br/>　　“你们二人不错，如今在外得到奇遇，已经是达到了如此实力，这种实力放在种子弟子中，已经不算末流了。”
<br/>　　“很好，我们北邙学院又多了两尊种子弟子，我们北邙学院，也变得更加强大了。”
