　　混沌中心某一处，一朵浩大的混沌青莲散发着无量造化之光，青莲之中，一个磅礴的大汉沉睡其中，他怀抱巨斧，仿佛亘古如此！
<br/>　　突然，在那浩大的混沌青莲之下，出现了一道神秘的身影，他看了一眼青莲保护之中的大汉，道了一句：“徒儿，你也该证道了！”
<br/>　　话毕，那道神秘身影又消失不见了，仿佛从来没有出现过。
<br/>　　突然，那浩大无边的青莲之上，一股浩瀚的气息陡然爆发，仿佛凌驾诸天之上，超越了种种存在的可怕力量，席卷整个混沌！
<br/>　　那道惊天的身影，赫然拿起霸道的斧头，脚下青莲璀璨无边，翻滚不止，仿佛屹立苍穹之巅！身上十二道不灭青光闪耀着，泛起阵阵古意！
<br/>　　“此次闭关，终有明悟，大道之境就在眼前！欲成就大道之境，是必要开辟混沌，化为天地。不过，我若证道，大道定然不允。毕竟混沌乃是大道意志笼罩之地，不会看着一个强大之人超脱它的掌控！
<br/>　　是故，在我证道之时，大道定然会降下劫数前来阻扰。不过，那又如何，大道在前，别说有些险阻，就算粉身碎骨，我也绝不后退半步！吾辈修士，何惧一死！生而不能一窥大道究竟，死亦何妨？！况且，我有混沌至宝混沌青莲与混沌开天斧，也不是没有机会。”
<br/>　　到了他这个境界，道心何等的坚固，生亦何欢？死亦何苦？
<br/>　　朝闻道，夕死可矣！
<br/>　　不证道，毋宁死！！！
<br/>　　……
<br/>　　苍茫无限，这一刻，整个混沌突然间震动起来，四面八方苍茫逆流而上，翻滚不休。
<br/>　　一股恐怖的杀机，劫煞的力量，遍布了整个混沌中的每一个角落！
<br/>　　“嘶！怎么回事？混沌劫来了吗？好恐怖啊？”
<br/>　　“啊！不！我才刚刚出世，混沌劫怎么会如此之快？”
<br/>　　“怎么可能？！”
<br/>　　……
<br/>　　苍茫界中，无尽修士，脸色大变，倒吸一口气，看向同一个方向，纷纷露出惊恐之色！
<br/>　　“这是？混沌劫？爆发了！”这一刻，哪怕是躲在老巢里修炼的时辰、魔祖、轮回等排名前五的混沌魔神也豁然起身，他们身体颤抖，脸色苍白，眼皮直跳地望着混沌中心！
<br/>　　陡然之间，在众人所望向的那个方向，一道霸道威武的声音，响彻整个混沌：
<br/>　　“吾为盘古氏，混沌第一魔神，当开辟混沌，衍化天地，以证大道，永恒不朽！我证道时，诸天寰宇，谁敢来阻，我便灭谁？大道若阻，又有何惧！！！”
<br/>　　这道声音带着无上的霸道，透露着沧桑与古老，响彻混沌，震动诸天，这是盘古的宣告，也是他的霸道！
<br/>　　混沌之中，伴着盘古这声宣告，滚滚的劫煞之气瞬间弥漫整个苍茫大地，充斥着整个混沌！
<br/>　　“原来如此，混沌劫？竟然是盘古开天证道，他若成功证道，那他开辟的天地将囊括整个混沌，到时候，没有混沌，吾等魔神又如何生存，他这是在坏吾等大道之基！嘶！”时辰、空间、造化等魔神，脸色震惊，眼皮直跳。
<br/>　　混沌深处，一股浩瀚无垠的波动在此刻陡然间涌现，三千道璀璨的光芒相互交织在一起，带着大道法则的玄妙光辉，笼罩无尽混沌。它们是三千大道，混沌的本源，大道的根本！
<br/>　　此刻，原本沉寂万古，隐于苍茫深处的大道本源，泛起了一丝涟漪，似乎冥冥之中，感受到一丝丝来自盘古的威胁。一股伟大的意志，在沉睡中苏醒！
<br/>　　“盘古欲毁灭混沌，再造乾坤，三千魔神，一切种种，共诛之！”一股浩瀚的意志，传入每一个混沌魔神元神空间。同时，一个强大的劫煞之气，酝酿在整个苍茫！
<br/>　　与此同时，一些修为不够的混沌魔神，瞬间就被这些劫煞之气所感染，他们的眼神，瞬间就变得通红无比，杀气也变得愈发浓郁，他们都纷纷朝着盘古所在的地方赶去。
<br/>　　就是有些第三步混元强者也随着时间的延长，也纷纷被劫煞所感染，不由自主的朝着混沌中心赶去。
<br/>　　只有混元七重天以上的魔神，能暂时够保持清明。不过，随着大劫的降临，他们迟早也会变得如同其他人一样，他们也感受到了，冥冥之中，有一股意志在驱使他们前去阻止盘古开天，若是无法阻止，他们就有身陨的危机。当即，犹豫了片刻，就纷纷赶往混沌中心。
<br/>　　混沌魔神乃大道所孕育，大道有令，谁能违背？
<br/>　　“混沌劫到了吗？也罢，吾为因果之主，当前往阻之！”混沌悟道古茶树下，元释听到那霸道的宣言，又收到来自大道的信息，知道了开天之劫到了。
<br/>　　他缓缓睁开双眼，眼中带着强烈的光芒，同时，身上泛起十一道不灭青光，一步踏出，那伟岸的魔神真身瞬间恢复到数百万丈之高。
<br/>　　在这混沌劫来临之际，他的修为显然是再次突破，到了混元大罗金仙十一重天，魔神真身更是突破到了混元十二重天。若是再加上他的种种手段，已经可以力敌混元十二重天的修士了。
<br/>　　巨手一抬，绽放着黑白双色的因果莲台瞬间飞到他的手心，莲台之上除了黑白双色之外，还掺杂了一些混沌色，显然是因果莲台经过元释众多神材的加入以及混沌之气的吸收，已经更进一步了，突破了原先的先天至宝极限，到了混沌灵宝的级别，当然，这只是一般的混沌灵宝，比不上四大混沌至宝！
<br/>　　就算如此，那样已经不是先天至宝所能比拟的了。而且，其内的混沌空间经过此次晋级再次扩大，混沌之气更加浓郁了。
<br/>　　只见元释大手一挥，混沌悟道古茶树顿时脱离混沌，化作一个圆点遁入因果莲台的混沌空间内。收了灵根，元释连它根下的泥土也不放过，统统搬了进去，还有悟道古茶树旁的一汪水流，看这水流蕴育九彩之色，灵气四溢，恐怕不比那什么后世的三光神水差吧？毕竟这也是混沌出品了，混沌出品，必属精品！
<br/>　　“对了，这玉石也不能放过，用来炼制宝贝什么的，这可是好材料啊！就算留在此处也会在盘古开天时被破灭，浪费是可耻的！”元释原本就准备离开了，可是发现古树根部深处还有一些玉石，连忙收入元始印之中，再看了看，并没有发现其他东西，元释才放心离开，朝着混沌中心走去。
<br/>　　一路上，元释也不快不慢，毕竟盘古开天也不是短时间就能完成的，先让一些修为差的魔神前去阻拦，他在后面收拾那些魔神的真身，毕竟，这些魔神真身之中可是有这本命大道法则，而他的元始印，需要的就是这些法则。只不过，这些魔神对于盘古来说，太弱了，大部分都直接化为飞灰，不留丝毫，只有少数修为强大的魔神，死后还能保留尸体。
<br/>　　元释的道念笼罩混沌，滚滚气势内敛，胸中战意升腾，大步迈向苍茫中心！
