    中午时分，春末的阳光照拂着红色的大地。虽然气温算不上炎热，但还是有些晒。一辆吉普车驰行着，车后卷起大片的泥土和草叶。<br/>    　　<br/>    　　“不是秦，是陈，Mr陈。”<br/>    　　<br/>    　　卡西开着车，陈兴双脚架在车门上，外躺着，正在纠正卡西的发音。<br/>    　　<br/>    　　“秦，Mr……”卡西吃力地咬着音节，“秦！”<br/>    　　<br/>    　　“是陈，陈。”陈兴耐心地教着。<br/>    　　<br/>    　　“秦。”卡西依然错误。<br/>    　　<br/>    　　“陈。”“秦。”“陈！”“秦！”“陈！”几个回合后，陈兴越教越暴躁，这小黑人的舌头怎么像直的一样。<br/>    　　<br/>    　　见陈兴快要发火了，卡西全神贯注，憋了半天，终于挤出一个字来，“秦！”<br/>    　　<br/>    　　陈兴彻底抓狂了，拔出手枪，顶着卡西的脑袋，沉声道，“陈！”<br/>    　　<br/>    　　卡西冒出了冷汗，用力地搓了搓脸，吐了吐舌头，深深地吸了口气，做足准备，缓缓发音：<br/>    　　<br/>    　　“秦……”<br/>    　　<br/>    　　“是秦，不是秦！”陈兴吼道，情急之下，发生了口误。<br/>    　　<br/>    　　“秦！”卡西兴奋地叫了起来，“我就说是秦嘛，哈哈哈，Mr秦，我发音准不准啊？”<br/>    　　<br/>    　　陈兴用力地拍了下自己的脑门，自言自语地说道，“我到底在干什么……”随后收起手枪，彻底放弃了纠正对方。这家伙缺了根筋，捋不直。<br/>    　　<br/>    　　下午四点过十分，陈兴两人开着吉普车，绕了一个大圈，追上了商队。<br/>    　　<br/>    　　尽管他们很看不起胖子头领，但他们是流民身份，进城需要商队提供的临时身份证明。<br/>    　　<br/>    　　整个商队死气沉沉的，所有人都无精打采，似乎还没有从刚才的打击中恢复过来。看到陈兴两人归队，胖子头领叹了口气，欲言又止，可最终什么也没说。<br/>    　　<br/>    　　“阿乔木被他们抓了，快想想办法，救救她吧！”白人佣兵跳下车，抓住陈兴的手臂，哀求道。<br/>    　　<br/>    　　陈兴楞了一下，这才记起对方说的是那个女佣兵。他们认识还不到一天，印象不是很深。且不说黒鹫团人多势众，萍水相逢的，怎么可能为对方涉险，正想开口拒绝，黑人小伙却先骂起来了：“兄弟，你是白痴吗？对面一百多杆枪，我们才三个人，别说救人了，就是给他们塞牙缝都不够。”<br/>    　　<br/>    　　白人佣兵听了，耷拉着脑袋，似乎也知道可能性不大，只是寄望于奇迹出现。<br/>    　　<br/>    　　卡西拍了拍白人佣兵的肩膀，说道，“兄弟，那帮人肥的流油，连武装蟋蟀都有，就算不救人，我也很想去打劫他们，只不过……”他耸了耸肩，“你知道的……”<br/>    　　<br/>    　　接着，他又叹了口气，有些同情地说道，“可怜的阿乔木，今晚肯定要遭罪了。”<br/>    　　<br/>    　　“他，他们真的会那么干吗？”白人佣兵脸色煞白，抓着卡西的双肩，就像在抓一根救命稻草。<br/>    　　<br/>    　　“忘了她吧……”卡西劝道，“一只落入狼穴的羔羊，等待它的只有狂欢和盛宴……”他习惯性地耸了耸肩，露出一副“你懂的”的表情。<br/>    　　<br/>    　　白人佣兵一下子瘫坐在地上，瞪大着眼睛，目光呆滞。<br/>    　　<br/>    　　同样的话，听在陈兴耳朵里，结果却大不相同。脑海中闪过一道灵光，但又转瞬即逝，有些抓不住重点。<br/>    　　<br/>    　　“你刚才说什么？”陈兴皱着眉头，略有所思地朝卡西问道。<br/>    　　<br/>    　　“啊？”卡西明显地楞了一下，没弄明白他的意思。<br/>    　　<br/>    　　“我是说，你刚才说了些什么，请再说一遍。”陈兴说道。<br/>    　　<br/>    　　“关于阿乔木的？”卡西不太确定地问道。<br/>    　　<br/>    　　“还有呢，更往上的。”陈兴继续说道。<br/>    　　<br/>    　　“我想去打劫他们？”卡西问道。<br/>    　　<br/>    　　“不是。”陈兴摇了摇头。<br/>    　　<br/>    　　“他们很富有？”卡西试探性地问道。<br/>    　　<br/>    　　“啪！”陈兴用力地锤击掌心，“对，就是这个！”<br/>    　　<br/>    　　两个因素加在一起，让事情产生了微妙的变化。<br/>    　　<br/>    　　第一，他们很富有；第二，他们今晚将会进行一场狂欢。<br/>    　　<br/>    　　富得流油且防御松懈，简直是天赐良机。<br/>    　　<br/>    　　陈兴目光一凝，说道，“我觉得，我们有责任营救自己的同伴！”他义正言辞，口气说不出的严肃认真。<br/>    　　<br/>    　　“什么？”卡西一脸错愕，像是听错了。这个狡猾的黄猴子，什么时候变得这么高尚了？白人佣兵则面露狂喜，一个劲地朝陈兴竖起大拇指，“你是英雄！”<br/>    　　<br/>    　　“你过来一下……”陈兴把卡西拉到一边，说明了其中的利益。<br/>    　　<br/>    　　“你简直就是个疯子！”卡西板着脸说道，然后眉头一展，双手比划着手枪，说道，“不过我喜欢。”<br/>    　　<br/>    　　随后，卡西握着拳头，认真地对白人佣兵说道，“阿乔木是个好姑娘，我们不能眼睁睁地看着她遭受折磨，我们必须救她！”<br/>    　　<br/>    　　就这样，三人一拍即合，开着胖子头领的吉普车往回走。胖子头领理亏在先，也不好意思阻止，还提供了一些补给。<br/>    　　<br/>    　　三人很快就到了之前的营地，由于时间过去的不久，黒鹫团的车痕还很明显。三人开着车，尾随痕迹，一路追踪。<br/>    　　<br/>    　　“快看，那是什么？”<br/>    　　<br/>    　　十多分钟后，眼尖的卡西发现车痕旁边有些东西，于是停了下来。<br/>    　　<br/>    　　走近一看，是块灰色的耐磨布，边缘很整齐，像是被利器割开的。卡西闻了闻，表情有些奇怪地说道，“是裤子……”<br/>    　　<br/>    　　再往前，他们又发现了一团白色的棉布。具体是什么，明眼人一看就知道。但大伙都心照不宣，没有说穿。过了一会儿，他们又发现了一件被刀子割开的衬衣，以及一个皮带状的罩子。很明显，这些东西都属于同一个主人。<br/>    　　<br/>    　　白人佣兵的脸色有些苍白。不难想象出，这里曾经发生了什么。<br/>    　　<br/>    　　“这帮混蛋！”卡西忍不住骂道，用力地捶了下方向盘。<br/>    　　<br/>    　　陈兴摇了摇头，却也没说什么。对方实在太猴急了，离开商队才十多分钟就开始了。<br/>    　　<br/>    　　接下来，三人都沉默着，继续追踪，终于在傍晚时刻，找到了黒鹫团的营地。<br/>    　　<br/>    　　营地建在一个矮山下的天然溶洞，外面停满了沙地车，其中红色的武装蟋蟀尤为显眼。<br/>    　　<br/>    　　望远镜中，溶洞的洞口拉着厚布帘子，看不见里面的状况，但有橙红的火光从缝隙中透出。外围只有七八个人在巡逻，都有些心不在焉的，不时望向溶洞的方向。那样子，既是好奇，又是期待。<br/>    　　<br/>    　　“老大，怎么办？”卡西朝端着望远镜的陈兴问道。不知不觉中，他已经将陈兴当成头领了。<br/>    　　<br/>    　　“等天黑。”陈兴放下望远镜说道，这里距离溶洞还有四五公里，直接过去肯定会被发现。<br/>    　　<br/>    　　“阿乔木她……”白人佣兵欲言又止。<br/>    　　<br/>    　　“兄弟，我理解你的心情……”陈兴拍了拍对方的肩膀，说道，“但我们现在过去，唯一的结果就是赔上自己。”<br/>    　　<br/>    　　白人佣兵叹了口气，望向远处溶洞的目光有些落寞，但也无可奈何。<br/>    　　<br/>    　　等待之中，太阳逐渐下山，月亮和星星缓缓升起。卡西取出一个小怀表，看了眼，报出了现在的时间——晚上七点零五分。<br/>    　　<br/>    　　“再等。”陈兴面无表情地说道，然后吃起了压缩饼干。<br/>    　　<br/>    　　一个小时后，白人佣兵忍不住了，但陈兴还是一句话，“等。”<br/>    　　<br/>    　　晚上十点，白人佣兵越来越急躁，忍不住催促道，“Mr陈，我觉得我们应该动手了。”<br/>    　　<br/>    　　“让他们再折腾一会儿吧。”终于，陈兴说出了真实的想法。<br/>    　　<br/>    　　白人佣兵一下就明白过来，顿时面如死灰，沉默了片刻，带着几分怨恨说道，“Mr陈，你真是个残酷的人。”<br/>    　　<br/>    　　陈兴的目的很简单，就是让对方在狂欢中耗尽体力，再一网打尽。至于那个女人的死活，他是没有算计在内的。上一世的经验告诉他，有些时候，盲目
