　　冰雪城，便是炎武王国北边的最后一个大城了。
<br/>　　它并非冰雪郡郡都。
<br/>　　仅仅是因为它在冰雪圣山之旁。
<br/>　　此时，萧逸与白冰雪二人，已在冰雪圣山之下。
<br/>　　“好冷。”白冰雪皱起了眉头。
<br/>　　两人站于冰雪圣山之下，明显感觉到一股远超之前十倍以上的冷意。
<br/>　　萧逸看了眼这座巍峨雪山，倒吸一口凉气。
<br/>　　早在之前于数座大城之外御空飞行时，已远远能看到这座雪山。
<br/>　　那时，哪怕隔着极远的距离，在眼中，雪山仍旧巨大无比。
<br/>　　现在到了面前，雪山之巨大，完全超乎了想象。
<br/>　　那一眼望不到顶，直插云霄的山巅。
<br/>　　那两头看不尽的山体，仿佛一睹白色巨墙，遮挡着两个世界。
<br/>　　据猎妖殿买来的资料记载，冰雪圣山，便是炎武王国最北边的国界。
<br/>　　其横跨于北山郡最边缘的位置，自然巨大无比。
<br/>　　“上去吧。”萧逸淡淡地说了一声，率先动身。
<br/>　　白冰雪连忙跟上。
<br/>　　两人不断飞跃，越是往上走，其冰冷，越发恐怖。
<br/>　　甚至于还未到山腰位置，已然风雪环绕，刺骨寒风扑面而来。
<br/>　　不多时。
<br/>　　当两人来到山腰位置，这里头的冷意，冻入骨髓。
<br/>　　若无地元境以上修为，绝对无法安然站立。
<br/>　　单单剧烈罡风，就能将人吹得寸步难行。
<br/>　　再继续前进。
<br/>　　当来到山腰再高一些的位置时。
<br/>　　白冰雪已经皱起了眉头。
<br/>　　“受得住吗？”萧逸淡淡问道。
<br/>　　“问题不大。”白冰雪清冷道，“估计再前行不远，就要开始调动些许元力护身了。”
<br/>　　“以我的修为，实力全开的话，到达山巅不成问题。”
<br/>　　冰雪圣教，位于冰雪圣山的山巅。
<br/>　　萧逸点了点头，继续前行。
<br/>　　根据猎妖殿的资料显示。
<br/>　　冰雪圣山，位于极寒之地的边缘。
<br/>　　恰好与炎武王国接壤，并被炎武王国纳入境内。
<br/>　　按理说，边缘位置，应该不至于太过冰冷才对。
<br/>　　但，不知为何，这座巍峨雪山的冰冷程度，比真正踏入极寒之地更为严重。
<br/>　　有传言说，冰雪圣山之下，正是雪脉之地。
<br/>　　但是否真如此，不得而知。
<br/>　　起码这边的猎妖殿，并无详细资料，也并不能查清楚。
<br/>　　言归正传。
<br/>　　两人疾驰着。
<br/>　　恰在此时，两道身影忽然从风雪中现身。
<br/>　　“来者何人？胆敢擅闯冰雪圣教？”两个身穿白色毛裘的男子冷喝一声。
<br/>　　“我们是裂天剑宗弟子。”白冰雪清冷说道。
<br/>　　“裂天剑宗弟子？”两个男子皱了皱眉头，道，“可有凭证？”
<br/>　　萧逸取出了北山剑主令。
<br/>　　两个男子看了一眼，道，“剑主令。”
<br/>　　随后，两人对视一眼，一人转身离去。
<br/>　　一人则对萧逸和白冰雪拱了拱手，道，“原来是剑宗弟子。”
<br/>　　“两位暂且稍等，我等得向山门执事通报一声。”
<br/>　　萧逸点了点头。
<br/>　　不多时。
<br/>　　一位白衣中年男子凭空而现。
<br/>　　“烦请阁下再出示一遍剑主令。”那中年男子说道。
<br/>　　萧逸再出示了一次。
<br/>　　中年男子点点头，道，“果真是剑主令。”
<br/>　　“两位，应该是要前往极寒之地参加冰缘大会的吧。”
<br/>　　“正是。”萧逸点了点头。
<br/>　　中年男子笑道，“不愧是代表剑宗参加冰缘大会的弟子。”
<br/>　　“竟敢在没有宗门长辈带领的前提下，独自前往。”
<br/>　　“必是宗内天骄吧。”
<br/>　　“来吧，随我上山巅。”
<br/>　　中年男子说着，率先领路。
<br/>　　奇怪的是，中年男子领路下，周边风雪，竟是自动散开数米之外。
<br/>　　暴烈罡风和风雪，再无法侵袭萧逸两人。
<br/>　　在萧逸的感知中，这位中年男子，也就是山门执事，修为只在地元五重。
<br/>　　若无猜错的话，必是冰雪圣教中人，掌握着一种极其高超的操控风雪之法。
<br/>　　其实也理所当然。
<br/>　　毕竟冰雪圣教坐落于冰雪圣山数百年了。
<br/>　　自然有教内强者，创出此等手段，以方便教内弟子、执事，出入圣山内外。
<br/>　　不多时，山门执事停下了领路的步伐。
<br/>　　萧逸皱了皱眉头，这里，大概只在雪山的三分之二高度。
<br/>　　距离雪山之巅，还有一段不小的高度。
<br/>　　山门执事笑道，“两人前来，是要请我冰雪圣教中的雪翼雕一族武者，助你们飞行赶路吧？”
<br/>　　“是。”萧逸点了点头。
<br/>　　之所以必须要来冰雪圣教。
<br/>　　是因为踏入极寒之地数千里后，极寒之地的环境将会剧变。
<br/>　　数千里范围内，没什么影响。
<br/>　　而数千里范围后，将是冰封万里，风雪如漫天鹅毛。
<br/>　　且，将会出现极寒妖兽，甚难对付。
<br/>　　而最重要的是，数千里范围后，高空将出现极寒罡风。
<br/>　　若无天元后期以上境界修为，将无法御空飞行。
<br/>　　等闲天元境武者，若强行御空，将在那极寒罡风下撑不过片刻便会冻成冰雕，随后化为冰屑齑粉。
<br/>　　极其凶险。
<br/>　　猎妖殿资料上，对极寒之地的概述，只有两句话。
<br/>　　地面，是弱小武者的坟场。
<br/>　　高空，是强大武者的禁地。
<br/>　　弱小的武者，无法御空，将面对极寒之地数之不尽的极寒妖兽。
<br/>　　简单来说，极寒之地，事实上像极了一座妖兽森林。
<br/>　　只不过这座妖兽森林充斥着冰天雪地，且更加凶险。
<br/>　　而强大的武者，只要还未踏入天元境后期，就无法御空，那里是一片禁地。
<br/>　　当然了，天元境武者，若在地面行走，还是有一定安全性的。
<br/>　　但哪怕是天元境后期武者，能够御空，仍旧未是百分百安全。
<br/>　　越是深入极寒之地，出现的极寒罡风，就会越发强大，甚至隐藏着未知的危险。
<br/>　　若想真正在极寒之地安然横行，或许得要有巅峰强者的实力，乃至更强。
<br/>　　不过，这其中有一个异数。
<br/>　　便是雪翼雕一族。
<br/>　　雪翼雕，向来是天空的霸主。
<br/>　　雪翼雕武魂拥有者，则是御空的王者。
<br/>　　哪怕只有地元境，也足以在极寒之地的高空横行无阻，来去自如。
<br/>　　这也是萧逸特地来冰雪圣教一趟的原因。
<br/>　　当然了，本身雪翼雕一族也有做这门生意。
<br/>　　炎武王国内的武者，一般若无足够实力和把握的话，都会前来找雪翼雕一族帮忙。
<br/>　　只是需要付出报酬就是了。
<br/>　　而萧逸本身，其实有着天元境后期，乃至巅峰强者的实力。
<br/>　　但实力归实力，境界未足，只有地元九重修为，并无相应的手段。
<br/>　　若强行在极寒罡风中御空飞行，不是不行，但其代价就是快速地消耗体内元力。
<br/>　　在极寒之地这等危险地方，肆意消耗元力，可不是明智之举。
