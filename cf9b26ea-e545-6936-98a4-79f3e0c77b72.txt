　　此时的任家大院里，虽然已经深夜，但主厅二楼依旧亮着点点烛光。
<br/>　　书房中，秃顶老人恭敬的站在任发身边，将一封书信交了上去，任发带上眼睛，查看了一眼，紧锁的眉头开始舒展开来。
<br/>　　“大人那边，还有什么口信没有？”将书信丢在一边，任发看着眼前的秃顶老人，语气严肃的开口追问道。
<br/>　　“鹰大人已经开始动身了，估计明天下午就能赶来，他说会亲手处理这件事情。”微躬着身子，秃顶老人颇为恭敬的回答。
<br/>　　“恩，那这件事情就没什么问题了，只要大人肯出手，光凭义庄那个家伙，掀不起什么风浪”舒了口气，任发彻底将这件事情放在了脑后，挥手让秃顶老人退了出去。
<br/>　　坐在红木打造成的桌子面前，任发犹豫了片刻，随后拿起桌上纸笔，洋洋洒洒的写了起来。
<br/>　　所有的布署差不多都已经完成，等明天下午鹰大人赶来，就立即可以动手了。
<br/>　　趁着现在还有时间，他还要写几封书信，通知其余人做一些准备，以免到时候出现了什么纰漏。
<br/>　　况且，刘哔的突然出现，也给他带来了一个很好的契机，借着这次机会，他也想做一些事情。
<br/>　　别看现在任家家大业大，但任发知道，家里百分之八十的资产都不属于他门家族中的任何一人，而他们的身份地位，实际上和仆人没多大区别。
<br/>　　但幕后那人实力夸张的厉害，几十年来，他每时每刻都想将其灭杀，如今终于有了一丝希望，他怎么可能不付诸行动！
<br/>　　夜已深，一阵冷风从窗外吹来，掀起了任发淡金色的衣袍，被风一吹，任发打了一个哆嗦，暂停下了手中的动作。
<br/>　　“要入秋了啊，还真是怪冷的，看来明天得多加一些衣服了。”嘴里嘟囔一声，任发挪动有些肥胖的身体，来到了窗户旁边。
<br/>　　就在窗户即将关闭的时候，任发的动作猛然顿住，眼中带着震惊，看着窗外缓缓浮现的一颗苍白人头。
<br/>　　“小家伙，好久不见。”突然，人头咧嘴笑了一声，森然的冲任发开口道。
<br/>　　蠕动了几下嘴唇，任发脸上挤出一抹比哭还要难看的笑容，道：“鹰大人，您来的这么快，也不提前知会一声，我好……”
<br/>　　不等任发将话说完，一柄闪烁着寒光的长剑就从他口中贯穿了过去，瞬间，一股剧烈的疼痛感传满全身，任发瞪着一双不可思议的眼睛，眼中都是浓浓的不甘。
<br/>　　“哼，敢背叛我，死有余辜。”苍白人头下面，一个身穿灰袍的老道站了起来，反手抽出了插在任发口腔里的长剑，向人家门外走去。
<br/>　　整个任家，此刻都充斥着刺鼻的血腥味道，鹅卵石铺成的道路上，四处可见散落在地面上的残尸断骸。
<br/>　　……
<br/>　　“叮~限时任务已经完成，后续任务开启，请宿主及时关注。”
<br/>　　系统清脆的提示音从脑海中响起，刘哔目光一闪，放下了手中样式古朴的深红色戒指。
<br/>　　这是他不久前刚从系统商城内兑换出来的储物戒指，虽然里面的空间只有十立方米，但还是足足花费了他二百积分。
<br/>　　空间装备的外貌，颜色都是可以根据宿主喜好自行定制的，开启它们，也需要微量的能量。
<br/>　　除此之外，刘哔还花费了三百积分，为自己兑换了一次低级技能强化卷轴，将他的‘吞噬’技能给强化了一次。
<br/>　　之所以兑换这些东西，也是他经过深思熟虑的，毕竟手头上的积分不多，这是性价比最高的兑换了。
<br/>　　其实想要短时间飙升战力的话，他大可兑换一些科技武器，那些东西需要的积分不多，但威力也是非常强劲的。
<br/>　　可根据系统之前给出的提示所说，这些科技武器，绝大多数情况下，对于鬼怪都没有杀伤效果。
<br/>　　免疫纯物理攻击，这是鬼魂最基础的特质。
<br/>　　综合所有的因素选择了一下，刘哔最终决定兑换这两样东西。
<br/>　　空间装备就不需要说明了，只是一个用来随身储存东西的物品而已，兑换这个，也是刘哔考虑到接下来的日子要四处奔波，并且距离回归不远，他也想从这个世界带一些物品回去。
<br/>　　说起来，这个戒指的性价比倒是最高的，有了它刘哔能方便不少，况且，等以后积分富裕了，有需要的话还可以扩充戒指里面的空间。
<br/>　　况且系统出品的东西都有‘专属’效果，也就是不可掉落，不可交易，也绝对不会丢掉，这枚戒指，他能戴很久很久。
<br/>　　至于低级技能卷轴，也是低级物品里面为数不多性价比超高的物品了，它可以给宿主任意一项不超过D级技能直接提升一个等级，让技能有更为强大的效果。
<br/>　　和其他的那些装备，物品不同，一项好的技能，同样也能让刘哔一直使用，故此，虽然花费的积分稍微贵了一些，但他还是咬牙买了下来。
<br/>　　考虑到即将要回归主世界，现如今他所拥有的技能，提升‘吞噬’或者‘再生’是最为划算的，考虑到技能的实用性，刘哔最终还是将‘吞噬’提升到了E级。
<br/>　　而提升之后的‘吞噬’技能，也拥有了更快速的吸收效率，并且限制也少了许多。
<br/>　　至少，不需要非得将目标给吃掉了，只要是他触摸到的任何能量体，只要花费一定的时间，他都可以将其吞噬。
<br/>　　唯一有些遗憾的是，吞噬的收获会根据目标死去的时间逐渐衰弱，大概一个小时之后，就彻底失去了吞噬价值了。
<br/>　　不过，即使是这样，他也已经十分心满意足了，和升级之前的‘吞噬’技能相比，两者简直有云泥一般的差别。
<br/>　　将戒指重新戴回到手上，刘哔打开系统任务面板，扫了一眼后续任务，陡然，瞳目猛地一缩。
<br/>　　名称：秘密（连环任务）
<br/>　　轮数：二。
<br/>　　内容：击杀来袭的鬼修‘鹰’。
<br/>　　等级：未知。
<br/>　　时限：五个小时。
<br/>　　失败惩罚：永久封闭后续任务。
