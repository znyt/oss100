　　“这是北门将军的绝学，赤焰魔瞳！”
<br/>　　“这小子是谁，竟然能让北门将军动用绝招！”
<br/>　　“你们还不知道吧，据说这家伙在城门口，当众斩杀了北山铁男公子！”
<br/>　　“原来是这样，难怪北门将军如此愤怒，这小子还真是不知死活啊，竟然还敢进城！”
<br/>　　……
<br/>　　在众人的议论声中，那两条火龙，已经将叶尘淹没其中。
<br/>　　而叶尘，似乎来不及抵抗，瞬间就已经被那两条火龙所吞噬。
<br/>　　刹那间，连同周围的温度，也急剧攀升，甚至远在百米之外的路上，都觉得浑身发烫，仿佛一下子置身于大熔炉之中，不由得再次连连倒退，同时暗暗惊叹，
<br/>　　“北门将军的赤焰魔瞳，果然厉害！”
<br/>　　“看来那小子，毫无任何抵抗之力啊！”
<br/>　　“恐怕已经被烧成灰了吧！”
<br/>　　……
<br/>　　就连十七公主，俏脸上也露出一抹喜色，
<br/>　　“看来根本不需要父亲大人出马，北山将军就足以将你……什么！”
<br/>　　十七公主自言自语的话，还没有说完，突然看到了让她无比惊恐的一幕。
<br/>　　只见，
<br/>　　待那炙热无比的火焰消散之后，之前被火龙吞噬的那道身影，竟然丝毫无损的站在那里，而且依旧一脸悠然之色。
<br/>　　“这怎么可能！”
<br/>　　就连那位北山将军，也不由得脸色大变。
<br/>　　他的赤焰魔瞳，修炼到极致，可以焚烧万物！
<br/>　　虽然他现在才只是入门，可是在御空境之中，绝对是最顶级的绝技。
<br/>　　便是他们罗刹城的城主，也不敢硬接。
<br/>　　可是现在，
<br/>　　居然未能伤到对方分毫！
<br/>　　叶尘冷冷一笑，
<br/>　　“就凭你这种垃圾火焰，也想伤我？真是可笑！”
<br/>　　说完这话，
<br/>　　叶尘手掌一翻，
<br/>　　嗖！
<br/>　　一道三昧真火，登时便出现在他的掌心之中。
<br/>　　“这是……”
<br/>　　北山将军顿时瞳孔一缩，
<br/>　　虽然对方的这道火焰，极为微弱，可是却给他一种难以形容的威胁感。
<br/>　　而那戴安娜，见识过三昧真火的威力，顿时吓得娇躯微微一颤。
<br/>　　“今天就让你见识一下，什么才是真正的火焰！”
<br/>　　说完这话，叶尘举掌一拍！
<br/>　　浑厚无比的掌力，夹杂着三昧真火，铺天盖地朝北山将军呼啸而去。
<br/>　　“这是什么火？怎么会这么强！”
<br/>　　北山将军这才发现，眼前这个年轻人，实力竟然如此强大，吓得他慌忙在坐下的灵兽身上一按，然后迅速暴退。
<br/>　　“彭！”
<br/>　　“嗷！！”
<br/>　　那道火焰巨掌，瞬间便将那头狼形灵兽吞没。
<br/>　　惨叫声戛然而止，偌大的一头灵兽，瞬间化成了一缕青烟，彻底消失在世间。
<br/>　　而那北门将军，虽然侥幸躲过一劫，却也狼狈无比，
<br/>　　就在他正打算转身逃跑之时。
<br/>　　不料，
<br/>　　一道冷漠的声音陡然在他耳边响起，
<br/>　　“想跑？你跑的了吗？”
<br/>　　轰！
<br/>　　一道紫金色的惊天剑芒，在空中划过一道优美的弧线，所过之处，仿佛连空间都被撕裂开来。
<br/>　　“什么！此人难道是冲霄境？”
<br/>　　北门将军望着空中，那恐怖的一剑，顿时更加骇然。
<br/>　　这一剑的威力，莫说是他，便是他们罗刹城城主，恐怕也未必能接得住！
<br/>　　北门将军拼命的想要躲闪，可是整个人，却仿佛被锁定了一般，任凭他如何挪移，那道惊天剑芒，最终还是精准无比的落在了他的头顶之上。
<br/>　　“不！！”
<br/>　　伴随着一道凄厉的惨叫声，在众人无比惊恐的目光中，这位罗刹城四大将之首，众人心中的战神，竟然被那道紫色剑芒，一剑劈成了两半！
<br/>　　“这……”
<br/>　　所有人看到这一幕，顿时吓得纷纷倒退，简直不敢相信，眼前所发生的一切。
<br/>　　罗刹城四大将之首的北山赤目，竟然被一名穿着普通侍卫的年轻人，一剑劈成了两半？
<br/>　　要知道，这北山赤目，可是御空境后期的实力，在整个罗刹城，也足以排进前五，便是他们的城主大人，也不可能如此轻松战胜他啊！
<br/>　　那十七公主早已经吓得脸色惨白，用玉手捂着小嘴，不敢再发出一丁点的声音，生怕引起这位恐怖煞星的注意。
<br/>　　而就在这时，
<br/>　　又有一道充满着威严的声音，陡然响起，犹如春雷滚滚，自城主府之中传来，
<br/>　　“什么人胆敢在我城主府门前放肆？”
<br/>　　声音落下，一名身穿格外高大，足足近三米的中年男子，大踏步从城主府中走了出来，身上散发着一股难以形容的霸气！
<br/>　　“是城主大人来了！”
<br/>　　“拜见城主大人！”
<br/>　　周围众人看到这个高大霸气的中年男子，顿时纷纷跪倒在地，高呼行礼起来。
<br/>　　原来，这人便是罗刹城的主人，纳兰百川。
<br/>　　“父亲大人！”
<br/>　　早已经快要吓晕过去的十七公主，也当即快步冲到那霸气中年男子跟前。
<br/>　　“十七，怎么如此慌张？发生什么事了？”
<br/>　　霸气中年男子，低头拍了拍十七公主的脑袋，脸上露出一抹宠溺之色。
<br/>　　“父亲大人！就是这个人，他杀了北山铁男，杀了东门统领，现在又杀了北山将军！”
<br/>　　十七公主指着叶尘，一脸惊恐道。
<br/>　　“什么？你是说，他杀了北山赤目？这怎么可能！”
<br/>　　罗刹城主顺着女儿指的方向，望了叶尘一眼，明显一脸的不信。

<br/>　　顺着女儿指的方向，罗刹城主这才看到，被叶尘一剑劈成两半的北山赤目，顿时目瞪口呆。

<br/>　　倘若北门赤目，真的是被眼前这个少年所杀，那此人的实力，极有可能还在他之上，轻易不可招惹！
<br/>　　叶尘一脸平淡，根本没有理会纳兰百川的问题，而是直接开门见山，道：
<br/>　　“你就是罗刹城主吧？三个月前，你们在城外的寒灵林，得到了一块万年玄冰，现在告诉我它的下落，否则，我屠你满城！”
