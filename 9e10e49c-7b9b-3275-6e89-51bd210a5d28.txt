　　羽毛如一把利刃，高速旋转着，在人群中穿梭。
<br/>　　这些生灵的实力，根本就不足以抵挡，基本上就是一群待宰的羔羊，羽毛从他们身边飞过，只听见一片惨叫声，邪物的肉身四分五裂，连魂魄也被切碎了……
<br/>　　“鲲鹏！！”
<br/>　　叶少阳大吼一声，眼睛里仿佛渗出火来。
<br/>　　杀了这些人之后，鲲鹏对玉玑子使了个眼色，一起跳下莲叶，一头扎进了无色海中，不见了踪影。
<br/>　　帅旗直直地飞下来，插在了影魅面前的地上，旗面展开，黑气渗出，在空中聚集成了一道巨大的人像：头戴纺锤管帽，身穿大红蟒服（严格说是状元服，钟馗当年在人间是状元，这是一段不堪回首的陈年往事，将来见番外篇），正是鼎鼎有名的钟馗大士。
<br/>　　九头之鬼，鬼中之魁。
<br/>　　哗啦……
<br/>　　一阵水花翻动，将叶少阳一干人全推到水里去，水流湍急，碧清用莲叶将几人纷纷抓住，根系扎在河下的土壤中，将自己固定住，不到一时半刻，水已经退了，剑阵也不在了。
<br/>　　碧清将他们松开，叶少阳睁眼看去，水已经退了，自己一干人站在一条被水冲刷出来的巨大的壕沟里，但是脚下和四周都是干的——身上也是干的，被没有被水浸湿分毫。
<br/>　　难道那真的是传说中的无色海？
<br/>　　“卧槽，好险好险，还是本王我福大命大，你们都沾了我的光……”小马从地上爬起来，抖着一身肥肉。吴嘉伟也爬了起来，看上去没什么异样。
<br/>　　碧清却躺在壕沟底部，身上盖着莲叶，看上去极度虚弱的样子，边上一朵莲叶打开，里面躺着昏迷不醒的清波。
<br/>　　“你怎么了？”叶少阳上前抱起碧清，碧清用力吐出一口气，弱弱地说道：“我没事，只是耗费了太多修为，虚脱了。”
<br/>　　之前在海水中，是她现出真身，拼了命地保护众人，叶少阳看在眼里，心里也是充满了感激，对她笑了笑，道：“怎么报答你呢。”
<br/>　　“以身相许吧。”
<br/>　　叶少阳尴尬了。
<br/>　　碧清笑道，“如果不能以身相许，那回去之后给我买一台最大的电视机。”
<br/>　　叶少阳有点无语，一台电视机，跟以身相许，这两个条件也太不对等了吧。
<br/>　　看到吴嘉伟附身检查清波，叶少阳忙问怎么样了。吴嘉伟表示没事，就是神识重伤，昏过去了。
<br/>　　“山羊，这少年挺不错，神识也足够强大，他是你茅山弟子，你将来应该好好培养他。”吴嘉伟很郑重地嘱咐道。
<br/>　　叶少阳点了点头，再看身边那些之前跟自己一样被法阵折磨的生灵，一个个坐在地上，都是一脸的虚弱和疲惫，这是神识受伤的结果，开始会有些恍惚，慢慢可以恢复。
<br/>　　见大家都没事，叶少阳也放心了，跳上壕沟，一眼就看到了钟馗那顶天立地般的高大身躯，站在他对面的影魅——那一道瘦削的影子，显得十分渺小，在他身后的丛林和山野间，站着十几道跟他一样的影子，自然是他的弟子传人一类的，那几个盘古僧不见了踪影。
<br/>　　钟馗瞥了叶少阳一眼，道：“小天师，他可不是你的菜，想要出气，去追杀鲲鹏他们吧，往西边逃了，你的兄弟们不知能否拦截得住。”
<br/>　　叶少阳犹豫了一下，让小马跟吴嘉伟一块过去，自己留下来照看碧清，毕竟自己跟她最熟，一边望着钟馗和影魅。
<br/>　　对有救兵赶来，他其实并不意外，但为什么来的是钟馗？他是跟谁来的？
<br/>　　眼下他在跟影魅对峙，倒不好去问。
<br/>　　“钟天师，许久不见……”影魅先开口，冲钟馗拱了拱手，道，“你能来到这里，我却是很意外，只是不知，你是代表阴司来的，还是你个人所为？”
<br/>　　钟馗冷笑一声，“有区别吗？”
<br/>　　“若你是个人而来，自没什么好说的，若是代表阴司而来，那我这边就不得不问一声了。”
<br/>　　钟馗拔下帅旗，扛在肩上，道：“问什么？”
<br/>　　“阴司，是要插手轩辕山之事么？”
<br/>　　钟馗笑道：“你莫要搬出轩辕山来压我。”
<br/>　　“不敢，钟天师在大帝门下，也是听诏不能宣，谁人敢压你？”
<br/>　　钟馗沉吟半晌，道：“影魅，你蛰伏轩辕山多年，本是化外强者，为何也要趟这股浑水？我也问你一句，你之所为，代表你个人，还是轩辕山？”
<br/>　　不等影魅开口，钟馗接着说道：“若是你个人所为，那你追随星月奴和清长风霍乱人间，便是霍乱人间，阴司不能不管，便我身为阴司一散人，也不能不管，若这是你轩辕山的态度，那阴司与你等，便要撕破脸皮了……”
<br/>　　影魅冷笑。“阴司要横插一脚，别怪轩辕山不讲情面。”
<br/>　　“好啊，星月奴本已是阴司通缉要犯，如今却要加你一个了。”
<br/>　　“等你们先出去再说吧……山水有相逢，少陪。”影魅又拱了拱手，人影极快地想后掠去，那十几个人影也跟着一起走。
<br/>　　“先收你点利息！”
<br/>　　钟馗右手突然探出，长袖拂过，在前方扇起了一道强烈的罡风，那群影魅的弟子中，有五六个跑得慢的，被罡风卷到，立刻被横着吹出去，钟馗骤然出手，一只巨大的手掌朝他们抓去。
<br/>　　这几人看走不掉，只好横下心来，人影连闪，凑到一处，合成一道巨大的幻影，朝大手撞过去。
<br/>　　“作死！”
<br/>　　钟馗一巴掌拍下去，打在幻影上，只是顿了一下，便硬生生拍下去，立刻打碎了两道黑影，其余地想跑，被大手牢牢抓住。
<br/>　　大手缩回到面前，钟馗只看了一眼，皱眉道：“最讨厌你这样的，我爱吃眼珠子做下酒菜，你这什么邪灵，连眼珠子都没有！”嘴巴一张，将这几道黑影扔进了口中，在一片惨叫声中，将黑影咬碎——不管是什么生灵，有没有实体，在钟馗这里，都是一样的可以吃的东西，只要进了他的嘴，再多的变化也没用。
<br/>　　吃完之后，钟馗从腰间拔下酒葫芦，灌了一口，望着远处的影魅一干人，咕哝道：“下一个就是你了。”
