<br/>　　<br/>　　  “在你们面前是连绵的东山，也是这座山阻隔了新奉县的发展，”王康伸手遥指。<br/>　　<br/>　　  “今天，我就要征服它，我要开山修路，盘活整个新奉县，而这只是我建设新奉县的第一步！”<br/>　　<br/>　　  “这条路，我将它命名为奇迹之路！”<br/>　　<br/>　　  “现在我问你们，有没有信心跟我一同创造奇迹！”王康大喊！<br/>　　<br/>　　  “有！”<br/>　　<br/>　　  所有的人都在卖力撕吼，王康这一番话，已经将城民的情绪调动，将他们的激情点燃。<br/>　　<br/>　　  更确切的说，是给了他们希望！<br/>　　<br/>　　  “那好，现在行报土，祭拜之礼！”王康沉声道。<br/>　　<br/>　　  这是地方习俗，修路动工，又称动土，开基！<br/>　　<br/>　　  报土，即是向土地公报告动工时间，唯恐惊扰土地山神，故向土地公打招呼，以求吉利！<br/>　　<br/>　　  在王康话音之后，立即有手下搬上条案长桌，其上已经摆满贡品，猪头，鸡，鸭和一些水果，糕糖。<br/>　　<br/>　　  正中放置着一顶香炉！<br/>　　<br/>　　  由王康率领站于中间，两旁则是林语嫣，聂中行，骆宾等主事之人。<br/>　　<br/>　　  仆从给每人都递来长香，王康点燃，一众人向着东山，躬身三拜。<br/>　　<br/>　　  这是必不可少的仪式。<br/>　　<br/>　　  在前世动土开基也有。<br/>　　<br/>　　  仪式过后，王康又转身面向下方开口道：“一般行报土，祭拜之礼后，都会铲第一锹土，以示开工！”<br/>　　<br/>　　  “但今天，我不铲这第一锹，而是以一种特殊的方式，庆祝开工大吉！”<br/>　　<br/>　　  听到此话，人们的目光都落在山脚之下，那里有一处已经被木栏所围，并画有线条。<br/>　　<br/>　　  代表着开挖，要从那里开始。<br/>　　<br/>　　  看到这一幕，董乾笑着道：“这可真是一场闹剧啊，看王康这般阵仗是要来真的，他也敢想？”<br/>　　<br/>　　  “东山连绵，多以乱石居多，开挖难度极大，凭那点人就想开凿吗？真是想太多了。”<br/>　　<br/>　　  “听说王鼎昌这几天亲赴各州开展生意，连开工仪式都不顾上参加，去努力赚钱，但只怕他赚的速度，跟不上他儿子败的速度啊！”<br/>　　<br/>　　  “还奇迹之路，我看是败家之路，这条路要么最终雷声大雨点小，要么会将伯爵府拖垮！”<br/>　　<br/>　　  “这么看来，他们拿到封地，别无益处，还是为害啊！”<br/>　　<br/>　　  “轰隆！”<br/>　　<br/>　　  几人笑谈间，突然从下方传来一声闷响，而他们所站的位置都是摇晃起来。<br/>　　<br/>　　  董易武瞬间变色，急忙道：“发生了什么？是地震了吗？”<br/>　　<br/>　　  摇晃片刻即归于平稳，而这时郑启功却有些愣神。<br/>　　<br/>　　  片刻后他好似想起了什么，惊声道：“我知道那个败家子，为什么有把握开山修路了……”<br/>　　<br/>　　  【作者题外话】：第一更，求银票，感谢<br/><br/><br/>
