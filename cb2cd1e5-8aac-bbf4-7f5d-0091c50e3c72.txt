　　不但凌嚣的脸色大变，
<br/>　　就连孙长老和绝无敌两人，同样脸色无比难看，
<br/>　　他们怎么也没有想到，以他们的身份地位，已经把姿态摆的如此之低，给足了叶尘面子，对方居然还敢要凌嚣的命！
<br/>　　“叶公子，你一定要做这么绝吗？”
<br/>　　孙长老寒声道，眼眸中透着一抹浓浓的怒火，显然在极力的克制。
<br/>　　如果换做是其他人，又或者叶尘没有取得那种成绩之前，他此刻肯定早就一巴掌扇过去了。
<br/>　　可是现在，不到万不得已，他实在不愿意和一个如此逆天的妖孽结下梁子。
<br/>　　叶尘冷冷一笑，
<br/>　　“绝吗？我并不觉得！若是此刻输的人是我，你还会这么说吗？”
<br/>　　孙长老默然，
<br/>　　的确，
<br/>　　若是叶尘输了，以凌嚣的脾气，肯定会先对叶尘羞辱折磨，然后再将他弄死！
<br/>　　而对于孙长老而言，一只蝼蚁的死活，他才不会在乎！
<br/>　　“叶尘，我们好话说尽，不过是想和你化解之前的不快，你如此固执，当真以为，我擎苍宗是好欺的吗？”
<br/>　　绝无敌也不由得脸色一沉，声音有些发寒道。
<br/>　　他其实心里对叶尘嫉妒的发狂，只不过如今叶尘锋芒正盛，若是无缘无故将其斩杀，必定会引起众怒！
<br/>　　所以，他才会决定退让一步。
<br/>　　可是他怎么也没有想到，自己已经把如此低声下气，对方居然还不知足，这顿时激起了他心中的强烈杀意。
<br/>　　在绝无敌看来，像叶尘这样的绝世妖孽，要么尽量不去得罪，可若是一旦矛盾不可调和，必须尽早除去，否则无论是对他个人，还是对整个擎苍宗，都会后患无穷！
<br/>　　凌嚣似乎也察觉到了，绝无敌和孙长老的杀意，顿时精神一震，也一下子有了底气，
<br/>　　“姓叶的，不要以为你登顶了千劫神塔，就觉得自己多了不起，在我擎苍宗的面前，你也不过就是一只小小的蝼蚁罢了！是，我是打赌把命输给了你，可是你有种就杀了我呀！你有这个胆子吗？你敢吗？”
<br/>　　凌嚣得意之下，开始肆无忌惮的冲着叶尘嘲讽起来。
<br/>　　在他看来，且不说叶尘有没有这个胆子杀他，即便叶尘真的想杀他，旁边的孙长老和绝无敌两人，也肯定都不会坐视不理。
<br/>　　以叶尘区区化神境一重的修为，当着一名合道仙君的面杀人，这有可能吗？
<br/>　　甚至，叶尘如果真的敢出手的话，那孙长老就可以此为借口，反杀了这个祸患！
<br/>　　叶尘听到这话，顿时缓缓抬起头来，一脸淡漠的望了一眼凌嚣，
<br/>　　“原本还想给你留点颜面，既然你不知好歹，那我就成全你！”
<br/>　　说完这话，叶尘脚下猛然一顿，
<br/>　　“时空法则！”
<br/>　　唰！
<br/>　　下一刻，
<br/>　　叶尘几乎犹如瞬移一般，来到了凌嚣的跟前。
<br/>　　速度之快，别说凌嚣根本没有反应过来，即便是那位身为合道仙君修为的孙长老，显然也没有想到，叶尘的速度，竟然如此之快！
<br/>　　而且，最重要的是，他刚刚没有察觉到任何真元波动，叶尘便已经跨越十几米的距离，这，简直不可思议！
<br/>　　彭！
<br/>　　叶尘欺身到凌嚣跟前后，没有任何犹豫，直接一掌拍在了他的脑袋之上，
<br/>　　“毁灭法则！破灭！”
<br/>　　毫无任何花哨的一掌，而且没有任何真元波动，看上去似乎好无力道。
<br/>　　然而，
<br/>　　这一掌落下，
<br/>　　彭！
<br/>　　凌嚣竟然犹如气泡一般，直接化成了一团血雾，甚至连惨叫都没有来得及发出。
<br/>　　嘶！
<br/>　　等到周围众人反应过来怎么回事，整个大会场，顿时死一般的寂静！
<br/>　　谁也没有想到，叶尘竟然一掌拍死了凌嚣！
<br/>　　而且，还是当着孙长老和绝无敌两人的面，就这么堂而皇之的杀人！
<br/>　　“他是疯了吗？”
<br/>　　所有人全都不可思议的望着那道身影，一时间全都懵掉了。
<br/>　　在众人看来，孙长老之前之所以不敢对叶尘动手，那是因为没有正当理由，怕引起众怒，说他以大欺小，嫉贤妒能！
<br/>　　可是现在，叶尘公然斩杀擎苍宗的弟子，就等于给了孙长老出手的机会啊！
<br/>　　甚至，
<br/>　　在很多人在怀疑，叶尘当众杀了凌嚣，孙长老根本就不是来不及阻止，而是故意为之，为的就是找到一个借口，借此铲除叶尘！
<br/>　　“竖子安敢！！”
<br/>　　孙长老反应过来之后，顿时气得肺都要炸了，
<br/>　　此刻根本顾不上去思考，叶尘刚刚用的是什么手段，连他都没有来得及阻止，就灭掉了凌嚣。
<br/>　　轰隆！
<br/>　　一道高达数千米的金色虚影，自孙长老的身上冲天而起，合道仙君的实力，一下子便彻底展露了出来！
<br/>　　呼！！
<br/>　　整个千劫山山巅，在孙长老的气息笼罩之下，犹如凭空掀起了一场飓风！
<br/>　　蹬蹬蹬！
<br/>　　一些离得近的修真者，顿时一阵东倒西歪，纷纷倒退！
<br/>　　合道仙君，在如今整个千劫山的山巅之上，绝对是最顶级的存在！
<br/>　　“诸位也都看到了！这小子狂妄无礼！杀我门下天才！根本没有把我擎苍宗放在眼里！老夫今日若不给此子一点教训，我擎苍宗的颜面何存？”
<br/>　　孙长老显然也是害怕激起众怒，并没有说要杀了叶尘，只是说给他一点教训，不过心里却暗暗打定主意，一定要想办法，断了叶尘的修道根基！
<br/>　　轰！
<br/>　　孙长老在说出这话的同时，已经伸出巨掌，闪电般朝叶尘抓去！
<br/>　　之前他是真心想和叶尘化解矛盾，可是到了现在，想要和解已然是不可能了，既然如此，那就要尽早铲除这个威胁！
<br/>　　叶尘两眼微微一眯，站在原地没有动弹，落在众人的眼中，似乎是已经吓傻了。
<br/>　　其实他们又哪里知道，叶尘在进入千劫神塔之前，就已经足以抵挡合道仙君。
<br/>　　而如今在千劫神塔中，又凝聚出了四十九道符文，实力提升了十倍都不止，既便是面对渡劫帝君，他也未必没有一战之力。
<br/>　　至于眼前的孙长老，他已经有绝对的把握战而胜之！
<br/>　　“看来只能提前暴露自己的实力了……”
<br/>　　叶尘在心里嘀咕了一句，准备动用四十九条法则符文，去抵挡孙长老的攻击。
