　　这声音中蕴含的威压，显然是一尊久居上位的大人物，自然而然散发而出的。
<br/>　　随着声音的传来。
<br/>　　裂缝突然扩大，从中一尊身影，缓缓走出，这是一尊中年人的样子，他的腰间，有着一把佩剑，一双眼睛，目光如炬。
<br/>　　楚风眠感觉到，此人身上隐藏的凌厉剑意。
<br/>　　现在这中年人看起来，只是一尊寻常武者，但是此人，绝对是一尊不可不扣的剑修，甚至是一尊极为强大的剑修。
<br/>　　他的剑意，现在不过是平息了下来，看似平常。
<br/>　　但是一旦是此人出剑，他身上的剑意，便是会彻底的激发出来，跟现在的样子，截然不同。
<br/>　　“这是我们北邙学院的一尊副院长，真正的大人物，夜众星。”
<br/>　　寒月漓在一旁，轻声开口道。
<br/>　　“此人也是夜家之人，小心一点。”
<br/>　　夜众星？
<br/>　　楚风眠一双眼睛，看向这名为夜众星的副院长。
<br/>　　北邙学院的副院长，许多可都是超越了老祖宗的人物，就像是眼前这夜众星，他的实力都超越了夜牧老祖。
<br/>　　不过跟玄北圣者相比，还逊色一些。
<br/>　　玄北圣者的境界，楚风眠还无法看出，但是至少也是在生死台阶上，走的很远的人物。
<br/>　　楚风眠双眼看向这夜众星，这夜众星的一双眼睛，也看向了楚风眠。
<br/>　　突然这夜众星的双眼目光，猛然变化，眼神如同一把利剑，直接向着楚风眠刺来。
<br/>　　目光化剑。
<br/>　　这是剑道达到极高的造诣，才足以做到的一点。
<br/>　　这一招目光化剑，像是试探，却也蕴含杀意，若是楚风眠抵挡不住，很有可能被直接轰杀。
<br/>　　“想要对付我？在剑道上，没有人可以战胜我！”
<br/>　　面对着这一道利剑刺来，楚风眠心中冷笑连连，他身上的剑意也是被直接催动起来。
<br/>　　楚风眠的目光，同样也是化为了一把利剑，两者相撞，这夜众星的剑意，都被楚风眠死死的压制住了。
<br/>　　下一刻，他的那一道灵剑，直接破碎。
<br/>　　反噬的力量，将夜众星都给逼退数步。
<br/>　　虽然无法将他震伤，但是这结果，却似乎也是在夜众星的意料之外。
<br/>　　“好强的剑道，这小子的剑道，甚至还在我之上？怎么可能？”
<br/>　　夜众星的心中，都满是不可置信。
<br/>　　要知道他可是北邙学院的副院长，老祖宗一般的人物，而楚风眠只是一个神力境的弟子。
<br/>　　这样的剑道交锋，本来应该是没有任何悬念才对，他心中的算计，本来都是想要趁着玄北圣者不注意之际，直接灭杀了楚风眠。
<br/>　　却没想到，在剑道交锋上，他居然是输了。
<br/>　　“天君死在此人的手中，倒是不冤。”
<br/>　　夜众星看着楚风眠，暗自想到。
<br/>　　夜天君是他们夜家重点培养的天才，本来放眼北域，都是同境界无敌的存在，死在了楚风眠的手中，他们夜家内部都是掀起了轩然大波，似乎都没有意料到。
<br/>　　但是这一次交手，夜众星便是清楚的明白了楚风眠的实力，起码在剑道上，楚风眠的造诣，深不可测。
<br/>　　“此子不杀，必成大患！”
<br/>　　夜众星心中杀意暴起，恨不得现在就要杀了楚风眠，但是一想到他来之前学院的安排，他只能是恨恨的压制下去心中的杀意。
<br/>　　“副院长。”
<br/>　　玄北圣者看到夜众星前来，冰冷的开口道。
<br/>　　“不知道副院长这次前来，所为何事。”
<br/>　　“没什么，你的弟子楚风眠，在这一次武斗大会上，表现出众，为我们北邙学院争光，这一次学院便是让我前来，代表学院，赐下奖励，以示嘉奖。”
<br/>　　夜众星平静的开口道。
<br/>　　这次武斗大会的事，北邙学院的高层自然也曾知晓，他这位副院长，便是代表着北邙学院前来。
<br/>　　“赐下奖励？嘉奖？”
<br/>　　听到夜众星的话，那夜牧老祖猛然开口道。
<br/>　　“众星，此人斩杀我们夜家弟子，今日必须要杀了他，怎么可能给他嘉奖！”
<br/>　　那夜天君，正是这夜牧老祖的孙子。
<br/>　　夜牧老祖，对楚风眠恨之入骨，恨不得将楚风眠现在都给活活撕碎。
<br/>　　听到夜众星居然是说要嘉奖，急忙开口道。
<br/>　　“夜牧，这是学院高层做出决定，更何况天君是在武斗大会上，死在擂台上的，怪不得别人。”
<br/>　　夜众星看着夜牧老者，摇了摇头说道。
<br/>　　“一切要以学院的决定为主。”
<br/>　　夜众星说完，便是看向了楚风眠，从他的手中，一枚金色龙丹突然出现，直接飞向了楚风眠。
<br/>　　“这是学院的奖励，代表着你为学院做出的贡献，接下来的归墟天，你也要在其中继续得到无数好处，为我们北邙学院扬名。”
<br/>　　说完，夜众星便是不在理会楚风眠，对着玄北圣者开口道。
<br/>　　“玄北，这次的事是夜牧鲁莽了，夜牧，我们走。”
<br/>　　夜众星跟夜牧老祖两人的身形，便是一同进入到了一道破碎裂痕之中，在这北邙圣域中消失了。
<br/>　　“这龙丹？好东西！”
<br/>　　楚风眠等到夜众星跟夜牧老祖离开，心中终于也是轻松了不少。
<br/>　　果然是跟他猜想的一样，这一次楚风眠夺得北邙学院榜首的位置，可是在北域扬名，北邙学院不止不会杀了楚风眠，还会赐下奖励。
<br/>　　这龙丹，楚风眠一眼望过去，便是知晓其中的价值。
<br/>　　这至少是一枚真龙的龙丹，而且还是至少一尊龙王的龙丹。

<br/>　　龙王，可都是相当于武者之中，超越生死的大能，眼前这枚龙丹生前龙王的实力，只怕是比起玄北圣者，都丝毫不弱，他的龙丹，简直是价值无限。
<br/>　　这一枚龙丹，简直是诸天奇物之中最为顶尖的东西，可遇而不可求的东西。
<br/>　　尤其是楚风眠修炼战龙决，这龙丹对他来说，绝对是最大的一个宝藏，比起任何东西，都要有用。
<br/>　　在楚风眠冲击生死大劫的时候，这龙丹有着大用。
<br/>　　楚风眠观察了一阵，便是大手一挥，直接将这龙丹收走，随着他的身形，便是直接回到了悟剑峰中，寒月漓跟玄北圣者，也是一同，落入到悟剑峰之中。
