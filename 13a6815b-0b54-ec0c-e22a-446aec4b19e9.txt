        明司寒见连星儿不肯跟自己说实话，也懒得再追问下去了。
<br/>
<br/>        因为，他用脚趾头想想就能猜到，这丫头十有八九是去见洛羽了。
<br/>
<br/>        每一次，她跟洛羽见面，都是偷偷摸摸。
<br/>
<br/>        这次只怕也不例外。
<br/>
<br/>        一想到洛羽，明司寒的语气格外尖酸刻薄：“不管怎么样，我和你彼此是双方父母见过面已经订下婚约的未婚夫妻，虽然我两都有意解除婚约，但是在这婚约还未解除之前，你都不可以跟其他男人有暧昧关系。”
<br/>
<br/>        “？！”听着明司寒这话，连星儿是一脸懵。
<br/>
<br/>        “懂了吗？”明司寒睐了连星儿一眼，问道。
<br/>
<br/>        连星儿一脸不高兴地闷哼道：“你事真多！”
<br/>
<br/>        “喂，这不算是我的事多吧？”明司寒伸手搭在连星儿的椅背后，身子向她倾近，勾唇一笑，“当然，我也会在这期间为你守身如玉，绝不跟任何一个女生暧昧。”
<br/>
<br/>        “……”连星儿侧头看着这家伙，顿时汗颜，反问，“你跟谁暧昧，跟我一点关系也没有。”
<br/>
<br/>        “！！！”明司寒俊脸紧绷，脸色瞬间极其不好看。
<br/>
<br/>        而这个时候，两人之间响起了一个嗲声嗲气的女音：“明学长呐，你可不可以，也帮我批阅一下，我的英语试卷呢？”
<br/>
<br/>        这段时期来教室里自习的大多数都是备考英语四六级的，所以，自习室里做英语测试卷的同学自然不止她连星儿一个。
<br/>
<br/>        现在已经是下自习的时间，所以教室里不再像刚刚那么安静。
<br/>
<br/>        连星儿和明司寒同时循声望去，只见坐在他们两对面的一个女生双手拿着卷子，娇滴滴地向明司寒这边递了过来。
<br/>
<br/>        明司寒淡漠地将目光从这个女生身上收回，云淡风轻地吐了字：“滚！”
<br/>
<br/>        女生身心一怔，不禁涨得满脸通红，有些无地自容地缩回了手。
<br/>
<br/>        顿时，坐在一旁的其他女生交头接耳，窃窃私语起来。
<br/>
<br/>        连星儿看了女生一眼后，又看了看身旁那些嘲笑这女生的其他女生们，为了化解这女生的尴尬，连忙帮腔道：“学姐的测试卷是六级的诶！我看你八成是不会吧！”
<br/>
<br/>        “你小看我？”明司寒不悦地蹙起了好看的剑眉。
<br/>
<br/>        连星儿嘴角微扬道：“那你倒是批个给我看看呀！正确答案和解析，一个都不能少。”
<br/>
<br/>        “如果，我全批对了。待会，你跟我去小树林里约会。”明司寒勾唇一笑。
<br/>
<br/>        连星儿脸上的笑容一僵，有些心虚地转动着眼珠子，讷讷地应道：“那也得等你全批对了再说吧！”
<br/>
<br/>        “好！”明司寒会心笑了笑，伸手从那女生的手中夺过卷子，再次拿起红笔批阅起来。
<br/>
<br/>        他在批阅试卷的途中，一旁的女生们纷纷凑了过来。
<br/>
<br/>        只不过一分钟的时间，长桌旁便围了两圈女生。
<br/>
<br/>        明司寒一边批阅试卷一边讲解做题思路，以及易错的知识点。
<br/>
<br/>        他的英语发音很标准，让在座的女生们听得聚精会神。
<br/>
<br/>        因为要讲解错题，所以批阅的时间长了点。
<br/>
<br/>        半个小时后，明司寒讲完最后一道错题，周遭女生们情不自禁地惊赞鼓掌。
<br/>
<br/>        连星儿瘪着嘴，但心里却对明司寒佩服得五体投地。
<br/>
<br/>        果然，她在他小时候住过的那间屋子里发现的奖杯、奖状、证书，他都不是白拿的。
<br/>
<br/>        待大家散去后，自习室里的同学也都走得差不多了，连星儿也开始收拾课本准备起身离开。
<br/>
<br/>        现在才晚上九点半，离宿舍宵禁还有一个半小时。
<br/>
<br/>        明司寒将自己的课本和笔袋拿起来后，直接放到了连星儿的手中，说道：“你帮我拿着，我去一趟洗手间。别忘了，你答应我的事情。”
<br/>
<br/>        “喏！”连星儿忍不住对明司寒吐了吐舌头，“我才没答应。”
<br/>
<br/>        明司寒只笑不语，而是转身往走廊外走去。
<br/>
<br/>        连星儿看着明司寒离开的背影，不经意间地扬起了嘴角，心里莫名其妙地感到有一丝甜意袭来。
<br/>
<br/>        当自习室里的同学都走光了后，自习室里就只剩下她连星儿一个人了。
<br/>
<br/>        明司寒才离开不过两分钟的时间，自习室里的灯突然灭了。
<br/>
<br/>        连星儿吓得连忙从座位上起身，向窗外眺望过去。
<br/>
<br/>        其他教学楼的灯也都灭了，只有校道上的路灯是亮着的，看样子是学校里停电了。
<br/>
<br/>        连星儿索性抱着书本从座位上起身，打算去走廊上等明司寒。
<br/>
<br/>        当她走到教室门口时，一个黑影突然窜到她面前，将她吓了一跳。
<br/>
<br/>        就在她还未来得及看清楚面前的人是谁时，对方便向她伸出了手，直接捂住了她的嘴巴。
