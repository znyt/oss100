　　第313章
<br/>　　程大雷回到蛤蟆寨时，见整个工程已经完成大半，不得不说，在建筑精通技能的加持下，工程进度是很快的。
<br/>　　到处都是忙碌的工人，每个人都干得热火朝天，可这只是表面好看，内里蛤蟆寨的消耗已经十分严重。
<br/>　　当程大雷坐在聚义厅，听着刘悲的汇报时，眉头不自觉的皱了起来。
<br/>　　“现在山寨干活的人超过三千，算上我们本来的弟兄，有超过四千人要吃饭，而如今山寨的存粮，最多支撑三天。这几日已经让子龙他们去抢劫周围的大户，却仍旧是入不敷出。”
<br/>　　“另外需要的大木山石已经让弟兄们去山上开采，但需要很多铜铁，这些东西只能高价买，价格很高，继续建设下去，至少还要一千斤铜铁，咱们已经拿不出这么多钱了。”
<br/>　　“……”
<br/>　　刘悲一笔笔汇报着，另外拿来账本给程大雷看。程大雷看到上面一笔笔账都记着很清楚，一目了然，笔迹也很娟秀。
<br/>　　“看不出来啊。”程大雷赞了一声：“皇叔还有这样好的小楷，难得账也算得清楚。”
<br/>　　“不是我算的账。”
<br/>　　“咦，那是谁？”程大雷。
<br/>　　“公主殿下。”
<br/>　　蛤蟆寨只有一位公主，在蛤蟆寨大家对李婉儿还是十分尊崇的。毕竟是帝国公主，有天生的威信，程大雷不拿公主这头衔当回事，不代表刘悲这样土生土长的帝国人也不当会事。
<br/>　　程大雷只是没想到，李婉儿还能算得一手好账，想来却是一直埋没了他。
<br/>　　据刘悲所说，自从工程开始后，各种各样的事情要处理，靠刘悲一人根本忙不过来，就算有李行哉帮忙也是一样，关键时刻，幸亏李婉儿出手，将每笔账目算得一清二楚，才让事情进入该有的轨道，没有增加无所谓的消耗。
<br/>　　至于如何处置眼前的困境，李婉儿是没有办法的。如今程大雷已经回来了，这些事情倒也算不得太大问题。
<br/>　　先派人去西门春长哪里，将带回来的草药卖了，换一船粮食回来救急。同时，莫忘买一百匹花布，这是黑岩预定的。
<br/>　　之后，程大雷又赶紧派人携带工具去落玉寨，把那座架在峡谷上的木桥整修加固，这座桥将是蛤蟆寨以后主要的经济支柱。
<br/>　　如今的程大雷，像是两只手抛五个球，只要有一个球落地，他的盘子也就算玩砸了。
<br/>　　在云中龙三人将粮食送来时，眼前这场危机总算过去了，程大雷也总算是没有玩砸。
<br/>　　三人送来的不仅是粮食，还有茶叶，绢布，铁器，瓷器等等。这些是准备送到草原交易的，新修好的木桥已经允许大批货物通过。这一次，程大雷便不再亲自出马，而是交给了……李行哉。
<br/>　　李行哉对一切充满好奇，吵着要体验带商队，他虽然偶尔有些不着调，但关键时刻还是靠谱的。于是，程大雷便让他带着福德勒一起去了。
<br/>　　一切迈入正轨，只要抢时间，能在莫鸣米醒悟过来时，把山寨升到三级，那么程大雷便不再怵他。到时候，究竟谁是这块土地的主人，程大雷就要找他好好商量商量。
<br/>　　程大雷留在山寨，开始阅读酒楼传过来的消息，有江湖传言这个技能加持，程大雷收到的信息覆盖整个帝国。当然，因为是江湖传言的缘故，其中几多真，几多假，还需要程大雷自己分辨。
<br/>　　『南海一伙夷商登岸，贩卖来自异域的女奴』
<br/>　　『草原王死后，戎族爆发内乱，诸多王子开始争权，现在大战还未开始，各自都是磨刀霍霍，蓄势待发』
<br/>　　『北蛮王病卒，将王位传给金问道』
<br/>　　『江南一地，有正义教起事，信徒数十万，朝廷已开始派兵围剿正义教信徒』
<br/>　　……
<br/>　　……
<br/>　　程大雷读得累了，刚抬起头，一杯茶已经摆在桌上。然后银眸背着剑退到屋子角落里，一言不发。
<br/>　　当然，她也不会说话，这些日子，银眸对程大雷是如影随形，永远怵在程大雷身后。一开始程大雷有些不适应，被人总是这样盯着，总有种毛骨悚然的感觉。但到后来，他也就习惯了，只要当她不存在就好。
<br/>　　程大雷抿了口茶，继续阅读传来的信息。有些信息完全无用，都是垃圾信息，而有些信息却是窥一斑而知全豹。风起于青萍之末，有些看上去无所谓的信息，却有大势酝酿其中。
<br/>　　正义教……
<br/>　　程当家将关于正义教的条子抽了出来，回头嘱咐酒楼那边多收集关于正义教的信息。草原上发生的事情，程大雷并不觉得意外，无非就是权力斗争，你杀我我杀你的。可这正义教的兴起，却让程大雷不得不郑重。
<br/>　　帝国千疮百孔，岌岌可危，如今一伙拥有十几万信徒的异教徒兴起，会不会成为压死骆驼的最后一根稻草。
<br/>　　据说，朝廷已派了大兵去剿匪，而苏樱就在江南，不知这场匪乱她们能否度过。
<br/>　　一想到这里，程大雷再也无法安稳的待在蛤蟆寨。他决定走一趟江南，把苏樱接回来。
<br/>　　“大当家，你不要冲动，你不要冲动……”刘悲几人死死的拦住程大雷。
<br/>　　“我冲动什么。”程大雷无语道：“我就是单枪匹马，当然徐军师一定是要跟着我的，靠我手中剑，一路上就算遇到什么危险，当也度得过。”
<br/>　　“我们都知道大当家的本事。”刘悲道：“可山寨目前刚起步，还有强敌环伺，大当家这一走，人心就乱了。”
<br/>　　“我去去就回，至多也就一个月。”
<br/>　　“完全可以派别人去么，张肥、子龙他们都可以去，以他们的本事，绝对出不了岔子。”
<br/>　　让别人去，终究没有自己去放心。可刘悲说得不错，这山寨目前的确离不开自己。
<br/>　　他颓然的在椅子上坐下，随之苦笑一声，却原来手下人越多，自己能做的事情却越来越少。
<br/>　　最后，程大雷派了秦蛮，高飞豹，关鱼，张肥前去江南，一口气派出四员大将，足见程大雷对这件事的重视。
<br/>　　至于赵子龙，是绝对不允许去的，万一逼的苏樱跳井了怎么办。
<br/>(本章完)
