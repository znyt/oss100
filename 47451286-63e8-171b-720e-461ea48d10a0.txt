　　后山上，萧逸参悟着凝丹五行录。
<br/>　　今日，易老的教导，让他受益匪浅。
<br/>　　起码不会如昨晚那般，如同看天书一样。
<br/>　　不得不说，易老的教导本事，确实厉害。
<br/>　　那些晦涩的炼药知识，复杂玄奥的药材组合，在他口中说出来，轻易间就成了简洁明了的知识。
<br/>　　丹药精炼，除去杂质的办法。
<br/>　　便是在炼丹过程中，丹药即将凝丹之时，加入妖兽内丹。
<br/>　　在炼制丹药过程中，加入的天材地宝，或多或少会带有各种属性。
<br/>　　如百年玄参，土属性；妖血炎心果，火属性；培元朱果，木属性；铁杉枝，金属性....
<br/>　　又如炼制雪魄丹的材料，大多都是冰属性材料。
<br/>　　故，炼制的丹药，其实或多或少，都会带有一点儿属性。
<br/>　　凝丹五行录，就是依靠这些属性，通过妖兽内丹，将丹药杂质去除。
<br/>　　当然，丹药有杂质，妖兽内丹，同样带有杂质和残暴气息。
<br/>　　但，五行，相生亦相克。
<br/>　　如，一颗带有冰属性的丹药。
<br/>　　在凝丹过程中，加入冰属性妖兽内丹，便可让其本身的冰属性大增。
<br/>　　在其完全结丹前，将杂质‘挤走’。
<br/>　　这就是所谓的相生。
<br/>　　至于冰属性妖兽内丹上的杂质和残暴气息，则只需加入火属性妖兽内丹，冰火对抗，将杂质和残暴气息消磨掉。
<br/>　　这就是所谓的相克。
<br/>　　由此，丹药上的杂质、妖兽内丹上的杂质，可完全消去。
<br/>　　总结起来，就是一句话。
<br/>　　相生，除丹药杂质；相克，除妖兽内丹杂质。
<br/>　　这就是凝丹五行录的原理，也是易老今日所教。
<br/>　　“难怪之前抢夺的那批商物中，除了匕首、毒药和成品丹药外，还有那么多妖兽精血和妖兽内丹。”萧逸沉声自语着。
<br/>　　“暗影楼，杀手众多。妖兽精血，是给杀手们药浴，提升修为的，或者还有其他用途。”
<br/>　　“妖兽内丹，则是暗影楼内的炼药师炼制无杂质的丹药所用。”
<br/>　　凝丹五行录，并非裂天剑派独有。
<br/>　　据易老所说，这是一些高深的炼药师，必须学会的手段。
<br/>　　不过，整个北山郡，能有这个本事的，不算裂天剑派在内，只有三人。
<br/>　　这三人皆是各大势力的炼药师，一个在猎妖殿，一个在暗影楼，一个则在北山郡王麾下。
<br/>　　而这三人，虽都是五品炼药师，但却远远比不上裂天剑派的药堂长老。
<br/>　　故这三人就算有这个本事，但炼制起无杂质的丹药来，非常慢，甚至是慢得可怜。
<br/>　　言归正传。
<br/>　　原理归原理。
<br/>　　原理，易老一天就教会了，并不难，也很容易理解。
<br/>　　可真正实施起来，却很是困难。
<br/>　　一颗丹药，带有一种属性的话，就要加入两颗相克属性的妖兽内丹。
<br/>　　但事实上，丹药，由于是许多种天材地宝融合炼制而成。
<br/>　　故属性不止一种。
<br/>　　这就需要加入更多的妖兽内丹了。
<br/>　　数量更多的妖兽内丹，代表着属性也变得更多。
<br/>　　于是，不同属性间的组合，就变得相当复杂了。
<br/>　　天材地宝在炼丹炉中，受高温火焰燃烧，其实大部分的属性都已经被烧没了，只会剩下一丁点儿。
<br/>　　而这一丁点儿，就是各种天材地宝的残余属性。
<br/>　　这些属性，到底孰强孰弱呢？是火属性更旺，还是木属性更盛？
<br/>　　这时，就要相应地加入能量强弱不同的妖兽内丹了。
<br/>　　总之，这些组合，千变万化，说也说不完。
<br/>　　最是考验炼药师的能力。
<br/>　　炼药，本身就是非常复杂的事；而凝丹五行录，虽然能去除杂质，却生生将难度提升了十倍、百倍不止。
<br/>　　就是因为这些太过困难的组合，需要炼药师的炼药知识无比丰富。
<br/>　　这才难倒了无数炼药师，别说叶铭，就是等闲五品炼药师，也是措手无策。
<br/>　　就算是那些成名已久的五品炼药师，懂凝丹五行录，可往往要在一颗丹药上费许多心神和时间，去考虑到底该如何加入妖兽内丹，故非常慢。
<br/>　　这对一个炼药师的领悟力，有着非常高的要求。
<br/>　　而萧逸的领悟力，便是高得吓人。
<br/>　　在易老的指导下，数月之内便可大致掌握这些组合，真正开始炼药。
<br/>　　易老用一日时间，教导原理；以后的时间，都是教导这些组合和炼药知识。
<br/>　　“数月时间，太久了。”萧逸摇了摇头，“而且那还只是大致掌握，炼制起来的速度也会很慢。”
<br/>　　“要想快速炼制无杂质丹药，只能达到融会贯通的的地步。可是，那不知要多少年以后。”
<br/>　　“太阴太阳之眼，这次又要靠你了。”萧逸自语着。
<br/>　　下一秒，他的双眼变了。左眼幽森如黑洞，右眼炽热如太阳。
<br/>　　而后，内视体内冰鸾剑武魂。
<br/>　　他不知道行不行，毕竟凝丹五行录不是丹方，而是炼药知识。
<br/>　　但他想试一试。
<br/>　　昨晚，他将凝丹五行录看第二遍，就是要将上面的内容全部记下来。
<br/>　　数秒后，当他双眼恢复正常，脸上明显露出了喜意。
<br/>　　因为，凝丹五行录上的一切知识，在他脑中，全部化为了最简单的内容。
<br/>　　那些所谓的复杂组合，五行相生相克，这一刻，对他来说，轻而易举。
<br/>　　他甚至有把握，自己现在足可完美地将无杂质丹药轻松炼制出来。
<br/>　　“竟然真的可以。”萧逸心中暗暗惊骇，且喜悦至极。
<br/>　　.....
<br/>　　翌日清晨，萧逸再次前往外门宝库。
<br/>　　当然，他没有告知易老，自己已经完全悟透凝丹五行录。
<br/>　　短短两三天就完全悟透，这已经不是天才了，简直是怪物。
<br/>　　易老一如既往地指导着。
<br/>　　但，萧逸却有意地加快了自己的学习速度。
<br/>　　半月后，他才告诉易老，自己已经完全悟透。
<br/>　　外门宝库内，易老咂舌地看着萧逸。
<br/>　　“小子，你是怪物吗？按照我的猜测，本以为你需要数月才能开始悟透；这才半个月，你就完全悟透了。”
<br/>　　这半月时间，易老完全是在各种惊讶表情中渡过的。
<br/>　　每日教导萧逸时，凝丹五行录上的困难之处，每每当他刚讲解完，萧逸已经举一反三，直接跳过。
<br/>　　甚至于，有些地方，还能给出更独特的见解，不，简直是堪称完美的见解。
<br/>　　有些时候，他倒觉得萧逸更像是自己的指导老师，比自己知道的还多。
<br/>　　萧逸苦笑一声，他早就料到易老会是这个表情。
<br/>　　幸好是推迟了半个月才说，否则，易老肯定直接将自己当怪物了。
<br/>　　至于那些举一反三，还有完美见解；他已经完美地悟透了凝丹五行录上的内容，自然很轻松就能做到。
<br/>　　“小子，你....”易老想说些什么，但最后却不知该说些什么。
<br/>　　萧逸这半个月来给他的惊讶，他完全归功于萧逸的恐怖领悟力。
<br/>　　他只是想着，或许，自己以前对萧逸的领悟力估计，还远远不足。
<br/>　　当然，他看着面前这个天赋无比厉害的‘弟子’，心头也是一阵满足。
<br/>　　作为一个‘老师’，‘弟子’出色，自然是非常高兴的。
<br/>　　“易老，支支吾吾做什么？”萧逸打趣道，“是想夸我厉害吗？”
<br/>　　“哼。”易老冷哼了一声，他确实想夸萧逸，但现在却道，“忘了我之前跟你说过的话了吗？切忌骄傲自满。”
<br/>　　“既然已经悟透了，便真正开始炼丹吧。”易老撇撇嘴说罢，直接扯开了话题。
<br/>　　“嗯。”萧逸点点头，他也想真正实炼一番。
<br/>
