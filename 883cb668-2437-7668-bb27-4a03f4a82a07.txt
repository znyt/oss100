　　轰...嘭...
<br/>　　萧逸与天星雷鳄的战斗，激烈而心惊动魄。
<br/>　　天星雷鳄的暗雷，一记就能让萧逸重伤。
<br/>　　要不是萧逸身上有灵器增幅，外加九大被金脉丹增幅过的主脉，联合激发出来的真气护身，说不定一记暗雷破就得被劈成齑粉了。
<br/>　　天星雷鳄也不好受。
<br/>　　萧逸的崩山斩，每一剑都让它皮开肉绽，接近重伤。
<br/>　　一人一兽都显得相当凄惨，浑身是伤，说不准哪方会先倒下。
<br/>　　但一旁观战的数百猎妖师，绝对是想天星雷鳄先死掉的。
<br/>　　“人类小子。”天星雷鳄也看出了萧逸的年纪并不大。
<br/>　　“虽然我被狂血玄君那老东西的禁制压制了修为，但以我操控雷电的本事，必是先天之下无敌，你赢不了我的。”
<br/>　　“最后先死的，必然是你。”
<br/>　　天星雷鳄狞笑着，满身是伤，却仍旧笑得出来，双眼中邪恶的目光充满了嗜血的意味，不愧是曾经的凶兽。
<br/>　　“我知道。”
<br/>　　萧逸第一次说话。
<br/>　　“你知道？还满脸不在乎的样子？”天星雷鳄胜券在握地笑道。
<br/>　　火刃与雷电的碰撞，剧烈的轰鸣，让得外人根本听不到他们的对话。
<br/>　　萧逸淡淡一笑，道，“就是因为我知道，所以我要拼一把。”
<br/>　　“拼，你拿什么拼？你以为拼得过？”天星雷鳄不屑地说道。
<br/>　　萧逸仍旧是淡淡的神色，“拿我的命，我这一生，拼过很多次，仅仅输过一次，丢了命，其余的，全拼赢了。”
<br/>　　唯一输过的一次，自然是前世抢回冰鸾剑那次。事实上，若不是冰鸾剑忽然有异动，自己动弹不得，根本不会死。
<br/>　　“你什么意思？”天星雷鳄眉头一皱，不明白萧逸所谓的丢了命什么意思。
<br/>　　但下一秒，它忽然瞪大了眼睛，脸上的不屑化作了惊骇。
<br/>　　只见萧逸同时凝聚了五十道火刃。
<br/>　　在烈焰手套的增幅下，每一道火刃的威力都不亚于先天九重的全力一击。
<br/>　　火刃，忽然在以极快的速度两两融合。
<br/>　　五十道火刃，融为了二十五道。
<br/>　　二十五道，仍旧在两两融合。
<br/>　　“人类小子，你想做什么？”天星雷鳄顿时脸色大变，“你在融合武技？该死，你疯了。”
<br/>　　萧逸动作不停，火刃仍旧在融合。
<br/>　　“疯子，快停下来。”天星雷鳄破口大骂。
<br/>　　另一边，猎妖师们也看到了萧逸的动作，立刻脸色大变。
<br/>　　李元惊声道，“糟糕，观那些火刃的强度，起码是玄阶武技，他竟然敢融合玄阶武技的攻击？”
<br/>　　王羽也道，“这需要对火焰极其精密的控制力，这样恐怖的控制力和领悟，几乎是不可能的事。连洞玄境武者，哪怕是绝世天才都做不到。”
<br/>　　天星雷鳄这边，惊声道，“快停下来，这种融合极其不稳定，一旦失去平衡，能量会暴走的。”
<br/>　　噬火百刃，每一道火刃都代表着一记玄阶武技。就算大成者，也只能同时打出一百道，以分散的形式。
<br/>　　从没有人敢去融合，也没有人有那个领悟力和控制力做得到。
<br/>　　更加没人敢去尝试。
<br/>　　因为，一旦失败，后果相当严重。玄阶武技的融合，可不是一加一等于二，很可能是一加一等于十。
<br/>　　一旦失去平衡，产生爆炸，谁也受不住。
<br/>　　天星雷鳄也不敢妄动，不敢去阻止，它也怕萧逸手一个不稳，这些火刃直接就爆炸了。
<br/>　　“人类小子，你这样只会害死你自己。”天星雷鳄忽然好言相劝。
<br/>　　“我的肉身乃是破玄境，你这种威力的攻击杀不了我的。”
<br/>　　萧逸嗤笑一声，“不用骗我，我知道你的肉身还在狂血玄君禁制当中。如今的你，不过是依靠能量凝聚出来的分身罢了。”
<br/>　　“否则，你不可能被我打伤。”
<br/>　　“你...”天星雷鳄咬了咬牙，道，“人类小子，待我破除禁制，我将是整个北山郡的最强者。到时，我许你一切要求。财富、名声、地位、实力，我一切都能满足你。”
<br/>　　天星雷鳄在示好，它已经能从那道火刃上感受到致命的威胁。
<br/>　　萧逸冷笑一声，“一旦你破除禁制，恐怕率先杀的人就是我吧。”
<br/>　　“你...”天星雷鳄没想到萧逸看穿了它的想法。
<br/>　　这时，五十道火刃已经完全融合成一道噬火百刃。
<br/>　　散发着恐怖的温度和骇人的气息，却又极不稳定。
<br/>　　萧逸竟然真的成功了。
<br/>　　萧逸脸上冷笑，心中其实也在拼，并无多大把握。
<br/>　　但他又毫无办法，一旦天星雷鳄破除禁制，掌控了洞府，所有人都要死，包括他自己。
<br/>　　他只能拼一把。
<br/>　　“暗雷破...暗雷破....暗雷破....”天星雷鳄知道火刃已成，再无回转的余地，立刻拼命似的凝聚暗雷破。
<br/>　　但它不是用来攻击萧逸，而是用来围绕在自己身边，以作防御。
<br/>　　萧逸脸色一冷，“看来，杀你这孽畜单靠噬火百刃还不够阿。”
<br/>　　说罢，萧逸忽然伸出手，握住了这融合后的恐怖噬火百刃，将其当成了自己手中的剑。
<br/>　　另一边，众猎妖师见到萧逸的动作，顿时冷汗直流。
<br/>　　“那强者，该不会是想把融合后的武技当作武器使用吧...”
<br/>　　李元等人立刻反应过来，“大家快跑。”
<br/>　　融合武技而成的攻击已经相当不稳定了，还要拿来当武器挥动。
<br/>　　就像一个用水装满的气球，随时都会爆开，却还拿去到处耍，想不爆都难。
<br/>　　话音刚落，所有人都纷纷往通道跑去。
<br/>　　通道上原本有雷电堵住，但萧逸牵制天星雷鳄后，那些雷电便都消失了。
<br/>　　“快离开这里，离开这个洞府。”
<br/>　　所有人仿佛都恨自己少生了一条腿般，卯足了劲向外跑。
<br/>　　只有浅沫，愣愣地站在原地，说什么都不愿意走，仿佛要等那道背影。
<br/>　　“浅沫，易霄兄弟已经死了，你别多想了。”
<br/>　　铁达大叔见情况危急，说了一句，见浅沫没反应，强行将她拉走了。
<br/>　　另一边，天星雷鳄见着自己的‘猎物’在逃跑，心中暗恨，却不敢有任何动作。
<br/>　　它现在自身难保。
<br/>　　“崩山斩。”萧逸大喝一声。
<br/>　　天星雷鳄更加惊骇，“疯子，你竟然用武技来释放武技？”
<br/>　　噬火百刃融合已经够吓人的了。
<br/>　　现在还要用来释放玄阶中级武技的崩山斩，能量会更加不稳定。
<br/>　　这绝对是萧逸的最强一击，甚至自己都无法预估强大到什么地步。
<br/>　　果然，萧逸一剑劈出。
<br/>　　噬火百刃融合成的火刃，在堪堪打到天星雷鳄身上时，瞬间失去平衡，轰然爆炸。
<br/>　　在这股爆炸能量之下，那些暗雷摧枯拉朽般消散，天星雷鳄的身躯，也在转眼间被火焰吞噬。
<br/>　　爆炸的力量，恐怖得骇人，暴走的火焰，充斥了整个宫殿，而后往16条通道外泄，几乎席卷了整个洞府。
<br/>　　那些才刚刚来得及逃出宫殿的猎妖师，刚准备彻底离开洞府，爆炸的余威已经涌来。
<br/>　　“噗..噗..噗...噗....”
<br/>　　所有人应声吐血而出，最后被轰飞，穿过灵气屏障，直接离开了洞府。
<br/>　　幸好这只是爆炸的余威，倒没有人死，只是修为较弱的猎妖师，重伤是在所难免的了。
<br/>(本章完)
