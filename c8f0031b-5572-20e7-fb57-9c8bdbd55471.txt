等夏妍妍找到赵立英时，赵立英的脸上已经多了几个拳头印，青紫交加的，看着滑稽又狼狈。
<br/>
<br/>        这件事一定是夏夕绾找人干的！
<br/>
<br/>        赵立英迅速找到了夏夕绾，愤怒道，“夏夕绾，是不是你找人揍我的？”
<br/>
<br/>        夏夕绾一下子就看到了赵立英脸上的伤，她没忍住，“噗”一声笑了。
<br/>
<br/>        “夏夕绾，你还有脸笑，肯定是你找人揍我的，那个人揍了我后就将大罗心经拿走了。”
<br/>
<br/>        夏夕绾停止了笑声，原来大罗心经是被赵立英给偷藏起来了！
<br/>
<br/>        那揍赵立英的人是…那个喜欢睡觉的男人？
<br/>
<br/>        这答案是肯定的了。
<br/>
<br/>        夏夕绾没有想到那个表面清冷又孤傲的男人竟然三番两次的来帮她，这一次她欠了他一声“谢谢”。
<br/>
<br/>        “赵立英，你说是我找人揍你的，你有证据么，你长了一张这么欠揍的脸，我怎么知道是不是你仇家太多了所以人家找上门揍你的？”
<br/>
<br/>        赵立英再一次见识了夏夕绾的伶牙俐齿，他气急了，“你！”
<br/>
<br/>        “你什么你，有本事我们到周主任那里去理论对峙一番，正好好好聊一聊你为什么要把大罗心经藏起来了，看来留院观察都不能稳住你一颗躁动的心，如果你想早点被赶出枢密研究院那我可以成全你！”
<br/>
<br/>        赵立英差点气吐血，他完全说不过夏夕绾。
<br/>
<br/>        这时夏妍妍及时的出声道，“赵组长，这件事算了吧。”
<br/>
<br/>        毕竟是他们偷藏了大罗心经，这件事闹不起来。
<br/>
<br/>        赵立英冷哼了一声，拂袖而去。
<br/>
<br/>        夏妍妍走到了夏夕绾的面前，拿出了那份烫金的邀请函在夏夕绾的面前炫耀的晃了晃，十分的嘚瑟，“绾绾，你看这是什么？”
<br/>
<br/>        夏夕绾看了一眼烫金的邀请函，“有话就快点说吧，要不然我连炫耀的机会都不会给你。”
<br/>
<br/>        “…绾绾，圣黎院校长已经飞来了海城，过两天她办了一个小型聚餐，这是邀请函，我可以去参加哦，听说这次聚餐海城所有的医学名人都会参加，好可惜哦，绾绾你没有资格去，也没有人邀请你去。”夏妍妍漂亮的小脸上写满了骄傲和虚荣。
<br/>
<br/>        夏夕绾挑了一下柳叶眉，算是知道怎么回事了，“那恭喜你了，我也没有时间去，我晚上要早点回家陪陆先生哦，你是不会懂做陆太太的烦恼的。”
<br/>
<br/>        说完，夏夕绾直接抬脚离开了。
<br/>
<br/>        夏妍妍一下子就僵在了原地，她恨不得扑上去狠狠抓花夏夕绾那张小脸，她嘴真毒，永远能一击即中她心里的痛处！
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾回了药房，她想亲口道歉的，但是那个喜欢睡觉的男人已经不在了。
<br/>
<br/>        夏夕绾刚坐下来，她的手机就响了，来电话了，是圣黎院校长陆茵茵打来的电话。
<br/>
<br/>        夏夕绾按键接通，“喂，校长，你好。”
<br/>
<br/>        陆茵茵亲切喜爱的嗓音迅速传了过来，“喂夕绾，好久不见，你还好么，几年前你拒绝了去圣黎院留学，现在有没有后悔？”
<br/>
<br/>        夏夕绾勾起了红唇，“好像…没有。”
<br/>
<br/>        “我受到了一万点的打击，我好在也是圣黎院的院长，你还是一如既往的不给我面子。”陆茵茵笑道。
