　　第6章
<br/>　　“军师，其实我觉得一直对不住你，我看不如你去投奔韩虎踞，你知道咱们山寨情况，给他带路，以我的人头做你的进身之阶，全你们父女富贵余生，你看如何？”
<br/>　　程大雷想来想去，也想不到如何成功防守这次攻击，只有四个人，挖个坑都得两天，怎么守？唯一的办法，也许就是把徐神机派过去，发挥他算无一策的属性，毒奶一口口奶过去，保证没过几天，整座黑石城就不攻自破，说不得自己还有机会成为黑石城新的主人。到时候直接从城池开始发展，省下好多环节，到时候肯定许多世家巴不得和自己通婚。
<br/>　　这件事徐神机还真的想过，其实他昨夜做了半宿的思想斗争，然而想想韩虎踞的残暴，自己若是投奔，莫说是献策，估计刚见面被韩虎踞杀了。所以，跟着程大雷未必不是一个选择，最起码他有自信呐。
<br/>　　如果被徐神机知道程大雷的自信从何而来，估计两人得一起崩溃。
<br/>　　徐神机面色涨红，伸手指着程大雷鼻子：“你，你看错我了，老寨主待我如兄弟，你不为他报仇，我都要为老寨主报仇！我们兄弟情义，死又如何，死了才有颜见老寨主于九泉！”
<br/>　　一旁的秦蛮眼睛一亮。
<br/>　　程大雷还是很遗憾的，这是自己目前唯一能想到的办法。其他办法，是真的想不到。
<br/>　　“军师，你觉得他们能给我们五天时间么？”程大雷问。
<br/>　　“怎么可能，从黑石城到这里，至多一天半，一来一去也不过三天。他绝不会给我们五天时间。”徐神机。
<br/>　　程大雷点点头，如此看来，至少有五天时间。这五天时间，就是自己最后的准备时间。
<br/>　　『第一次山寨防守』
<br/>　　这是山贼之心给出的任务，还特别提醒难度，当然，任务难度越大，奖励也定很丰富。
<br/>　　“大当家，大当家，有人来打我们山寨！”徐灵儿急匆匆跑过来，累得满头大汗，胸前玉兔轻轻颤动。
<br/>　　“来得这么快！”程大雷一惊：“来多少人……”
<br/>　　“一个人。”徐灵儿拍着胸口，晃得程大雷眼晕。
<br/>　　“就一个？”
<br/>　　“是啊，只有一个人，一个拿枪的，模样还挺好看。”徐灵儿害羞道。
<br/>　　“走吧，去看看。”程大雷更加困惑，谁会抽疯到一个人来攻打山寨。这么想的时候，程大雷完全忘了，昨天自己怎么劫的别人。
<br/>　　四人向寨门走去，徐神机嘴里嘟嘟囔囔：看吧，看吧，我就说要早点跑路。
<br/>　　咚！
<br/>　　一只手重重搁在徐神机肩膀，徐神机回过头，看到秦蛮铁塔般杵在自己身后。
<br/>　　“你，你想干嘛？”
<br/>　　“好汉子，刚才你的话我都听见了，我知道你也是一个好汉子。”秦蛮捶捶徐神机的胸口：“以后，你就是秦某人的兄弟！”
<br/>　　“咳咳咳……应该的，大家都是好汉子。”徐神机无奈的揉着发疼的胸口，心中郁郁的想着，自己和这个智商为零的莽夫，还真的是无法勾通呐。
<br/>　　还是和程大雷互飙演技感觉好。
<br/>　　山寨大门前，一人一马，持枪而立。
<br/>　　“蛤蟆寨的狗贼，速速出来受死”
<br/>　　“苏家小姐但凡有半点闪失，我把你们刀刀斩尽，个个杀绝！”
<br/>　　等程大雷四人来到寨门，就看到这样一幕。林少羽骑在马上，用铁枪拍着拦门的木栅栏。
<br/>　　一见程大雷，林少羽更加激动：“你们四个——不，你们三个，本少侠不和女人动手。你们三个那个出来送死！”
<br/>　　本以为是黑石城的人打过来，却原来是林少羽。程大雷冲秦蛮道：“秦大哥，把他赶走。”
<br/>　　秦蛮答应一声，走出山寨。林少羽把枪背在身后：“回去取你的兵器马匹，本少侠不占你便宜。”
<br/>　　“用得着嘛？”
<br/>　　秦蛮白他一眼，忽然向前跨步，宛若一头野豹发动，猛地抓住林少羽的枪杆，顺带着把林少羽从马上抓下来，拎着他后背衣服，甩手抛出。骨碌碌……林少羽一落地，顺着山坡滚落，远远的山下传来他的呼声。
<br/>　　“我还会回来的！”
<br/>　　“待会他爬上来，也别难为他，让他把马牵走。”
<br/>　　程大雷准备返回山寨，思索着如果防守黑石城即将到来的攻击。他刚踏出一步，动作忽然一僵，原因是脑海中出现一行小字。
<br/>　　嘟，成功完成『第一次山寨防守』，奖……励……
<br/>　　程大雷猛地顿住，这不会是搞错了吧，特别提示的超难任务，就这么完成？不应该呐，这么简单的任务，系统不会给难度提示……
<br/>　　程大雷忽地一激灵，想到一种可能：系统遇到问题了，也就是说，这是……BUG ！
<br/>　　程大雷本以为『第一次防守任务』，防的是黑石城，系统的任务是根据此触发的。以两方的实力差距，肯定是超级难度的任务。
<br/>　　可是，事情出了意外，谁也没想到，林少羽二杆子精神大爆发，单枪匹马来攻打蛤蟆寨，他代替了本来任务中的黑石城兵马。
<br/>　　所以，超级任务难度一下缩水。
<br/>　　程大雷心脏砰砰直跳，难度缩水，奖励应该不会缩水吧，S级任务，E级难度，而奖励……必须还是S级。
<br/>　　奖……励……是……
<br/>　　大概系统现在也很郁闷，程大雷感觉它给出的提示这么慢，都带磕巴的。他捂住胸口，心脏砰砰直跳。
<br/>　　“快点吧，反正事情已经如此，您也干脆点，别磨叽。”
<br/>　　嘟，奖励精致铁枪一柄。
<br/>　　嘟，奖励山寨所有坐骑升级
<br/>　　嘟，奖励水井一口
<br/>　　嘟，奖励山寨技能：治愈（伤病在山寨内恢复增幅）
<br/>　　嘟，奖励山寨随机一人技能升级
<br/>　　嘟，奖励农田十亩
<br/>　　嘟，奖励木箱十口
<br/>　　程大雷两腿发酸，噗通倒在地上。整张脸呈现出一种酡红色，心脏宛若打鼓一般，砰砰跳个不停。
<br/>　　“大当家，大当家……”徐神机将程大雷扶起：“你怎么了？”
<br/>　　“爆了，爆了……”
<br/>　　程大雷呐呐自语，嘟嘟声在耳边响个不停，他从未感觉这声音如此悦耳，这都是幸福的声音呐。
<br/>　　嘟，接到任务『第二次山寨防守』，此次任务难度超过山寨目前实力。
<br/>　　这是最后一道提示信息，程大雷知道这些本是防御黑石城的奖励，但黑石城还是会攻打自己。而且，就算成功防守住黑石城的攻打，奖励也不可能像这次一样丰厚。
<br/>　　现在并不是高兴的时候，第二次攻打才是真正的挑战，才是真正的危机。
<br/>　　正这个时候，一个人影连走带爬从山坡下冒出来，手中举着一柄铁枪，身上破破烂烂。
<br/>　　“哈哈，我又回来了，没想到吧。”
<br/>　　
<br/>(本章完)
