　　周遭罡风、气流，走势已变。
<br/>　　原本向外喷发的罡风，此刻化作了吸收。
<br/>　　这代表着，金光险地，天色已亮。
<br/>　　每至入夜，金光险地便会随处飘荡着金光力量，那是金光禁地内喷发而出的金光。
<br/>　　而每到夜色褪去，金光力量便会消散无踪，那是金光力量被吸收回金光禁地之内。
<br/>　　而算算时间，从他利用八龙焚火炉吸收金光力量，再到后来激战北隐宗大长老，再到最后封炉进入这金光禁地，确实差不多过去一晚时间了。
<br/>　　他在金光禁地内，对付18把上品圣器，又耗费了些许时间。
<br/>　　现今查探一番洞府周遭，时间恰好过去了一晚。
<br/>　　金光险地，如今必是已进入白天。
<br/>　　至于萧逸此刻脸色大变的原因…
<br/>　　便是因为他终于知晓为何那一具具尸体上都带着千刀万剐的伤势痕迹。
<br/>　　这时，道道浑厚金光力量倒涌而回。
<br/>　　“不好。”萧逸脸色一变，身影一闪，瞬间退离。
<br/>　　萧逸的速度，陡然飙升到极致。
<br/>　　从这个洞口退离，回到了原本进来的那个洞口。
<br/>　　金光力量倒涌的速度极快。
<br/>　　他现今退回到的，是最开始进来的那个洞口。
<br/>　　这个洞口之上，金光力量早已被他用八龙焚火炉吸收殆尽。
<br/>　　但存在于坑洞之外，金光险地四处飘散的稀薄金光力量，正在吸力下不断而回。
<br/>　　萧逸看了眼洞口之内的那巨大洞府。
<br/>　　此时，其余17个洞口处的金光力量，正以恐怖的速度，倒涌而回。
<br/>　　洞府坐落的这片巨大范围，不过片刻便已被金光所充斥。
<br/>　　整个范围，仿佛成了一片金色的海洋。
<br/>　　巨大洞府坐落其中，光芒耀眼至极。
<br/>　　远远看去，整个洞府霎时散发着庄严、神圣而强大的气势。
<br/>　　萧逸仅仅看了一眼，便已觉眼眸刺痛，无法直视。
<br/>　　嗖…
<br/>　　下一秒，萧逸身影一闪，直接冲天而起。
<br/>　　金光禁地之下，已经不能再待了。
<br/>　　否则，一旦金光力量充斥其中，危险程度难以想象。
<br/>　　萧逸御空飞起。
<br/>　　但下方，洞口处，却以极快的速度涌出着金光。
<br/>　　这个巨大坑洞的金光力量，早已被八龙焚火炉吸收殆尽。
<br/>　　但其余17个巨大坑洞内的金光力量，可是处于充盈的状态。
<br/>　　也就是说，其余17个巨大坑洞的充盈金光力量，在穿过洞府后，此刻尽皆循着洞口，倒涌回这个巨大坑洞。
<br/>　　周遭，狂风大作。
<br/>　　那股恐怖的吸力，足以将整个金光险地的所有飘散金光吸收而回。
<br/>　　可想而知这股吸力之恐怖。
<br/>　　狂风，四处激荡。
<br/>　　原本插在山壁上的武器，瞬间被狂风吹起。
<br/>　　破损的武器，随风乱舞。
<br/>　　本来，这些破损武器已无多少威力。
<br/>　　但一接触到这些浑厚的金光力量后，霎时金光熠熠，威力惊人。
<br/>　　哪怕是破损的亚圣器，其发挥的威力、锋锐程度，亦丝毫不亚于一并中品圣器。
<br/>　　而那些破损的下品圣器，被浑厚金光力量增幅后，更是威力滔天。
<br/>　　死在这里的武者，何止千数。
<br/>　　落在这周遭山壁的武器，又有多少？
<br/>　　这般狂风吹袭下，无数武器激荡。
<br/>　　霎那间，整个坑洞之内，成了一武器狂卷的风暴。
<br/>　　嗤…
<br/>　　萧逸冲天而起的身躯，一个不慎，一把破损武器散发着金光在他身前飞跃而过。
<br/>　　胸膛上，当即鲜血淋漓。
<br/>　　“好锋利的刀身。”萧逸脸色一变。
<br/>　　那只是一把破损的刀类下品圣器，但沾染了金光后，锋利滔天。
<br/>　　萧逸的肉身，根本抵挡不住。
<br/>　　嗖…嗖…
<br/>　　萧逸的速度，不断激增，不断御空飞起。
<br/>　　不过，别忘了，现在可是狂风倒吸之时。
<br/>　　萧逸往上飞，下方等同出现了一股惊人吸力，大大减缓了他的速度。
<br/>　　同时，下方倒涌而回的浑厚金光力量，正不断跃上。
<br/>　　上空头顶，稀薄的金光力量亦正吸收而回。
<br/>　　一旦被那些金光力量追上。
<br/>　　金光力量，加上这剧烈狂风，以及插满山壁的武器。
<br/>　　届时，他将置身金光武器的风暴之中。
<br/>　　嘭…
<br/>　　萧逸瞬间祭出紫炎火翼，振翅而起。
<br/>　　“该死，好强的吸力。”
<br/>　　飞跃不过数秒，下方吸力陡然剧增。
<br/>　　萧逸上升飞起的速度，再次减弱。
<br/>　　想也知道，偌大个金光险地，何等庞大？
<br/>　　但这区区坑洞的数十米长宽大小，所产生的风力，却能将稀薄金光力量悉数吸回。
<br/>　　其风力之吸收，何等之强。
<br/>　　嘭…嘭…嘭…
<br/>　　一股股火焰，不断在紫炎火翼中爆发。
<br/>　　萧逸借着紫炎火翼的速度，以及火焰的不断爆发，强行冲天而起。
<br/>　　只是，他加快速度的同时；下方吸力，亦同样在增加。
<br/>　　这些金光力量，竟不止增加着武器的威力，连这里的狂风亦被增加了力量。
<br/>　　狂风的速度、吸力，瞬间飙升。
<br/>　　“不好。”萧逸脸色大变。
<br/>　　还未反应过来，下方金光已倒涌而上，越过了他的头顶上空。
<br/>　　吹袭的罡风中，一把把金光武器密集激荡。
<br/>　　嗤嗤嗤…
<br/>　　不过数秒，萧逸已遍体鳞伤。
<br/>　　若非黝黑鳞甲覆盖下的双手护住了要害，恐怕他现今已然身死。
<br/>　　“好可怕的金光力量。”
<br/>　　萧逸双手护在身前。
<br/>　　激荡的金光武器，打在他双手之上，除了发出锵鸣外，并未能伤他双手分毫。
<br/>　　倒是其他的身体部位，已然鲜血淋漓。
<br/>　　除了他的双手以及左脚外，其余身体部位，根本抵挡不住这些激荡的金光武器。
<br/>　　嗖…
<br/>　　恰在此时，一道白雾闪过。
<br/>　　雾妖凭空而现。
<br/>　　“主人。”
<br/>　　雾妖一把捉过萧逸，继续向上飞跃。
<br/>　　“雾妖。”萧逸脸色一喜，微微松了口气。
<br/>　　雾妖的实力，绝对不亚于武道大能。
<br/>　　加上它乃是雾体虚影，应该不惧这些金光武器。
<br/>　　不过，数秒后，萧逸再次脸色一变。
<br/>　　雾妖幻化，包裹了萧逸的身体。
<br/>　　激荡的金光武器，随着风暴四处穿梭。
<br/>　　一道流光，一闪而过，穿透了雾妖的身躯。
<br/>　　雾妖看似无损，但萧逸明显感知到，雾妖身上的力量，正在流逝。
<br/>　　这些金光力量，连雾妖的虚幻身体都能伤。
<br/>　　“撑着，先离开这坑洞再说。”萧逸咬牙说了一声。
<br/>　　“是。”雾妖笨拙地应答了一声，包裹着萧逸不断飞跃而起。
<br/>　　只是，雾妖那空洞的双眸上，此时明显闪过人性化的疼痛表情。
<br/>　　一把把金光武器，不断在它身上穿过，它身上力量，亦不断流逝。
<br/>　　呼…
<br/>　　一人一妖，在这般恐怖吸力下，大概上升到了坑洞一半高度的位置。
<br/>　　这一瞬间，下方吸力，金光倒涌速度，瞬间飙升到极致。
<br/>　　原本包裹萧逸的雾妖，瞬间在吸力之下被倒吸而下。
<br/>　　“雾妖。”萧逸脸色一变。
<br/>　　若是被吸回坑洞最底部，怕是雾妖也会在那武器风暴下被绞杀而死。
<br/>　　“该死。”萧逸脚步一瞪，上升的速度，陡然化作下降。
<br/>　　有力的手臂，一把捉住了雾妖的镰刀，“回来。”
<br/>　　下方恐怖吸力，吸摄不断。
<br/>　　即便萧逸捉住了雾妖的镰刀，但再这般下去，雾妖也支撑不了多久。
<br/>　　雾妖那如雾身体，完全被风力所吸摄。
<br/>　　“主人…走…”雾妖只懂得吐出简单的话语，表达心头想法。
<br/>　　“要走一起走。”萧逸手臂一拽，强行顺着镰刀将雾妖拉回身旁。
<br/>　　下方吸力惊人。
<br/>　　萧逸眼眸一冷，“混账，老子炸了你这烂坑洞。”
<br/>　　嘭…嘭…嘭…
<br/>　　一连六股强悍火焰，瞬间凝聚而出。
