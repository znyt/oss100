　　第338章
<br/>　　无论程大雷怎样头疼，都不得不承认一个事实。这些人加入山寨后，为自己省了很多麻烦。
<br/>　　这五十余优秀级别的生活职业，被安排到各个职位，原先莫鸣米留下的班子已经考察完毕，如今重新被启用。本来杂乱无章的事情，渐渐迈入正轨。
<br/>　　程大雷也可以稍稍闲下来，抽时间补补觉，浇浇花，和苏樱逗逗闷子什么的。
<br/>　　尤其是吴用，不愧为祸害了梁山泊的高手，虽然目前还没有升到满级，但初步已经露出些峥嵘模样。
<br/>　　唯一让程大雷有些头疼的是，毒计军师果不愧为毒计军师，出的主意又损又毒，他也不怕断子绝孙。
<br/>　　这一日，程大雷在案前处理公务，目前很多事情已经不用他亲力亲为，只有一些手下处理不了的事情，才会送到程大雷这里。
<br/>　　他看着刚送来的东西，下意识皱起了眉头。这已经是今天送过来的第三份了，内容几乎一样。
<br/>　　程大雷占领琴川关后，原先很多莫鸣米的手下就带着人逃跑了，有些人下落不明，有些人沦落成匪，在琴川关一带四处抢劫。
<br/>　　“吴军师，这件事你怎么看？”程大雷仍有考量吴用的意思。
<br/>　　谁知道这智多星，是不是真正的智多星。
<br/>　　“依属下看，这件事倒也好办。”
<br/>　　“好啦，你说？”程大雷。
<br/>　　“琴川一地，大概有十八家势力，以前都被莫鸣米控制。如今大当家取代莫鸣米，可这些人到现在也没有表明态度。依我的主意，可以把他们请到琴川关……”
<br/>　　“拉拢人心？”程大雷眼前一亮。
<br/>　　“不。”吴用做了个割喉的姿势：“一网打尽！”
<br/>　　程大雷倒抽一口凉气，竟然恐怖如斯。
<br/>　　派出去人去给这些人送信，约定日子把他们都聚到琴川关，程大雷决定亲自给他们上上课。
<br/>　　这件事，派云中龙三人去做，毕竟云中龙和他们都认识，互相说话也方便。
<br/>　　同时，程大雷派兵出关，开始追杀莫鸣米的残党。这正好也是个练兵的机会，被俘获的符箓，也锻炼一下他们的默契。
<br/>　　这次行动，有张肥和高飞豹轮流领队，二人都是豪横性格，每次带五百人出关，将琴川一带搞得风声鹤唳。
<br/>　　每天都能带很多人头回来，同时，这也是给琴川关的俘虏看看，让他们明白，与程大雷作对是怎样下场。与之相反，好好为程大雷做事，则是好吃好喝好招待。
<br/>　　莫鸣米的残党，有些是带着人投奔别人去了。有些则是留在琴川，平时抢枪附近的村庄什么的。最近这些日子，他们被高飞豹和张肥折磨得欲仙欲死。
<br/>　　稍不留神，就被发现，然后大军杀过来，手段残忍至极。这次投降也没用，全部被杀得干干净净。
<br/>　　只花了七八天时间，张肥和高飞豹带着人马，已经杀得他们在琴川关没有容身之地，这些人或者是被杀了，或者是被赶到别的地方。
<br/>　　同时，十八家头领也在赶来琴川关的路上。这些几乎是同时得到的消息，说程大雷要他们到琴川关一聚。
<br/>　　他们不敢来，但也不敢不来。不敢来的原因毕竟是和程大雷有过节，怕程大雷搞这一处是为了一网打尽。可不来的话，又害怕遭了程大雷的毒手。
<br/>　　还是云中龙一再保证，说程当家心怀仁义，和莫鸣米是不一样的。
<br/>　　心怀仁义？众人心想云中龙是不是对这四个字有什么误解。程大雷没到琴川关之前，大家已经听说过他的残暴之名。如今他逼死莫鸣米一家老小，又用事实证明了传言。
<br/>　　仁义，不知是这个字眼侮辱了程大雷，还是程大雷侮辱了这个字眼。
<br/>　　最后，大家也不敢不来，派二把手来的念头也不敢有。到正日子的时候，所有人都揣着忐忑的心情，踏上了前往琴川关的路。
<br/>　　“龙爷，这次真不会出什么事吧？”有人问。
<br/>　　原先云中龙在十八家首领中不算头面人物，后来跟着程大雷混的时候，更是站在了所有人的对立面。但如今形势发生改变，云中龙的身份也是水涨船高，完全成了程大雷的代言人。
<br/>　　“肯定没问题啦。”云中龙骑在马上，趾高气扬道：“我和程当家什么关系，他什么都听我的，有我在，绝对没有问题。”
<br/>　　“那就全靠龙爷了，到时候可要为我美言几句。”
<br/>　　“放心啦，包在我身上。”
<br/>　　众人同时抵达琴川关，穿过城门的时候，刚平静下来的心脏立刻又紧张起来了。
<br/>　　众人以前不是没有来过琴川关，这次再来，感觉气氛和以前有些不同。
<br/>　　曾经的琴川关就是一座军事堡垒，除了莫鸣米外，所有人都活在恐惧中。但今天的琴川关比较活泛些，虽然每个人都在忙碌，可却偶尔有笑声响起。
<br/>　　看来程大雷和莫鸣米果然是不同的。
<br/>　　大家由刘悲引着来到将军府，在左右椅子上坐了，帅案后面空着，程大雷尚未出现。
<br/>　　赵子龙也退了出去，大厅内只剩下他们，这些人的心情有紧张起来。
<br/>　　安静了一会儿，便响起议论声。
<br/>　　“龙爷，程当家怎么还不来？”
<br/>　　“程大雷很忙的，可能还要等会儿吧。”
<br/>　　周围很安静，好像随时会冲出一波伏兵，将大家一个个都杀死。
<br/>　　“龙爷，真不会出什么问题吧，我们可都是听你的话才来的。”
<br/>　　“安啦，肯定没问题，我不是和你们在一起么。”其实云中龙心里也有些含糊，不过他至少可以确认一件事：自己是绝对不会有问题。
<br/>　　他、乔鹤、游九楼都是很早就跟着程大雷一起混的，就算程大雷把所有人都杀了，他们也不会有事。
<br/>　　这时候，忽听得外面脚步声响，程大雷穿着一件宽大袍子，身后还跟着一个捧剑背斧的银发姑娘。
<br/>　　随着他走进大厅，众人心脏砰砰直跳，几乎同时站起身。
<br/>　　“程当家。”
<br/>　　程大雷出现在帅案后，随意摆摆手：“坐，坐，大家都这么紧张干什么，我刚才有点事耽搁了，劳大家久等，坐，坐。”
<br/>　　
<br/>(本章完)
