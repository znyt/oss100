　　“本来说好了是配合武警特战部队进行行动，但是实际情况总是会跟计划相悖。当时我们的位置距离边境线不到五百米，那些武装分子要从我们的伏击圈中逃脱，只需要一个冲刺就能冲出边境线。”
<br/>　　慕容飞回忆着自己第一次在执行任务时，开枪射杀那个试图逃出边境线的武装分子的过程，“按照规定，只要对方冲出了边境线，就不允许再开枪射击，而这次行动……”
<br/>　　慕容飞一低头，看到佟千雪已经靠着自己睡着过去，慕容飞轻轻摇头，横抱起佟千雪往卧室的方向走去。
<br/>　　“真是只猪！”慕容飞对着佟千雪皱了下鼻子，嘀咕道。
<br/>　　慕容飞话音刚落，腰间软肉就被佟千雪用手揪住，用力一旋，佟千雪没有睁眼，轻哼了声，说道：“敢骂我，晚上给我做顿面条来贿赂我，不然你就惨了。”
<br/>　　“你没睡着啊？”慕容飞问道。
<br/>　　“睡着了，可是又醒了。”佟千雪回答道：“谁让你的胸口这么硬的，就跟靠在石头上一样，一点都不舒服。”
<br/>　　慕容飞嘴角狠狠抽了抽，这是把自己当成靠枕了啊。
<br/>　　陪着佟千雪午睡了一会儿，慕容飞去厨房给佟千雪做面条去了。厨房里的食材很全，足够两个人吃两天的，晚饭过后，佟千雪又是一阵犯困，可还是被慕容飞拉着出去散步了。
<br/>　　门卫处的保安换了个人，看大佟千雪和慕容飞赶紧出来向两个人打招呼，为上午的误会替那个新来的保安道歉。
<br/>　　回到家的时候，佟千雪直接躺倒在沙发中，用她自己的话说，她感觉自己的手跟脚都不是她的了，连去洗澡的力气都没有了；可随后，慕容飞一句“我帮你洗”，让佟千雪立刻满血原地复活，从客厅沙发中逃走了。
<br/>　　“哼，别以为我不知道你在想什么，想占我便宜，哪有这么容易！”
<br/>　　佟千雪在心中把慕容飞鄙视了一番，拿着换洗衣服进入了主卧里的浴室。
<br/>　　“秋老虎”让这几天的气温一直维持在三十多度，晚上睡觉也要开着空调，才不至于睡的一身黏糊糊的汗。
<br/>　　这一晚上，对于慕容飞来说实在是无比的折磨，自己身边，佟千雪就穿着一条睡裙躺在自己身边，一阵阵沐浴露的香味从佟千雪身上传来，飘入慕容飞鼻子中，就像有人拿着一根羽毛在他心尖不断撩拨着他，让他一阵心痒痒。
<br/>　　下午跟佟千雪在沙发中打闹的时候，忍忍也就过了，现在佳人衣着单薄的就躺在身边，而自己却偏偏能看不能动，这特么的也太折磨人了！
<br/>　　背对着自己的佟千雪突然转过身来，睁开眼睛看着正在装睡的慕容飞，吐气如兰的说道：“很辛苦？要不要我帮你？”
<br/>　　慕容飞闻言，立刻睁开了眼睛，黑暗中，慕容飞的眼睛就跟自带发光源一样，闪亮闪亮的。
<br/>　　看到慕容飞闪烁着眼睛，佟千雪哪里还能不知道这家伙在想什么，直接一盆冷水对着他浇了下去，说道：“那件事情你想都不要想，那个味道那么腥，别指望我再帮你。”
<br/>　　“那就算了。”慕容飞赶紧闭上眼睛，还往后面退了退，说道：“你别说话，也别靠近我就是帮我最大的忙了。”
<br/>　　佟千雪嗤嗤一笑，不再去逗慕容飞也是闭上了眼睛。佟千雪因为怀孕的关系，睡眠要比慕容飞好很多，很快就睡着过去，而我们的慕容飞同学毫无疑问的失眠了，他自己都不知道自己是什么时候睡着的，迷迷糊糊睡着过去后，慕容飞做了一个旖旎的美梦……
<br/>　　等到第二天早上醒来的时候，慕容飞意识到了不妙，掀开自己身上的空调被看了眼，慕容飞差点就哭出来了。妈的，自己居然梦遗了！作为一个有老婆的人，居然做了春-梦，而且还梦-遗了！虽然梦中对象是佟千雪，可还是让慕容飞感觉自己丢脸丢到了外太空。
<br/>　　佟千雪不在床上，浴室中有着水声传来，想来佟千雪这个时候应该是在浴室里洗漱，慕容飞打算趁着佟千雪洗漱这段时间赶紧起床，换掉自己身上的内裤，销毁“罪证”。
<br/>　　可是设想是美好的，现实是残酷的。
<br/>　　就在慕容飞跳下床，跑到衣帽间中去找内裤的时候，浴室中突然传来马桶抽水的声音，吓得慕容飞赶紧又逃回到床上，用空调被盖住了自己。
<br/>　　佟千雪走出来看着用空调被把自己裹得跟蚕宝宝一样的慕容飞，疑惑道：“怎么把自己裹得跟个肉包子一样，生病了吗？”
<br/>　　说着走到床边，伸手在慕容飞额头上摸了一下，说道：“也没有发烧啊。”
<br/>　　“给我倒杯水。”慕容飞紧紧裹着空调被，对佟千雪说道。
<br/>　　“喂，你让一个孕妇给你倒水，你也真好意思说出口！”佟千雪插着腰，瞪着慕容飞看了一会儿，随后叹了口气，说道：“等着！”
<br/>　　看着佟千雪转身离开卧室去客厅给自己倒水，慕容飞赶紧从床上跳起来，跑进了衣帽间，把门关上洗澡换衣去了。
<br/>　　佟千雪端着杯子走进卧室看着空荡荡的床，有些纳闷不知道慕容飞这是闹得哪一出，当看到掀开的空调被上的痕迹，立刻就反应过来。看着紧闭的衣帽间，好气又好笑，摸着自己小腹，说道：“宝贝，你爸现在肯定特后悔把你怎么早就弄出来。”
<br/>　　慕容飞想尽办法要隐瞒，佟千雪自然是不会戳穿他，等到他从浴室出来的时候，抱着床上的空调被朝着卧室外走去，说道：“去拿条洗干净的空调被出来，这条空调被睡了两个礼拜了，我去洗一下。”
<br/>　　……
