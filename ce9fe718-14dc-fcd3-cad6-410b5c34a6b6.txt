　　见她一直把玩着纳戒，慕清霄搂住她的细腰，嗅着体香，耳鬓厮磨起来。
<br/>　　“等你修行到了元婴，开发出神识，在纳戒上刻上灵魂印记，除非对手灵魂力量高出你一大截，否则，是不可能打开的。”
<br/>　　欧阳紫嫣认真的听着，道：“那刚才的滴血......”
<br/>　　“滴血是最普通的一种保护措施，实力比你强的，都能强行破开，但也不用担心，地球上能比你强的，暂时没出生呢。”
<br/>　　微微摇头，慕清霄平淡的说道。
<br/>　　在地球这种，灵气如此稀薄的地方，哪怕资质在高，修炼到练气巅峰，就算是逆天了。
<br/>　　要知道，欧阳紫嫣现在可是金丹七重，除了王语嫣她们，真没有对手，所以刚才所说的事情，是绝对不可能发生的。
<br/>　　“嗯，我知道了。”
<br/>　　说着，欧阳紫嫣抬起臻首，在他脸上轻轻的吻了一下，旋即，逃也似的离开他的怀抱。
<br/>　　摸了摸有些湿润的脸颊，慕清霄道：“紫嫣，让雅儿和珠儿上来一趟吧。”
<br/>　　“好，我这就给她们部门打电话。”
<br/>　　经过几年的相处，欧阳紫嫣这种心细的女人，怎么可能猜不中他的想法，于是就拿起座机，拨通了电话。
<br/>　　挂断电话，欧阳紫嫣走到一旁，拿了一杯茶水递了过来。
<br/>　　接过茶杯，慕清霄道：“对了，语嫣，小依和雪琪她们，最近怎么样？”
<br/>　　“依旧是老样子，不过，我们互相切磋，实力倒是提高了一些，有语嫣在，我们进步也很快，那丫头的天赋，你是知道的。”
<br/>　　说到语嫣的天赋，欧阳紫嫣美眸中都闪过一丝羡慕。
<br/>　　慕清霄抿了口茶水，语嫣天赋本就惊人，过目不忘，推演功法。
<br/>　　最后，经过两人双修，龙元的改造，资质绝对是她们姐妹中，最顶尖的存在，哪怕天赋异凛的小依，都有所不如。
<br/>　　“带回来的东西有不少，肯定能让你们的实力提高许多......”
<br/>　　就在这时，办公室房门轻轻敲响，大门被推开，两道倩影就走了进来，看到慕清霄后，美眸中皆是闪过温柔。
<br/>　　慕清霄也是一视同仁，将她们搂在怀中亲密一番，倒也没做出荒唐的事情，随后取出两枚纳戒交给她们。
<br/>　　拿到纳戒，雅儿与珠儿都为其神奇所惊叹，纷纷奉上香吻，最后，因为工作原因，她们只能恋恋不舍的离去。
<br/>　　“对了，最近集团的发展如何？”
<br/>　　“蒸蒸日上，集团开发出来的东西，能够设计各大领域，最近底下的管理人员，也成立了一座影视城......”
<br/>　　慕清霄颇为满意的点头，进军各项领域，若是辉煌集团在各项领域中都取得成绩，距离商业帝国就不远了。
<br/>　　他现在虽然不缺钱，但商业帝国，终究是他弱小时的一个梦想，圆梦之后，心境才能圆满。
<br/>　　按照辉煌集团的发展速度，相信很快就能将手伸向国外......
<br/>　　茶水喝完，慕清霄放下茶杯，站起身来，走到落地窗前，负手而立，几乎将大半个静海市尽收眼底。
<br/>　　望着他的背影，欧阳紫嫣美眸中露出痴迷之色。
<br/>　　当初，就是身前的青年，给了她任何人都不敢的承诺，带给她幸福快乐，让她接触到普通人，永远都接触不到的东西。
<br/>　　想到此处，不禁有些发呆起来。
<br/>　　转过身来，慕清霄见欧阳紫嫣脸颊有些微红，道：“我先回趟家，阿碧和雪琪估计在家呢，到时候去接语嫣她们。”
<br/>　　“我让人送你回去。”
<br/>　　走上前来，替他整理了下衣领，欧阳紫嫣一脸温柔，气质与平时高高在上的女总裁形象，完全不符。
<br/>　　“也好，工作完早点回来。”说着，慕清霄就转身离去。
<br/>　　几分钟后，来到金帝别苑门口，已经有一辆白色的宝马五系停在了门口，车门前站着一名身穿职业装的女子。
<br/>　　看到女子的瞬间，慕清霄微微一愣，有点印象，若是没记错的话，面前秀丽的女子，好像叫何白萍。
<br/>　　当初，他来集团视察的时候，有过一面之缘。
<br/>　　听说此女工作能力不错，凭自己的努力，已经从小组长，荣升为了财务部副总监。
<br/>　　年薪百万，在辉煌集团，虽然算不得最高，但对普通人而言，已经是一个非常恐怖的数字。
<br/>　　不过，有朱儿压着，副总监已经是她的人生巅峰了吧。
<br/>　　见慕清霄走过来，何白萍察觉到他身上有股莫名的气质，不禁楞了一下，微微弯腰，道：“董事长，欧阳总裁已经吩咐过，让我送您回去。”
<br/>　　慕清霄微微点头，只是心里却有些腻歪，紫嫣这是要干嘛，考验我吗？
<br/>　　就算你要考验我，也必须找个非常漂亮的吧，何白萍虽然长得秀丽，但也只能算一般般。
<br/>　　这种女子，满大街都能看到，对于娇妻成群的他来说，根本没有半点吸引力。
<br/>　　当然，他也就随便想想而已，自己的女人怎么可能做这种无聊的事情，估计也有碰巧吧。
<br/>　　打开车门，慕清霄就坐在后面座位上，望着专心驾驶的何白萍，道：“集团，工作怎么样？”
<br/>　　“集团的工作非常的好，多亏了总监的照顾。”说到辉煌集团，何白萍眼中满是荣幸，为自己当初的选择而庆幸。
<br/>　　“你的努力，我都听说了，好好工作。”慕清霄微微一笑，随后就闭目养神起来。
<br/>　　听的慕清霄的夸奖，何白萍眼中满是惊喜，看来她的努力，确实被高层看在眼中，于是受宠若惊。
<br/>　　“多谢董事长。”
<br/>　　半个时辰后，白色宝马五系停在了金帝别苑外，慕清霄张开双眼，打开车门走了下去。
<br/>　　“行了，你去忙吧。”说罢，便迈步向金帝别苑中走去，走上熟悉的道路上，他的心情极为平静。

<br/>　　移动门开启，前脚刚踏进别墅院落中，一股清新的空气就包裹了过来。
<br/>　　望着院落两旁的绿竹，周围的花草，其中种植着几株灵药，比外界空气清新也是自然。
