　　第500章
<br/>　　攻势若疾疾之风，咄咄若催命之鬼，手中一柄断刃，便若风中之雨。
<br/>　　风雨扑面而来。
<br/>　　程大雷从未想到，一个人功夫能强到这种程度。他一直都没敢小看聂隐娘，真正交手才明白，自己还是低估了她。
<br/>　　也是被逼得没有办法，程大雷才会刺出阿飞快剑，这一招直来直往，例不虚发。但今天面对聂隐娘，这一剑并没有起到任何作用。
<br/>　　因为聂隐娘比他更快。
<br/>　　手中的匕首中平刺，这一招平平无奇，但因为快到极致，所以有锐不可当的威力。
<br/>　　匕首刺中了程大雷的喉咙。
<br/>　　空气像突然静下来，这时候刘悲才能看清程大雷和聂隐娘的身形，只见聂隐娘一柄匕首陷在程大雷喉咙。
<br/>　　那一瞬间，刘悲的第一个念头就是：大当家被杀死了。
<br/>　　程大雷在这一瞬间，也发出了无数先辈发出的感慨：吾头尚在否！
<br/>　　他的脑袋当然还在身上，这一招并没能伤到他。关键时刻，照月蛤蟆甲抵挡致命一击的属性生效，护颈帮程大雷挡住了匕首。
<br/>　　聂隐娘怔了怔，自己手中的匕首乃是世间罕见的神兵利器，凡铁根本无法防御，可这一次却被程大雷身上的盔甲挡住了。
<br/>　　程大雷可没有发呆，毕竟性命就在一瞬之间，他立刻施展身无彩凤双飞翼，轻身功夫施展到自己能做到的极致，迅速与聂隐娘拉开距离。
<br/>　　这时候除了转身就逃，根本没有其他选择。
<br/>　　刘悲也反应过来，道：“大当家，向城中逃，她伤不了你。”
<br/>　　这的确也是程大雷第一个念头，但程大雷稍想了想，便没有这做。逃回蛤蟆城，的确有很大概率能得一条活命，但是活下来又怎样，难道一直躲在地牢中不出来么。要知道，躲得了一时，可躲不了一世，聂隐娘这个麻烦总是要解决掉的。
<br/>　　况且，程大雷今天告别李婉儿，心情可以说是极其不开心。人在心情糟糕的情况下，总是会铤而走险，做出些冲动冒险的决定。
<br/>　　程大雷决定今天要冒险除掉聂隐娘，一劳永逸。
<br/>　　所以他没有向城中逃，而是转身向八百里琴川平原狂奔。
<br/>　　“不用找我，本大家去去就回。”程大雷一边大喊一边脚下不停。
<br/>　　聂隐娘自然紧紧跟在他身后，一步都不肯落。其实，她这些日子过得也并不轻松，窝在草原上，饥一顿饱一顿，实在是没想到程大雷竟如此胆小，今天他好不容易出城，被自己找到机会，聂隐娘怎么肯放过他。
<br/>　　草原上两道狂奔的身影，一前一后，相距大概五十步左右。程大雷再前，脚尖一点，便是七八丈的距离，地上的大树、山坡、怪石被他一步跨过。跟在他身后的聂隐娘堪称草上飞，脚尖点地，如同疾风从草原上行过，紧紧跟在程大雷身后。
<br/>　　程大雷这样做，并不是没有半点考虑没有。这聂隐娘的确很强，强得有些变态，真与她动手，程大雷没有一丁点胜算。但人力有时而穷，她的剑法强，身法快，没有理由她的耐力同样变态，她毕竟是个女人。
<br/>　　只要自己把战场拉远，引着她长途奔袭，一点点将她的体力耗尽，到那时候，她还不案板上的羔羊，剥光的美女，人自己宰割。
<br/>　　说白了，今年不比功夫，比长跑。
<br/>　　程大雷和聂隐娘一前一后，一溜烟便没了踪影。他顿了顿，立刻醒悟过来，以最快的速度回到蛤蟆城，将山寨的主要人物召集到一起。
<br/>　　徐神机、关鱼、秦蛮一听是如此事情，那还了得，人人三尸神暴跳，五灵豪气腾空，要倾巢而动，去杀掉那女人，保护程大雷。
<br/>　　最后还是被刘悲拦下了，关键是这不是一场群殴战，你派步兵出动，派出去的人再多，连他们的影子都看不到，又有什么用。最后，还是赵子龙的龙字骑出动，秦蛮等人也带上自己的亲卫，骑着快马去寻程大雷的踪迹。
<br/>　　程大雷临行前是嘱咐刘悲不用找，但这种情况下怎么可能不找，山寨能动用的力量都用上了。
<br/>　　大家分散开了，沿着程大雷逃走的方向寻找，心里总觉得，程大雷再能跑，两条腿也跑不过四条腿。
<br/>　　可结果出人预料，一连追出五十里，都没看到程大雷的影子，不过，却有其他发现。
<br/>　　“军师，军师……”一名小卒道：“这好像是大当家的盔甲。”
<br/>　　刘悲赶紧骑马过来，果然见地上散落着程大雷的照月蛤蟆甲，这里丢着肩铠，那里丢着护膝。
<br/>　　“莫非大当家被刺客分尸了。”小卒胆战心惊道。
<br/>　　“休要胡说八道，这明显是大当家自己脱下来的。”刘悲向前看了一眼：“看来我们的方向没错，继续追。”
<br/>　　小卒不敢说话，忙将程大雷的盔甲收起来，心中还想：大当家没事脱盔甲做什么。
<br/>　　这一个月时间，程大雷衣不卸甲，虽然很是惹得别人嘲笑，但在关键时刻的确救了他一命。而现在，是比赛长袍，身上的盔甲自然用不到了，在逃跑的过程中，他将沉重的盔甲卸了下来。
<br/>　　如此速度陡然又获得提升，竟又与聂隐娘稍微拉开了一些距离。
<br/>　　为了提升速度，程大雷什么都可以豁出去，他差点将匹夫剑也抛了。但想了想，真要手无寸铁，要是被聂隐娘追上，也就只能伸着脖子被人砍了。
<br/>　　其实，让他把身上衣服脱光，在草原上果奔，程大雷也同样不在乎。如果聂隐娘稍微有些羞耻心，说不得这样做还能对她造成一些精神伤害。
<br/>　　但看聂隐娘的样子，程大雷估计有些够呛，所以才没这么做。
<br/>　　现在的他，他提着匹夫剑，只穿着贴身小衣，在草原上狂奔。
<br/>　　刚开始，二人的速度都极快，但跑出五十里后，二人的速度就都慢了下来。这也是没有办法的事情，毕竟谁也不是机器做的，程大雷跑得气喘吁吁，而聂隐娘也没好到哪里去，她两腮发红，汗水一层又一层的冲刷着肌肤。
<br/>　　但要杀程大雷的决心，一丁点也没有减弱。
<br/>(本章完)
