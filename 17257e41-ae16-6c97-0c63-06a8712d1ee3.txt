        就在傅夜沉感到狐疑的时候，童筱颖讷讷地开口，弱弱地问道：“昼景他……是不是做了什么‘犯规’的事情？”
<br/>
<br/>        “没有。”傅夜沉故作淡定地笑了笑。
<br/>
<br/>        童筱颖抿了抿唇，声音哽咽了起来：“我也算是学建筑的，所以，我大概能听懂大哥你刚刚通话里的内容……”
<br/>
<br/>        “筱颖你不要多想。”傅夜沉试图安抚童筱颖的心情。
<br/>
<br/>        童筱颖眸光黯然地仰起头来，诚恳地看着傅夜沉的眼睛，几近哀求道：“大哥，如果，昼景真的做了什么不好的事情，大哥，你一定要保他，好吗？”
<br/>
<br/>        傅夜沉点头，深沉地回答道：“就算你不说，我也会想办法保全阿景。”
<br/>
<br/>        话到这里，傅夜沉微微垂眸，不打算再对童筱颖隐瞒。
<br/>
<br/>        “只不过，我的能力终究是有限的。因为，这件事情闹得很大。上头派人下来查的话，没有爷爷出面，终究难保阿景平安无事。”傅夜沉若有所思地说。
<br/>
<br/>        他向童筱颖讲述的只不过是事实，他在军警界是有“势力”，但是政界里的熟人，还得靠爷爷去笼络。
<br/>
<br/>        郑忠威已经将爷爷傅炎廷的意思转达给他了，爷爷坚决要把阿景推出的话，那就代表爷爷他压根就不想救阿景。
<br/>
<br/>        又或者说，爷爷是在惩罚阿景，阿景自己犯下的错，得阿景自己去承担。
<br/>
<br/>        傅夜沉明白爷爷傅炎廷的意思，一旦爷爷自己做下的决心，就很难有人再去改变他的心意。
<br/>
<br/>        童筱颖娇俏的脸上，突然露出了坚定的表情：“大哥，带我去见爷爷好吗？我有话想跟爷爷说。”
<br/>
<br/>        “你想去求爷爷？”傅夜沉一语道破了童筱颖的心思。
<br/>
<br/>        童筱颖也不打算藏着掩着，非常坦白地向傅夜沉点了点头。
<br/>
<br/>        其实，傅夜沉的心里也没底，但还是答应了童筱颖的要求。
<br/>
<br/>        他明明知道筱颖去了也是白去，但是他也知道，他若不答应筱颖的要求，筱颖一定不会就此甘心。
<br/>
<br/>        童筱颖甩开了自己那被季运恒“侮辱”的糟糕心情，比起自己的“不幸”，她更紧张傅昼景。
<br/>
<br/>        她回宿舍换了衣服，上了傅夜沉的车，跟傅夜沉一起回了傅家大庄园。
<br/>
<br/>        童筱颖回来得正是时候，厨房里正在给傅老太爷备用宵夜。
<br/>
<br/>        为了能讨傅老太爷傅炎廷的欢心，童筱颖亲自给傅老太爷下厨。
<br/>
<br/>        当童筱颖端着自己做的清淡又护胃的养生粥，来到傅老太爷的书房里时，傅老太爷一眼就看出了童筱颖此次的来意。
<br/>
<br/>        上次傅老太爷傅炎廷让连华生试探童筱颖的事情，童筱颖的刚烈不阿，倒是让他这老头子对她童筱颖刮目相看。
<br/>
<br/>        在那种情况下，哪怕是牺牲她自己的性命，她也能做到紧牙关地保密，证明她确实是一个信守承诺的女孩子。
<br/>
<br/>        相对于比起来，苏漫雪的行为，就不太令他这位傅老太爷满意了。虽然，苏漫雪最后也没把秘密说出来，但是她的所作所为，确实过火了点。
<br/>
<br/>        傅老太爷看得出，苏漫雪十分忌惮童筱颖。
<br/>
<br/>        也正因为这次试探，才让傅老太爷对苏漫雪起了疑心。
<br/>
<br/>        苏漫雪和童筱颖既然没有血缘关系，为什么会长得一模一样？
<br/>
<br/>        苏漫雪又为何想要至童筱颖于死地？
<br/>
<br/>        就算童筱颖联合傅昼景想对傅夜沉不利，那她苏漫雪想要铲除的人难道不应该是傅昼景吗？
<br/>
<br/>        种种疑惑，让他这位老头子下令彻查了苏漫雪和童筱颖之间的所有事情。
<br/>
<br/>        虽然，长孙傅夜沉对苏漫雪和童筱颖之间的事情闭口不谈，但是，他这位老人家可是位“老姜”，眼线众多，所以，没有他不知道的事情。
<br/>
<br/>        长孙傅夜沉出任务身受重伤那晚，其实是被童筱颖所救，留下的傅家祖传玉坠，被苏漫雪一时的贪念，顶替了童筱颖的身份，成为了傅家的大少奶奶。
<br/>
<br/>        苏漫雪应该是怕自己会露馅，才整容成童筱颖的样子，想要以此来糊弄他的长孙。
<br/>
<br/>        若不是当时傅夜沉一直有任务在身，从未怀疑过刘管家是否真的接对了人，否则，他又岂能容她苏漫雪胡作非为这么久！
<br/>
<br/>        苏漫雪可以哄得住傅夜沉的母亲，但是哄不住他傅炎廷这只久经“商战”，阅人无数的“老狐狸”。
<br/>
<br/>        至于童筱颖，她品性纯良，傅炎廷这点是完全可以肯定。
<br/>
<br/>        傅炎廷既然能体会自己长孙傅夜沉为爱的隐忍，当然也能明白童筱颖对傅昼景的一片真心。
<br/>
<br/>        童筱颖手里还端着粥碗，还未放到傅炎廷的书桌上，傅炎廷便冷漠地开口说道：“你做的东西，我是不会吃的。”
