　　第517章
<br/>　　白衣尼姑一手托着拂尘，一手托着陶罐，一瘸一拐的下山。
<br/>　　虽然她修养了七日，但身上的伤势并未完全好转，如今还留着隐疾。
<br/>　　孟玄青立在茅屋前，望着她下山的背影，长长的叹了口气。
<br/>　　终究也只是个可怜人而已，执着旧梦，不肯醒来。现在已由佛入魔，天下间还有谁记得她姓甚名谁。
<br/>　　为了杀一人，便是屠城绝户也在所不惜。可便是杀了程大雷，又有什么意义，世上怕已没人记得那百年前的一十九国。
<br/>　　西蜀一脉，怕活着的也就是自己与她二人而已，至于其他十八国，估计情况也好不到哪里去。
<br/>　　想到这里，他忽然心中一悲，宛若什么东西噎在喉咙，吐，吐不出来，吞，吞不下去。天地之大，如今记得自己名字的，怕也就只有她了。若她也死了，自己可真就随意生，随意死，无人在意了。
<br/>　　这个念头一起，心里便放不下去。她这一去，该不会出现什么意外吧。以她的本事，又是悄悄下毒，照道理来说不应有差错。可那程大雷听他所说，便不是好对付的人物，第一次可以伤到她，第二次为何不能杀了她。
<br/>　　心中突然有些放心不下，这世上有个记得自己的人终究不是坏人。不若，自己悄悄跟在她身后，也不露面，若她遇到危险，自己也可暗中相助。当然，如果她没事，能平安下毒屠城，自己再悄悄回到大雪峰上。
<br/>　　难道真的要下山么？
<br/>　　他回身看着立在雪峰上的茅屋，轻轻叹了口气，自己都不记得，上次下山是什么时候了。如今下山的念头一起，他整个人似乎变得年轻起来，迈步钻入茅屋。
<br/>　　屋内的摆设潦草且杂乱，任谁也想不到，当年他吃道菜，便是前后八道工序，等菜热气腾腾端到面前，多一刻或者少一刻他都不会动筷子。
<br/>　　如今，这样狼藉的茅屋兽皮便能安身了。当然，若没有寒暑不侵的功夫，他也不可能在这大雪峰上活下来。
<br/>　　掀开兽皮毯子，床铺下出现一包阵囊，毒针三十六，本就是他的看家本领。不过经久不用，铁针已经生锈。
<br/>　　他将针囊拴在腰上，提了一根木杖，踏出了茅屋。寒风扑面而来，他顶着风前行，一路尾随着白衣尼姑的脚印，等下得雪山，就能看到远处若隐若现一个白影。
<br/>　　若论功夫，他是不如这白衣老尼的，人不疯魔不成活，白衣老尼一意孤行，一身本事早已到惊世骇俗的地步。
<br/>　　所以他不敢跟得太近，只是远远在后面缀着，一路随着白衣尼姑。
<br/>　　白衣尼姑倒是真没想到，孟玄青会跟着自己下山，以她的判断，孟玄青宁肯老死在雪山上，也不会下山一步。
<br/>　　所以她心里也没有提防，并不知孟玄青就缀在身后。
<br/>　　在程大雷面前，狠狠踹了铁板，一口气现在还没有理顺。这次得到这屠城剧毒，她如虎插翼，迫不及待要去找程大雷报仇。
<br/>　　当然，孟玄青说这蛤蟆毒可以屠一城灭百里，可能也就是随口说说而已。当真不能以为把这蛤蟆随手丢进河里，蛤蟆城也就灭了。
<br/>　　为了保险起见，她决定将这蛤蟆丢进城主府的水井中，毒随水走，只有喝过井中水的人，便有性命之忧。至于这蛙毒会随水走多远，那便不是她所关心的了。
<br/>　　下山已经是晚上，她一路疾行，行到蛤蟆城的城门前，天还未亮。如今正是一天中，人最疲惫的时候，守城之人也打着哈欠，等着天亮换班。
<br/>　　她隐在城下，脚尖一点地，便是一跃数丈，再往城墙上借力，就又是数丈的距离。这一手轻功，则又比聂隐娘高出一个层次。
<br/>　　但聂隐娘可以悄无声息的入城，她却做不到，城门一队卫兵发现了她。立刻举起武器，持铜锣的准备用铜锣传讯。
<br/>　　一道白衣人影陡然加速，宛若一阵风，风卷过，五名卫兵瞬间毙命。他们或被拂尘勒死，或被重拳击碎肺腑，临死前没来得及把消息传出去。
<br/>　　她身子飘摇入城，心中也知道，城墙上的尸体，定然很快会被发现。自己必须赶这时间，将这毒蛙投入城主府的水井中。
<br/>　　于是接下来，她加快了脚步，稍微辨认一下方向，便奔向城主府。
<br/>　　在其刚刚下城不久，就有一道黑影攀上了城墙，看了看地上的尸体，微微撇撇嘴，也同样潜入了城。
<br/>　　片刻之后，白衣尼姑已来到城主府，立在大殿之顶，扫视着城主府的布局。偌大一个城主府，当然不可能只有一口水井，厨房、单独的院落，都挖着井，不过，地下的水脉都是相通的，她随意择了一口，便从房顶上跃了下来。
<br/>　　夜里的城主府静悄悄，一个人影也无。程大雷现在还在呼呼大睡，面对这白衣老尼，程大雷可没有对付聂隐娘的精气神。
<br/>　　白衣老尼来到井口前，四周看了看，这里当是府中的马圈，半夜不会有人接近这里。只要将井上的石板挪开，明天早上提水做饭，程大雷也就必死无疑了。
<br/>　　这样的时辰，的确没人会靠近后院的马圈，但并不代表，马圈中没有清醒的存在。
<br/>　　程大雷的坐骑墨玉乌月牛，这个时辰就没睡。白衣老尼来到后院，它就发现了，稀里糊涂也不知道这个白衣老尼要干什么，睁大眼睛看着她。
<br/>　　对于自己的坐骑，程大雷是十分看重的，有专人照顾，每日精草细料，对于一头牛而言，绝对可以称得上是养尊处优了。
<br/>　　它此时不知怎么会事，哞地就叫了一声。
<br/>　　这声音吓了白衣老尼一跳，毕竟是做贼心虚。但在看清是头黑牛时，才稍稍松了口气。
<br/>　　可谁料想，这叫声惊动了罐子里的蛤蟆，它被塞在罐子里，本就满心的不痛快。听到类似同类的声音，顿时激动起来，在罐子里发出阵阵怪叫。
<br/>　　一声赛过一声高，一头牛一只蛤蟆越发激动。
<br/>　　白衣老尼手忙脚乱，顿时不知该如何是好。她本该悄悄的进村，打枪得不要，现在这叫声响打鼓一般，整座城主府都被惊动了。
<br/>(本章完)
