　　随着学员们到齐，围观的学员都高呼起来，除了古薰儿与萧炎，其余名气学员都是众星捧月。
<br/>　　至于小医仙，曾经有学员听说过，她与古薰儿一样都是名花有主，但五年也不曾见所谓的主现身。
<br/>　　于是，谣言不攻自破。
<br/>　　此刻，她虽独自站在中央，但学员中有许多都是她的拥护者与追求者。
<br/>　　白山，吴昊在迦南学院中资质都算上等，自然也是众星捧月，身为迦南学院副院长孙女的琥嘉更是如此。
<br/>　　毕竟，迦南学院副院长可是一个权势滔天的职务，丝毫不比大陆上的一流势力弱。
<br/>　　而且，因为迦南学院的特殊地位，哪怕她的爷爷只是斗皇，面子也丝毫不比一些斗宗小。
<br/>　　毕竟，不提以前从迦南学院中走出的强者，就光凭学院明面上的势力，也足以将某些一流势力震慑的不敢胡来。
<br/>　　待参赛学员全部到齐，广场周围看台上的人数，几乎拥挤到爆棚。
<br/>　　黑压压的人头，一眼望去看不到尽头，各种喧嚣声交汇，最后在整个广场上空响彻不断。
<br/>　　“咚…咚…”
<br/>　　在蔚蓝天空耀日高升之际，清脆响亮的钟鸣声，终于在广场上空响了起来，待钟声响起，喧闹声终于是缓和了一些。
<br/>　　随后，所有目光几乎是同时望向极其宽敞的广场。
<br/>　　只见副院长琥乾站起身来，目光环视四周，雄浑的声音犹如闷雷一样在广场半空盘旋而起。
<br/>　　“昨日，内院选拔赛已经选出最后五十位学员，他们都有进入内院修行的资格，不过，内院中同样明确等级划分，各位想在内院中获得更好的修炼条件，那就需要在即刻到来的比赛中尽可能提高名次……”
<br/>　　琥乾目光瞥了一眼激动的学员，笑道：“往年的选拔赛都是选择回合比试，经过学院会议后，今年的比试则改成了另外一种模式。”
<br/>　　琥乾话语刚落，全场学员一愣，旋即将困惑的目光投向广场内。
<br/>　　“现在，请前五十名学员全部进入场内。”
<br/>　　虽然有的学员不明白，但依旧是按照琥乾的话语，一个个全都掠向广场。
<br/>　　随着越来越多的人影出现在广场上，看台上的气氛一瞬间就被太高了起来，欢呼声一片。
<br/>　　特别是当一袭白衣，玉树临风的白山，身材火爆的琥嘉，霸气非凡的吴昊出现后，气氛直接被带上了高/潮。
<br/>　　小医仙瞥了一眼古薰儿，经过彩鳞的提醒，她知道后者是本场比试最强的劲敌。
<br/>　　“你记住，比赛只有靠你自己，本王无法出手，否则便会被察觉到。”
<br/>　　说罢，彩鳞叮嘱道：“若是封印松懈，立刻停止比赛，你出个三长两短，等他回来，本王没法交代。”
<br/>　　当初，慕清霄离开的时候，彩鳞就答应会照顾好小医仙，五年的相处，两人也成为了要好的姐妹。
<br/>　　语气虽然有些冰冷，但何尝不是一种关心。
<br/>　　“彩鳞姐，我自己有分寸，你就放心吧。”
<br/>　　微微颔首，小医仙就收回目光，随后缓步走进广场。
<br/>　　彩鳞金色眸子瞥了眼小医仙手中的白色纳戒，暗自传音道：“老东西，关键时刻，不要让她受到伤害。”
<br/>　　“小女娃，你就不懂得尊重长辈吗？”
<br/>　　雪白色纳戒一亮，旋即变得暗淡下去。
<br/>　　“本尊灵魂已经恢复的七七八八，但迦南学院可没有想象中的简单，暗中有一位斗尊，本尊也不可能拉下脸皮对付一帮小娃娃，但是封印已经在消弱，已经不能在加固封印，否则……”
<br/>　　小医仙入场没有花哨的姿势与优美的动作，但这样普通的进场方式，依旧是引来了一群追求者的欢呼声。
<br/>　　随着小医仙，古薰儿与萧炎进场，原本就火热的气氛，再度抬高了一个层次。
<br/>　　等前五十名学员全部进入广场，琥乾目光在场上扫了一圈。
<br/>　　当目光扫向一些优秀弟子的时候稍稍停顿了一下，特别是看向古薰儿后，忍不住叹息一声。
<br/>　　别人不知道，他可是知道一些内幕。
<br/>　　听说古薰儿似乎是来自某个古老的家族，连院长都特意传讯回来，有必要的时候要照顾一番。
<br/>　　不过，后者暗中有强者保护，自身实力也不弱，根本不需要学院操心。
<br/>　　目光在萧炎身上扫过，琥乾眼底闪过一抹赞赏，最后目光落在了小医仙身上。
<br/>　　这名看似柔弱的女子也不是好惹的主，不禁让他想起，五年前与她一同来到迦南学院的青年，后者天赋……
<br/>　　许久，琥乾收回目光，声音如闷雷一样响起。
<br/>　　“今日的比试并非回合赛，而是一场淘汰赛，淘汰赛中坚持的越久，名次就越高，比赛中无论你们用任何手段，或者联合对敌，只要能在混战中坚持下来，便是胜利。”
<br/>　　琥乾话语刚落，广场中便是响起一阵窃窃私语，显然，与以往不同模式的比赛，使得他们都有些措手不及。
<br/>　　“比赛开始后，出场者便是输，因此，只要你能在场内多待一秒，说不定你在内院修炼的资源就会更高……”
<br/>　　萧炎与古薰儿站在一起，目光在场内扫视一圈，最后落在白山等人身上。
<br/>　　“比赛模式不错，正好我们占据优势，既然能够联手，我们就先将碍眼的踢出去吧。”
<br/>　　“嗯。”
<br/>　　古薰儿语气严肃道：“比赛模式对于白山，琥嘉甚至吴昊都有极大的好处，或许比赛开始后，他们会拉拢到不少人。”
<br/>　　萧炎有些无奈，目光看向小医仙，道：“难道要选对付她？她应该是慕师兄的女人，不太好吧？”
<br/>　　古薰儿白了他一眼，道：“如果没必要，自然不需要先对她下手，我们静观其变。”
<br/>　　琥乾在场内扫视一圈，道：“规则就这些，既然各位都已经明白，那么我宣布，内院选拔赛最后一轮淘汰赛，现在开始！”
<br/>　　“轰！”
<br/>　　随着琥乾声音落下，顷刻间，中央场地内就爆射出数十道颜色不同的斗气。
<br/>　　随着一声声闷响，场中学员几乎都是向边缘掠去，但也有的学员选择先发制人。
<br/>　　……
<br/>　　……
