　　第617章
<br/>　　程大雷望着周桃，悠悠道：“姑娘也是个可怜人呐。”
<br/>　　周桃一怔，抬头看了程大雷一眼，像是心里有话，只是没选择说出口。
<br/>　　乱世之下，多少人争权夺势，精心布置着自己的势力，想着要与群雄逐鹿天下。但大部分还是周桃这种人，伤痕累累，内里有太多不堪与外人讲起的事。她还算有些本事的，尚且如此，就更不用提那些普通人了。
<br/>　　“都是你这种人，才将世道搞成这个样子。”周桃忽然发作：“只有将你们这些人一个个都杀了，天下才能恢复太平。”
<br/>　　接着她又道：“莫郎便是一个有理想的人，他终会将太平还给黎民百姓。”
<br/>　　程大雷无语的看着她，心道上苍给了她美貌的同时，一定忘了给她脑子。如此，她会在茫茫人海中倾心莫将难，便不是没有原因。
<br/>　　看上去，她像是恨自己入骨，从她那里肯定听不到什么好话。程大雷索性把注意力放在外面，瞧瞧莫将难打算做什么。
<br/>　　从程大雷的角度，只能看到外面一角，只能凭声音判断外面情景。
<br/>　　今天是个满月，梨园内亮堂堂的，农神庙前聚的人越来越多，已经有两三千人。多是万马堂的人马，但同时也有京州凉州两地绿林道的豪杰。
<br/>　　莫将难转了一圈，最后回到农神庙前，他站在月光之下，开口朗声道：“诸位压言，请听莫某人一句，今天大家为何来到此地，许多人应该已经知道了。天下大乱，各路诸侯割据，咱们江湖人多是成了诸侯的走狗。这天下是有德人居之，各路诸侯能取，我们兄弟们为什么不能取。莫某召集各位英雄在此聚义，大家反了他娘的。”
<br/>　　梨园内一片安静，这些人平日里就是干些没本钱的买卖，每个人手上都有几条人命，但杀人是一回事，造反又是另外一回事。万马堂的人马，莫将难已经提起忽悠好了。但其他人仍旧有些含糊。
<br/>　　“莫老大，你想做事，弟兄们都服你。可凭咱们这些人，想着造反取天下，咱们行嘛？”有一人开口道。
<br/>　　“是呐，咱们才多少人，和官兵打仗，咱们都不会呐。”
<br/>　　一片乱哄哄的声音，许久方才安静下来。莫将难似乎早已料到会有这一幕发生，倒也不以为意，他一直等众人安静下来，方才开口道：“我想问一句，诸位可听说过那琴川之主程大雷。”
<br/>　　“这谁没听过，在江湖上混，要是没听过他的名声，大家就不用混了。”又是一片乱糟糟的声音。
<br/>　　“诸位，那程大雷本事如何？”莫将难问道。
<br/>　　“那还用说么，定是天下一等一的恶贼，敢和朝廷对着干，并州一战，他杀了多少人。现在蛤蟆城多大势力，做贼做到他那份上，算是做到家了。”
<br/>　　提起程大雷，大家都开始滔滔不绝，但奇怪的事，他们说的都是关于程大雷的事，但却又极力避免提及程大雷这三个字。
<br/>　　似乎这三个字有莫名的魔力，提起来就会厄运加身。
<br/>　　等人群安静下来，莫将难才缓缓道：“说来，我与那程大雷也有过一面之缘，他也并非三头六臂。想他当初起事之时，整座山寨不过才五人。如今才过去几年，便有偌大的家业。大家都是娘生爷养，比他又差了那里。天下大事，不怕你争，就怕你不争，只要敢争，这天下未必没有弟兄们的份儿。”
<br/>　　“说是这么说，可咱们怎么和那人相比，莫老大也不是那一位。”
<br/>　　莫将难呵呵笑了一声，道：“如果我告诉大家，我能亲手杀了他呢。”
<br/>　　月光下突然变得鸦雀无声，所有人像是被卡住喉咙一般，有一人忍不住问道：“莫老大不会是在看玩笑吧？”
<br/>　　“今天是什么日子，大家做的又是怎样的事，我岂会开玩笑。”莫将难道：“不瞒诸位，那程大雷就在此地，我要杀他就在此刻。”
<br/>　　说着，莫将难拍了拍手。
<br/>　　一众人伸长了脖子，睁大了眼睛，就见程大雷从农神庙里被押了出来。所有人都盯着他看，上上下下将他打量好几遍，与内心中程大雷的印象比对。
<br/>　　“程当家。”
<br/>　　莫将难对程大雷倒是客气，抱拳行了一礼。
<br/>　　“今日要用程当家的人头，祭我们兄弟的义旗，还请程当家勿怪。”
<br/>　　“好说，好说。”程大雷轻轻点头，目光从人群中扫过。
<br/>　　他双目扫过的地方，人们纷纷避让，不敢与其目光对视。这个时候，场上已经响起窃窃私语之声，有些人已经悄悄向外溜。
<br/>　　毕竟，程当家恶名在外，从他出世至今，又有那个和他做对的讨到好出。大家的性命要紧，还是不要趟这次浑水。
<br/>　　当然，也有许多人留了下来，难以置信的看着这一幕。心中隐隐对莫将难升起了信心。
<br/>　　梨园外忽然响起乱糟糟的声音，几匹快马奔进林子，其中一人高声喊道：“小桃，小桃你在何处。”
<br/>　　程大雷听到这声音就觉得有些头疼，那柳追风来了。
<br/>　　柳追风被周桃甩掉，他当然不会以为周桃故意甩脱她，还以为周桃遇到危险。他在方圆百里还是有些名声的，于是便找到这里碰碰运气。
<br/>　　见到周桃后，柳追风心中松口气，骑在马上道：“小桃，你快过来，咱们不搅这次乱。”
<br/>　　今天是西北绿林道的聚会，场上还颇有人和柳追风认识，见到他后纷纷打招呼。柳追风眼里却只有周桃，口中连唤几声，却没有得到周桃的回应。
<br/>　　莫将难哈哈笑了一声：“这位便是扶风一剑柳先生吧，实不相瞒，周桃本就是我万马堂的人。今日我们在此地聚义，如果柳先生愿意相助，我便做主让周桃给柳先生做妾。”
<br/>　　这话一出口，有两个人大吃一惊。一个是柳追风，他虽然一往情深，但却对周桃谨守君子之礼，不敢有半点无礼的行动。这时代，男人之间互赠姬妾是稀松平常的事情，他也不觉得怎样。
<br/>　　另外一人便是周桃，她的芳心早已暗许莫将难，却没料到莫将难会将自己轻易送人。
<br/>　　“你瞧瞧……”程大雷嘴角浮现一丝笑容：“事情怎么成了这样子。”
<br/>(本章完)
